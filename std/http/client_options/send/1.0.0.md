# std.http.client_options.send

Issues an HTTP OPTIONS request and returns the response (including any response body describing allowed methods and CORS settings).

## Signature

(url: string, headers: map[string, string]) -> result[http_response, string]

## Behavior

Opens a connection, serializes an OPTIONS request with no body, and reads the response (status, headers, and any body).

## Tests

+ returns the server's status, headers, and body
+ typically expects an Allow header in the response
- returns error when the URL is not http/https
- returns error when the server is unreachable

## Assumptions

? OPTIONS may be used for CORS preflight or server-capability discovery
