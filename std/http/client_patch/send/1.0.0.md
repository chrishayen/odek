# std.http.client_patch.send

Issues an HTTP PATCH request and returns the response.

## Signature

(url: string, headers: map[string, string], body: bytes) -> result[http_response, string]

## Behavior

Same orchestration as `client_post.send` but uses the PATCH method. Sets Content-Length automatically from the body.

## Tests

+ issues a PATCH with the supplied body
+ sets Content-Length automatically
+ returns the server's response verbatim
- returns error when the URL is not http/https
- returns error when the server is unreachable

## Assumptions

? PATCH is NOT idempotent by default per RFC 5789
