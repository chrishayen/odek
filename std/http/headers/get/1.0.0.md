# std.http.headers.get

Returns the value associated with a header name, matching case-insensitively, or none when absent.

## Signature

(headers: map[string, string], name: string) -> optional[string]

## Behavior

Lowercases `name` and looks it up in the map. When multiple values are present for one name, returns the combined value (comma-joined per RFC 7230) already stored in the map entry. Returns `none` when the name is absent.

## Tests

+ returns some(value) for a present header, regardless of stored case
+ returns some(value) regardless of the case of the query name
+ returns none for an absent header
+ returns none for an empty headers map
- returns a value unchanged from the stored entry (no trimming here)

## Assumptions

? headers maps are stored with lowercased keys; callers using raw maps pre-lowercase
? multi-valued headers are already comma-joined in storage
