# std.http.headers.serialize_line

Builds one `Name: value\r\n` HTTP header line.

## Signature

(name: string, value: string) -> result[string, string]

## Behavior

Validates `name` and `value`. Emits `<name>: <value>\r\n`. Preserves the input case of the name.

## Tests

+ returns "Content-Type: application/json\r\n" for the respective input
+ preserves the input case of the name
+ always terminates with CRLF
- returns error when name fails validate_name
- returns error when value fails validate_value

## Assumptions

? single space between colon and value
? CRLF terminator
