# std.http.headers.delete

Returns a new headers map with the entry for a given name removed, if present.

## Signature

(headers: map[string, string], name: string) -> map[string, string]

## Behavior

Lowercases `name`. Returns a copy of the input with that entry removed. Returns an equivalent map when the name is absent.

## Tests

+ removes the entry when the name is present (case-insensitive match)
+ returns an equivalent map when the name is absent
+ does not mutate the input
+ preserves all unrelated entries
- never grows the map

## Assumptions

? name matching is case-insensitive
