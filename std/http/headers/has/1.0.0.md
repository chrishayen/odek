# std.http.headers.has

Returns true when the headers map contains an entry for the given name (case-insensitive).

## Signature

(headers: map[string, string], name: string) -> bool

## Behavior

Lowercases `name` and checks for its presence as a key. Does not inspect the value.

## Tests

+ returns true when the header is present
+ returns true for case-insensitive matches
+ returns false when absent
+ returns false for an empty headers map
- returns true when only a different header is present under no circumstance

## Assumptions

? keys in the map are stored lowercased
