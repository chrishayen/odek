# std.http.headers.set

Returns a new headers map with the given name set to the given value, replacing any prior entries.

## Signature

(headers: map[string, string], name: string, value: string) -> map[string, string]

## Behavior

Lowercases `name`. Returns a copy of the input with the entry set to `value`. Replaces any existing entry for the same (lowercased) name — never appends.

## Tests

+ sets a new header when absent
+ replaces an existing header of the same name (case-insensitive match)
+ does not mutate the input
+ preserves all unrelated entries
- never produces a map with two entries for the same lowercased name

## Assumptions

? name is stored lowercased
