# std.http.headers.parse_line

Parses one HTTP header line (CRLF stripped) into a lowercased name and trimmed value.

## Signature

(line: string) -> result[header, string]

## Behavior

Splits `line` at the first `:`. Validates the name via `validate_name`. Trims leading and trailing optional whitespace (SP, HT) from the value, then validates via `validate_value`. Returns the name lowercased and the trimmed value.

## Tests

+ returns name "content-type" and value "application/json" for "Content-Type: application/json"
+ trims leading/trailing whitespace from the value
+ lowercases the name
+ accepts tabs as well as spaces between colon and value
- returns error on a line missing ":"
- returns error when the name fails validate_name
- returns error when the value fails validate_value

## Assumptions

? obs-fold (line continuation) is rejected
? name is always stored lowercased
