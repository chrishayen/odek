# std.http.headers.get_all

Returns every value associated with a header name, splitting a comma-joined value per RFC 7230.

## Signature

(headers: map[string, string], name: string) -> list[string]

## Behavior

Looks up the (case-insensitive) name. When the value contains commas outside of quoted strings, splits on those commas and trims each element. Returns a list of the separated values. Returns an empty list when the header is absent.

## Tests

+ returns a single-element list for a non-list header
+ returns a multi-element list when the stored value contains commas
+ preserves order
+ trims leading/trailing whitespace from each element
+ returns an empty list for an absent header
- does not split commas that occur inside quoted strings

## Assumptions

? quoting rules follow RFC 7230 quoted-string; `\` escapes inside quotes
? Set-Cookie intentionally is NOT a comma-separated list; callers use parse_set_cookie instead
