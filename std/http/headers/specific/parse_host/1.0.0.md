# std.http.headers.specific.parse_host

Parses a `Host` header value into a hostname (or IP literal) and optional port.

## Signature

(value: string) -> result[host_port, string]

## Behavior

Splits at the last `:` that is not inside `[]` (IPv6 literal brackets). Parses the right side as a port via `parse_decimal` when present.

## Tests

+ parses "example.com" into host "example.com", port none
+ parses "example.com:8080" into host "example.com", port 8080
+ parses "[::1]" into host "[::1]", port none
+ parses "[::1]:443" into host "[::1]", port 443
- returns error when the port is not a valid integer
- returns error when the port is outside 1..65535

## Assumptions

? the IPv6 brackets are preserved in the host component
