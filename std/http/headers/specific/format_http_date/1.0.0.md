# std.http.headers.specific.format_http_date

Formats a Unix epoch second count as an IMF-fixdate HTTP-date header value.

## Signature

(epoch_seconds: i64) -> string

## Behavior

Emits the canonical IMF-fixdate form: `<day>, <DD> <Mon> <YYYY> <hh>:<mm>:<ss> GMT` with English day/month names and UTC.

## Tests

+ emits "Sun, 06 Nov 1994 08:49:37 GMT" for the canonical test moment
+ always emits GMT offset
+ always uses English abbreviated day and month names
+ zero-pads day, hour, minute, second to two digits
- does not emit RFC 850 or asctime style

## Assumptions

? only the IMF-fixdate format is emitted (RFC 7231 recommendation)
? always UTC
