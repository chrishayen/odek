# std.http.headers.specific.parse_authorization

Splits an `Authorization` header value into its scheme token and credentials substring.

## Signature

(value: string) -> result[authorization, string]

## Behavior

Splits at the first run of whitespace. The token before becomes the scheme (lowercased). The remainder (trimmed) is the credentials string. Does not interpret credentials beyond that.

## Tests

+ parses "Basic dXNlcjpwYXNz" into scheme "basic", credentials "dXNlcjpwYXNz"
+ parses "Bearer eyJhbGc..." into scheme "bearer", credentials "eyJhbGc..."
+ lowercases the scheme
- returns error when the input contains no whitespace
- returns error when the scheme is empty
- returns error when the credentials portion is empty

## Assumptions

? scheme is compared case-insensitively; stored lowercase
? credentials are returned verbatim; scheme-specific runes decode further
