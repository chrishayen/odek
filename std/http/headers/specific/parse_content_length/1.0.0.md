# std.http.headers.specific.parse_content_length

Parses a `Content-Length` header value into a non-negative integer byte count.

## Signature

(value: string) -> result[i64, string]

## Behavior

Trims surrounding whitespace, then parses via `parse_decimal`. Rejects negative values.

## Tests

+ returns 0 for "0"
+ returns 42 for "42"
+ returns a large integer for a multi-million-byte length
- returns error for a negative number
- returns error for a non-numeric value
- returns error for an empty value

## Assumptions

? Content-Length is exactly one integer; multiple integers comma-separated are rejected
