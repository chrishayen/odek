# std.http.headers.specific.parse_cache_control

Parses a `Cache-Control` header value into a map of directives.

## Signature

(value: string) -> result[map[string, string], string]

## Behavior

Splits the value into comma-separated directives via `parse_list`. Each directive is either `name` (flag, mapped to empty string) or `name=value`. Names are lowercased. Values are unquoted when double-quoted.

## Tests

+ parses "no-cache" into {"no-cache": ""}
+ parses "max-age=60, private" into {"max-age": "60", "private": ""}
+ unquotes double-quoted values (e.g. `no-cache="Cookie"`)
+ lowercases directive names
- returns error on a directive with an empty name
- returns error when a quoted value is unterminated

## Assumptions

? duplicate directives follow last-wins
