# std.http.headers.specific.parse_authorization_basic

Decodes the credentials portion of a `Basic` Authorization header into a user/password pair.

## Signature

(credentials: string) -> result[basic_credentials, string]

## Behavior

Base64-decodes the credentials, validates UTF-8, then splits at the first `:` into username and password. The password may be empty; the username may not.

## Tests

+ returns user "alice" and password "s3cret" for base64 of "alice:s3cret"
+ accepts empty password (credentials like "alice:")
- returns error when base64 decoding fails
- returns error when decoded bytes are not valid UTF-8
- returns error when the decoded string contains no ":"
- returns error when the username is empty

## Assumptions

? credentials are already trimmed
? base64 is standard (RFC 4648), not URL-safe
