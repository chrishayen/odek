# std.http.headers.specific.parse_http_date

Parses an RFC 7231 HTTP-date header value (IMF-fixdate, RFC 850, or asctime) into Unix epoch seconds.

## Signature

(value: string) -> result[i64, string]

## Behavior

Tries the three historical formats in order: `Sun, 06 Nov 1994 08:49:37 GMT` (IMF-fixdate), `Sunday, 06-Nov-94 08:49:37 GMT` (RFC 850), `Sun Nov  6 08:49:37 1994` (asctime). Returns the parsed moment as epoch seconds UTC.

## Tests

+ parses IMF-fixdate successfully
+ parses RFC 850 date format
+ parses asctime format
+ accepts "GMT" timezone suffix (required for the two modern formats)
- returns error on any other format
- returns error on an invalid day/month/time

## Assumptions

? only GMT is accepted; any other offset is an error
? years in RFC 850 format are windowed to [1970, 2069]
