# std.http.headers.specific.parse_etag

Parses an `ETag` (or `If-Match`, `If-None-Match`) header value into a weak flag and opaque tag string.

## Signature

(value: string) -> result[etag, string]

## Behavior

Recognizes an optional `W/` prefix for weak tags. The remainder must be a double-quoted string; returns the unquoted contents as the tag. `*` returns the "any" wildcard.

## Tests

+ parses "\"abc\"" into weak=false, tag="abc"
+ parses "W/\"abc\"" into weak=true, tag="abc"
+ parses "*" into the wildcard variant
- returns error when the value is not quoted and not "*"
- returns error when the quoted string is unterminated

## Assumptions

? tag may contain any visible ASCII byte except `"` and `\`
