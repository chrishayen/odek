# std.http.headers.specific.parse_range

Parses a `Range: bytes=` header value into a list of byte ranges.

## Signature

(value: string) -> result[list[byte_range], string]

## Behavior

Strips the required "bytes=" prefix. Splits on commas. Each element is `start-end`, `start-` (open-ended), or `-suffix` (last N bytes). Returns the list of range variants.

## Tests

+ parses "bytes=0-99" into [(0, 99)]
+ parses "bytes=100-" into [(100, unbounded)]
+ parses "bytes=-500" into [suffix 500]
+ parses "bytes=0-99, 200-299" into two ranges
- returns error when the unit is not "bytes"
- returns error when a range has neither start nor suffix
- returns error when start > end in an explicit range

## Assumptions

? only the bytes unit is supported
