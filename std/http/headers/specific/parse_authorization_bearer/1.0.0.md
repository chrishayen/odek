# std.http.headers.specific.parse_authorization_bearer

Validates the credentials portion of a `Bearer` Authorization header and returns the token.

## Signature

(credentials: string) -> result[string, string]

## Behavior

Trims surrounding whitespace. Validates that every remaining byte is in the RFC 6750 `b64token` set (`A-Z a-z 0-9 - . _ ~ + /` plus `=` at end). Returns the token.

## Tests

+ returns the token for a standard JWT-like value
+ accepts tokens ending in padding "="
+ trims surrounding whitespace
- returns error when the token is empty
- returns error when a character outside the b64token set appears

## Assumptions

? character set is RFC 6750 b64token (superset of base64url)
