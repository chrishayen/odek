# std.http.headers.append

Returns a new headers map with the given value appended to an existing header, comma-joined per RFC 7230.

## Signature

(headers: map[string, string], name: string, value: string) -> map[string, string]

## Behavior

Lowercases `name`. When the header is absent, behaves like `set`. When present, concatenates the prior value and the new value with `", "` (comma + single space) and stores the combined string.

## Tests

+ sets a new header when the name is absent
+ joins a new value with the existing one using ", "
+ preserves the prior value's content
+ does not mutate the input
- does not produce a separate duplicate entry; uses comma-joining instead

## Assumptions

? separator is always ", " per RFC 7230 section 3.2.2
? Set-Cookie is the only real exception in HTTP; callers use a different function for it
