# std.http.headers.parse_block

Parses a complete HTTP header block (lines terminated by CRLF, ended by a blank CRLF) into a name/value map.

## Signature

(block: string) -> result[map[string, string], string]

## Behavior

Splits the input on CRLF. Stops at the first blank line. Runs each non-blank line through `parse_line` and accumulates via `append` so duplicate headers are comma-joined.

## Tests

+ returns a map containing every header in a well-formed block
+ stops reading at the blank-line terminator
+ comma-joins duplicate headers via append
- returns error when any line fails parse_line
- returns error when the block does not end with CRLF CRLF

## Assumptions

? line separator is CRLF; bare LF is not tolerated
? terminator is CRLF CRLF
