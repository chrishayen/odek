# std.http.headers.value.parse_list

Splits an RFC 7230 `1#element` comma-separated header value into its components, honoring quoted strings.

## Signature

(value: string) -> list[string]

## Behavior

Walks the input, splitting on commas that are not inside a double-quoted string. Trims leading and trailing whitespace from each element. Empty elements (two consecutive commas) are dropped.

## Tests

+ returns ["a", "b", "c"] for input "a, b, c"
+ returns ["a,b", "c"] for input "\"a,b\", c" (comma inside quotes is preserved)
+ trims whitespace around elements
+ returns [value] when the input has no unquoted commas
+ drops empty elements from consecutive commas
- returns [] for an empty input

## Assumptions

? quoted strings use `"` and `\"` escapes per RFC 7230
? trimmed whitespace is SP and HT only
