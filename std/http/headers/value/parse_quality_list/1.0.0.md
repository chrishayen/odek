# std.http.headers.value.parse_quality_list

Parses a comma-separated list where each element has an optional `;q=<float>` quality weight.

## Signature

(value: string) -> result[list[weighted], string]

## Behavior

Splits via `parse_list`, then for each element runs `parse_parameters` to extract `q`. Parses `q` as an f64 in [0.0, 1.0]. Defaults missing `q` to 1.0. Returns the list of (token, weight) pairs in input order.

## Tests

+ returns [("a", 1.0), ("b", 0.5)] for "a, b;q=0.5"
+ defaults missing q to 1.0
+ accepts q values in [0.0, 1.0]
+ preserves input order
- returns error when q is outside [0.0, 1.0]
- returns error when q is not a number

## Assumptions

? sorting by weight is the caller's job
