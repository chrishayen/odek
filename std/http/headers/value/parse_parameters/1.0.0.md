# std.http.headers.value.parse_parameters

Parses a header value of the form `base; key=value; key2=value2` into its base and parameter map.

## Signature

(value: string) -> result[parameters_result, string]

## Behavior

Splits the input at the first `;`. The part before is the base token. Each remaining segment is `key=value`; keys are lowercased; values are unquoted when surrounded by `"`. Returns base + map.

## Tests

+ returns base "text/html" and empty params for "text/html"
+ returns base "text/html" and {"charset": "utf-8"} for "text/html; charset=utf-8"
+ unquotes double-quoted values
+ lowercases parameter keys
- returns error when a parameter segment has no "="
- returns error when a key is empty

## Assumptions

? parameters use `;` as separator per RFC 7231
? values containing special characters must be quoted
