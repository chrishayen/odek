# std.http.content_type.match

Returns true when a concrete content-type matches a pattern (which may use wildcards).

## Signature

(offered: content_type, pattern: content_type) -> bool

## Behavior

Returns true when `pattern.type == "*"` or matches `offered.type`, AND `pattern.subtype == "*"` or matches `offered.subtype`. Does not require pattern parameters to match.

## Tests

+ returns true when both patterns match exactly
+ returns true for "*/*" matching any offered type
+ returns true for "text/*" matching "text/html"
+ returns false when type differs
+ returns false when subtype differs and pattern subtype is not "*"
- does not require parameters to match

## Assumptions

? wildcard is "*"; partial wildcards like "te*" are not supported
