# std.http.content_type.parse

Parses a Content-Type string. Facade over the header-namespace parser for discoverability.

## Signature

(value: string) -> result[content_type, string]

## Behavior

Delegates to `std.http.headers.specific.parse_content_type`.

## Tests

+ parses "text/html" correctly
+ parses "application/json; charset=utf-8" correctly
- returns error on malformed input (same as underlying)

## Assumptions

? facade; canonical parser lives under http.headers.specific
