# std.http.content_type.extract_charset

Returns the `charset` parameter of a content type (lowercased), or none when absent.

## Signature

(ct: content_type) -> optional[string]

## Behavior

Looks up `charset` in the content-type's parameters map. Returns `some(charset_lowercased)` when present, `none` otherwise.

## Tests

+ returns some("utf-8") for `text/html; charset=UTF-8`
+ returns none when charset is absent
+ lowercases the charset value
- returns none for an empty parameters map

## Assumptions

? charset lookup is case-insensitive on the key
