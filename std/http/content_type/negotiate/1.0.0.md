# std.http.content_type.negotiate

Picks the best match from the server's offered content types against the client's weighted Accept list.

## Signature

(offered: list[content_type], accepts: list[weighted_media_type]) -> optional[content_type]

## Behavior

For each client-accept entry (sorted by quality desc, specificity desc), returns the first offered type that matches via `match`. Skips entries with q=0. Returns none when no acceptable offer exists.

## Tests

+ returns the best-quality acceptable offer
+ skips entries with q=0 (explicit rejection)
+ returns the most specific match when multiple offers match
+ returns none when all offers are rejected
- prefers higher-quality entries over lower

## Assumptions

? specificity tiebreak: `type/subtype` > `type/*` > `*/*`
