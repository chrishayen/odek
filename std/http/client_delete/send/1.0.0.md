# std.http.client_delete.send

Issues an HTTP DELETE request and returns the response.

## Signature

(url: string, headers: map[string, string]) -> result[http_response, string]

## Behavior

Opens a connection, serializes a DELETE request with no body, and reads the response. DELETE is idempotent and typically has no request body.

## Tests

+ issues a DELETE to the URL
+ serializes without a Content-Length (no body)
+ returns the server's response verbatim
- returns error when URL is not http/https
- returns error when the server is unreachable

## Assumptions

? DELETE carries no request body by default
