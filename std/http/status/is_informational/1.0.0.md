# std.http.status.is_informational

Returns true when an HTTP status code is in the 1xx (informational) class.

## Signature

(status: i32) -> bool

## Behavior

Returns true when `100 <= status < 200`, false otherwise.

## Tests

+ returns true for 100 (Continue)
+ returns true for 101 (Switching Protocols)
+ returns false for 200
+ returns false for 99 or less
- returns true for a 2xx under no circumstance

? range classification; no individual-code semantics here
