# std.http.status.is_client_error

Returns true when an HTTP status code is in the 4xx (client error) class.

## Signature

(status: i32) -> bool

## Behavior

Returns true when `400 <= status < 500`, false otherwise.

## Tests

+ returns true for 400, 401, 403, 404
+ returns true for 499
- returns false for 399
- returns false for 500

## Assumptions

? range classification only
