# std.http.status.is_server_error

Returns true when an HTTP status code is in the 5xx (server error) class.

## Signature

(status: i32) -> bool

## Behavior

Returns true when `500 <= status < 600`, false otherwise.

## Tests

+ returns true for 500, 502, 503, 504
+ returns true for 599
- returns false for 499
- returns false for 600
