# std.http.status.is_success

Returns true when an HTTP status code is in the 2xx (success) class.

## Signature

(status: i32) -> bool

## Behavior

Returns true when `200 <= status < 300`, false otherwise.

## Tests

+ returns true for 200
+ returns true for 204 (No Content)
+ returns true for 299
- returns false for 199
- returns false for 300
