# std.http.status.is_redirect

Returns true when an HTTP status code is in the 3xx (redirect) class.

## Signature

(status: i32) -> bool

## Behavior

Returns true when `300 <= status < 400`, false otherwise.

## Tests

+ returns true for 301, 302, 303, 307, 308
+ returns true for 304 (technically a redirect class)
- returns false for 200
- returns false for 400

## Assumptions

? 304 is included in redirect class even though it is not followable
