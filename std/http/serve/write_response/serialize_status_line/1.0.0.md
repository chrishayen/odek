# std.http.serve.write_response.serialize_status_line

Builds the `HTTP/1.1 <code> <reason>` status line for an HTTP response.

## Signature

(status_code: i32, reason_phrase: string) -> result[string, string]

## Behavior

Formats the input into the status line `HTTP/1.1 <code> <reason>\r\n`. The code is rendered as exactly three decimal digits. Rejects codes outside 100-599 and reason phrases containing CR or LF.

## Tests

+ returns "HTTP/1.1 200 OK\r\n" for input (200, "OK")
+ returns "HTTP/1.1 404 Not Found\r\n" for input (404, "Not Found")
- returns error when status_code is below 100
- returns error when status_code is above 599
- returns error when reason_phrase contains CR or LF

## Assumptions

? protocol version is HTTP/1.1
? always terminates with CRLF
