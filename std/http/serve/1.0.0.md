# std.http.serve

Serves HTTP requests on a TCP listener by dispatching each to a handler function.

## Signature

(listener: tcp_listener, handler: func(http_request) -> http_response) -> result[void, string]

## Behavior

Accepts connections in a loop, parses each as one or more HTTP/1.1 requests (honoring keep-alive), invokes `handler` with the parsed request, and writes the returned response back. Blocks the caller until the listener is closed. Malformed requests produce a 400 response without invoking the handler.

## Tests

+ dispatches each parsed request to the handler
+ writes the handler's response back to the client
+ honors HTTP/1.1 keep-alive for multiple requests on one connection
+ closes the connection when the client sends "Connection: close"
+ returns a 400 response for malformed requests without invoking the handler
- returns error when the listener is closed or drops
- returns void when the listener is closed normally

## Assumptions

? single-threaded connection handling unless extended by the host
? protocol is HTTP/1.1 only; HTTP/2 requires a different server
