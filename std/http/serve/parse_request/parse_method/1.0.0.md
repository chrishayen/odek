# std.http.serve.parse_request.parse_method

Validates and returns an HTTP method token.

## Signature

(token: string) -> result[string, string]

## Behavior

Accepts the token from the start of an HTTP request line up to the first space. Returns it unchanged when it is a non-empty sequence of upper-case ASCII letters (A-Z). All other inputs are rejected.

## Tests

+ returns "GET" for input "GET"
+ returns the input unchanged for any sequence of A-Z up to 20 characters
- returns error for an empty input
- returns error when the input contains a lowercase letter
- returns error when the input contains a digit or punctuation

## Assumptions

? RFC 7230 allows any token; this rune restricts to A-Z for simplicity
? method length is capped at 20 characters to limit memory
