# std.http.serve.parse_request.read_body_fixed

Reads exactly `length` bytes from a connection and returns them as a buffer.

## Signature

(conn: tcp_conn, length: i64) -> result[bytes, string]

## Behavior

Issues repeated reads against the connection until exactly `length` bytes have been gathered. Returns the concatenated bytes in arrival order. An input length of 0 returns an empty buffer without touching the connection.

## Tests

+ returns exactly `length` bytes when the connection provides at least that many
+ returns an empty buffer when length is 0
+ retries on short reads until the full length is gathered
- returns error when the connection closes before `length` bytes have been read
- returns error when length is negative

## Assumptions

? short reads are retried transparently
? the connection cursor is advanced by exactly `length` bytes on success
