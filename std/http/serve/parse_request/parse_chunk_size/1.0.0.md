# std.http.serve.parse_request.parse_chunk_size

Parses a chunked-encoding chunk-size line into a non-negative byte count.

## Signature

(line: string) -> result[i64, string]

## Behavior

Accepts a chunk-size line (CRLF stripped) from a chunked Transfer-Encoding stream. Reads the leading hexadecimal digits, ignores any `;name=value` extension suffix, and returns the decoded integer. Rejects inputs with no hex digits or with characters other than hex / `;` / space.

## Tests

+ returns 0 for input "0"
+ returns 255 for input "ff"
+ ignores chunk extensions like "ff;name=value"
- returns error when the input has no hex digits
- returns error on a hex value that overflows i64
- returns error on characters outside hex/`;`/space

## Assumptions

? hex digits are case-insensitive
? extensions are silently discarded, not returned
