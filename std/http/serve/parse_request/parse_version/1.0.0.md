# std.http.serve.parse_request.parse_version

Validates and returns the HTTP version token from a request line.

## Signature

(token: string) -> result[string, string]

## Behavior

Accepts the third token of an HTTP request line. Returns it unchanged when it is exactly `HTTP/1.0` or `HTTP/1.1`. All other inputs are rejected.

## Tests

+ returns "HTTP/1.1" for input "HTTP/1.1"
+ returns "HTTP/1.0" for input "HTTP/1.0"
- returns error for "HTTP/2.0"
- returns error for "http/1.1" (lowercase)
- returns error for an empty input

## Assumptions

? only HTTP/1.0 and HTTP/1.1 are supported; HTTP/2 and HTTP/3 require separate servers
? the literal "HTTP/" prefix is fixed and case-sensitive
