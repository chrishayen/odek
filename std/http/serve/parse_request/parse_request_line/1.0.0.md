# std.http.serve.parse_request.parse_request_line

Splits an HTTP request line into its method, target, and version components.

## Signature

(line: string) -> result[request_line, string]

## Behavior

Accepts the first line of an HTTP request (CRLF stripped). Splits it on exactly two single-space separators into three tokens, then validates each token via its corresponding parse rune. Returns a `request_line` value carrying all three on success.

## Tests

+ returns the three components for "GET / HTTP/1.1"
+ returns the three components for "POST /submit HTTP/1.0"
- returns error when the line does not contain exactly two spaces
- returns error when any of the three sub-tokens fails its validator
- returns error on an empty line

## Assumptions

? separator is a single SP byte (0x20); tab or multi-space separators are rejected
? trailing CRLF must be stripped before calling; this rune operates on a single line
