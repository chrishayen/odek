# std.http.serve.parse_request.parse_target

Validates and returns the request-target (origin-form path) from an HTTP request line.

## Signature

(token: string) -> result[string, string]

## Behavior

Accepts the token between the method and HTTP version in a request line. Returns it unchanged when it is a non-empty origin-form target beginning with `/` and containing only characters in the RFC 3986 pchar + `/` + `?` set. All other inputs are rejected.

## Tests

+ returns "/" for input "/"
+ returns "/path/to/resource" for input "/path/to/resource"
+ returns "/path?query=1" for input "/path?query=1"
- returns error for an empty input
- returns error when the input does not start with "/"
- returns error when the input contains a space or control character

## Assumptions

? only origin-form is accepted; absolute-form and authority-form belong to separate runes
? percent-encoded octets are treated as valid pchar and not decoded here
