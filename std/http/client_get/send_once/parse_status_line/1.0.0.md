# std.http.client_get.send_once.parse_status_line

Parses the first line of an HTTP response into version, status code, and reason phrase.

## Signature

(line: string) -> result[status_line, string]

## Behavior

Accepts the first line of an HTTP response (CRLF stripped). Splits on two single-space separators into three tokens: version, code, and reason. Validates that the version is `HTTP/1.0` or `HTTP/1.1` and that the code is a three-digit decimal integer. Returns all three.

## Tests

+ parses "HTTP/1.1 200 OK" into version "HTTP/1.1", code 200, reason "OK"
+ parses "HTTP/1.1 404 Not Found" preserving a multi-word reason
+ accepts HTTP/1.0 and HTTP/1.1 versions
- returns error when the status code is not exactly three digits
- returns error when the version is not HTTP/1.0 or HTTP/1.1
- returns error when the line has fewer than two spaces

## Assumptions

? reason phrase may be empty (the trailing space before CRLF is preserved)
? line must already be trimmed of CRLF
