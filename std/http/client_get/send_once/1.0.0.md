# std.http.client_get.send_once

Issues exactly one HTTP GET and returns the raw response without following redirects.

## Signature

(url: string, headers: map[string, string]) -> result[http_response, string]

## Behavior

Opens a TCP (or TLS) connection to the URL's host and port, writes a GET request with the supplied headers, and reads the response. Any 3xx redirect status is returned as-is; the caller decides whether to follow. Applies a default connect and read timeout chosen by the implementation.

## Tests

+ returns an http_response with status 200 for a direct success
+ returns a 3xx response as-is with the Location header preserved
+ returns the response body bytes verbatim
+ preserves response headers in a case-insensitive map
+ sends a default User-Agent when the caller does not supply one
- returns error when the URL has no scheme or an unsupported scheme
- returns error when the server is unreachable
- returns error when the response cannot be parsed

## Assumptions

? only http and https schemes are supported
? timeout defaults are implementation-defined
