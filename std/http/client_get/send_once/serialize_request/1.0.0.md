# std.http.client_get.send_once.serialize_request

Builds a complete HTTP/1.1 GET request byte string.

## Signature

(target: string, host: string, headers: map[string, string]) -> string

## Behavior

Formats `GET <target> HTTP/1.1\r\n`, appends a `Host: <host>\r\n` header, appends every entry of `headers` as `Name: value\r\n`, and terminates with a blank CRLF. Supplies a default `User-Agent` only when the caller has not already set one.

## Tests

+ starts with "GET <target> HTTP/1.1\r\n"
+ includes "Host: <host>\r\n" immediately after the request line
+ appends each caller-supplied header verbatim
+ supplies a default User-Agent when headers do not contain one
+ terminates with a blank CRLF
- omits no caller-supplied header

## Assumptions

? target is already validated origin-form
? Host header is forced; caller-supplied Host is overwritten with the url's host
