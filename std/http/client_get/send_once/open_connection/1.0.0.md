# std.http.client_get.send_once.open_connection

Opens a TCP (or TLS) connection to the host and port of an HTTP URL.

## Signature

(url: url) -> result[tcp_conn, string]

## Behavior

Reads host, port, and scheme from the URL, opens a TCP connection via `std.net.tcp_connect`, and performs a TLS handshake when the scheme is `https`. Returns the ready-to-use connection. Uses an implementation-defined connect timeout.

## Tests

+ returns a plaintext TCP connection for an http URL
+ returns a TLS-wrapped connection for an https URL
+ uses port 80 for http when no port is in the URL
+ uses port 443 for https when no port is in the URL
- returns error when the URL scheme is neither http nor https
- returns error when the remote host is unreachable
- returns error when the TLS handshake fails

## Assumptions

? TLS certificate verification is enabled by default
? connect timeout is implementation-defined
