# std.http.client_get

Issues an HTTP GET request and returns the response.

## Signature

(url: string, headers: map[string, string]) -> result[http_response, string]

## Behavior

Sends an HTTP GET request to the absolute URL with the supplied request headers. Follows up to 10 redirects by default. Returns the final status code, response headers, and body bytes. Applies a default connect and read timeout chosen by the implementation.

## Tests

+ returns an http_response with status 200 for a successful GET
+ returns the response body bytes verbatim
+ preserves response headers in a case-insensitive map
+ follows 301/302 redirects and returns the final response
+ accepts empty headers and sends a default user-agent
- returns error for a URL without a scheme
- returns error when the server is unreachable
- returns error when more than 10 redirects occur

## Assumptions

? only http and https schemes are supported
? redirect policy follows at most 10 hops; extending is a separate feature
