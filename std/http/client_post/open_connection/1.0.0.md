# std.http.client_post.open_connection

Opens a TCP (or TLS) connection to the host and port of an HTTP URL for issuing a POST.

## Signature

(url: url) -> result[tcp_conn, string]

## Behavior

Reads host, port, and scheme from the URL, opens a TCP connection via `std.net.tcp_connect`, and performs a TLS handshake when the scheme is `https`. Identical to the GET-side equivalent but exposed under the POST namespace so the two client flows can evolve independently.

## Tests

+ returns a plaintext TCP connection for an http URL
+ returns a TLS-wrapped connection for an https URL
+ uses the scheme default port when none is specified
- returns error when the URL scheme is neither http nor https
- returns error when the remote host is unreachable

## Assumptions

? TLS certificate verification is enabled by default
? connect timeout is implementation-defined
