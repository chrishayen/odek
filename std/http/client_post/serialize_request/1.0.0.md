# std.http.client_post.serialize_request

Builds the request-line and header block of an HTTP/1.1 POST request (body is sent separately).

## Signature

(target: string, host: string, headers: map[string, string], body: bytes) -> string

## Behavior

Formats `POST <target> HTTP/1.1\r\n`, adds a `Host` header, appends every caller-supplied header, and terminates the block with a blank CRLF. Returns only the head of the request; the caller writes the body bytes after.

## Tests

+ starts with "POST <target> HTTP/1.1\r\n"
+ includes the Host header immediately after the request line
+ appends each caller-supplied header verbatim
+ terminates the serialized head with a blank CRLF
- does not include any body bytes in the returned string

## Assumptions

? body is written separately by the caller
? Host is forced; caller-supplied Host is overwritten
