# std.http.client_post.parse_response

Reads and assembles a full HTTP response from a connection after a POST has been sent.

## Signature

(conn: tcp_conn) -> result[http_response, string]

## Behavior

Reads the status line via the shared `parse_status_line`, the header block via `parse_response_headers`, and the body via `read_response_body`. Returns the assembled `http_response` value.

## Tests

+ returns a response with status, headers, and body populated
+ returns an empty body when Content-Length is 0
+ handles chunked responses by delegating to read_response_body
- returns error when the status line fails to parse
- returns error when the header block fails to parse

## Assumptions

? response parsing is identical to the GET path; this rune exists only to namespace the composition
? connection is left in whatever state read_response_body leaves it
