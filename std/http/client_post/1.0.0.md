# std.http.client_post

Issues an HTTP POST request with a body and returns the response.

## Signature

(url: string, headers: map[string, string], body: bytes) -> result[http_response, string]

## Behavior

Sends an HTTP POST request to the absolute URL with the supplied request headers and body. Sets `Content-Length` automatically unless the caller provides `Transfer-Encoding`. Does not follow redirects on POST by default (301/302 responses are returned as-is).

## Tests

+ returns the response for a successful POST
+ sets Content-Length automatically from the body length
+ preserves caller-supplied headers, overriding automatic ones only when not set by caller
+ returns the response body bytes verbatim
+ accepts an empty body
- returns error for a URL without a scheme
- returns error when the server is unreachable

## Assumptions

? redirects are not followed automatically on POST; caller re-issues if desired
? Content-Type is not set automatically; callers supply it
