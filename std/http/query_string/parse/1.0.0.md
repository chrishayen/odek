# std.http.query_string.parse

Parses a URL query-string value into a list of key/value pairs.

## Signature

(value: string) -> result[list[pair[string, string]], string]

## Behavior

Splits on `&`. For each segment, splits at the first `=` into key and value (value defaults to empty). Percent-decodes each via `decode_key_value`. Decodes `+` as space (form-encoding). Preserves order and duplicates.

## Tests

+ parses "a=1&b=2" into [("a","1"), ("b","2")]
+ decodes "a=hello+world" to ("a", "hello world")
+ decodes percent-encoded octets
+ preserves duplicate keys in order
+ treats a key without "=" as (key, "")
- returns error on malformed percent-encoding

## Assumptions

? input must not contain the leading "?"; caller strips it
? + is decoded to space (form-encoding conventions)
