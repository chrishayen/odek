# std.http.query_string.encode_key_value

Encodes one key or value for a URL query string, using "+" for space.

## Signature

(s: string) -> string

## Behavior

Percent-encodes all non-unreserved characters, then replaces encoded space (`%20`) with `+` to follow form-encoding convention.

## Tests

+ returns "hello+world" for "hello world"
+ returns "a%26b" for "a&b"
+ returns empty for empty input
+ leaves unreserved chars unchanged
- does not leave "+" in a literal form (a literal "+" must be encoded to "%2B")

## Assumptions

? + is reserved; literal "+" in input is encoded as "%2B"
