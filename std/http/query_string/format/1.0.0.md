# std.http.query_string.format

Formats a list of key/value pairs as a URL query string.

## Signature

(pairs: list[pair[string, string]]) -> string

## Behavior

Encodes each key and value via `encode_key_value` (percent-encoding + `+` for space) and joins with `&`. Emits `key=value` for each pair. Returns the empty string for an empty list.

## Tests

+ emits "a=1&b=2" for [("a","1"), ("b","2")]
+ encodes spaces as "+"
+ encodes special characters via percent-encoding
+ preserves input order
- does not emit a leading "?" (caller adds it)
- returns "" for an empty input

## Assumptions

? leading "?" is the URL builder's responsibility
? + for space, not %20, by default (form-encoding)
