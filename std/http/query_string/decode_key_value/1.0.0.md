# std.http.query_string.decode_key_value

Decodes one key or value from a URL query string, translating "+" to space.

## Signature

(s: string) -> result[string, string]

## Behavior

Replaces every `+` with a space, then percent-decodes via `url.decode_component`. The two-step order means a literal `%20` remains a space and a literal `+` is decoded to space.

## Tests

+ decodes "hello+world" to "hello world"
+ decodes "a%26b" to "a&b"
+ decodes empty input to empty
- returns error on malformed percent-encoding

## Assumptions

? this is form-decoding behavior; URL-component decoding (no + substitution) is a separate rune
