# std.http.cookies.parse_set_cookie_header

Parses a server-sent `Set-Cookie` header into its name, value, and attributes. Facade over the shared header parser.

## Signature

(value: string) -> result[set_cookie, string]

## Behavior

Delegates to `std.http.headers.specific.parse_set_cookie` unchanged.

## Tests

+ parses "id=abc; Path=/" into name "id", value "abc", attrs {"path": "/"}
+ returns the same result as parse_set_cookie
- returns error on malformed input

## Assumptions

? facade; canonical implementation lives under http.headers.specific
