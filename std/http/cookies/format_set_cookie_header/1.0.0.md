# std.http.cookies.format_set_cookie_header

Formats a cookie and its attributes as a server-side `Set-Cookie` header value.

## Signature

(name: string, value: string, attrs: map[string, string]) -> string

## Behavior

Emits `<name>=<value>`, then for each attribute emits `; <key>=<value>` (or `; <key>` when the attribute value is empty, for flag attributes like `Secure`). Preserves caller-supplied attribute case.

## Tests

+ emits "id=abc" when attrs is empty
+ emits "id=abc; Path=/; HttpOnly" for attrs {"Path": "/", "HttpOnly": ""}
+ flag attributes (empty value) emit without "=" sign
+ preserves attribute iteration order
- does not validate attributes beyond basic non-empty-key

## Assumptions

? attribute key case is preserved for readability
