# std.http.multipart.parse_part_body

Returns the body of a multipart part as a byte buffer, stripping any Content-Transfer-Encoding framing. Placeholder for future decoder dispatch.

## Signature

(bytes: bytes) -> bytes

## Behavior

Currently returns the input unchanged. When Content-Transfer-Encoding decoders (base64, quoted-printable) are added, this rune dispatches to them based on the part's headers.

## Tests

+ returns the input unchanged when no encoding is applied
+ returns empty for an empty input

## Assumptions

? transfer-encoding decoding is deferred to a later version; current behavior is pass-through
