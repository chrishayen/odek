# std.http.multipart.parse_boundary

Extracts the `boundary` parameter from a parsed Content-Type.

## Signature

(content_type: content_type) -> result[string, string]

## Behavior

Looks up the `boundary` key in the content-type's parameters map. Returns the boundary string verbatim. Rejects empty boundaries.

## Tests

+ returns the boundary for `multipart/form-data; boundary=abc`
+ accepts quoted boundaries (unquoting is done by parse_parameters)
- returns error when the boundary parameter is missing
- returns error when the boundary is empty

## Assumptions

? caller has already parsed the Content-Type via parse_content_type
