# std.http.multipart.split_parts

Splits a multipart body into its parts using the boundary marker.

## Signature

(body: bytes, boundary: string) -> result[list[bytes], string]

## Behavior

Scans for occurrences of `--<boundary>` preceded by CRLF (or at start of body). Returns the bytes between consecutive boundaries, excluding the boundary lines themselves. Stops at the closing `--<boundary>--` marker.

## Tests

+ returns each part's bytes for a well-formed multipart body
+ strips the leading preamble (bytes before the first boundary)
+ strips the trailing epilogue (bytes after the closing boundary)
+ returns an empty list when the body has only boundaries (no parts)
- returns error when the closing `--<boundary>--` is missing
- returns error when a part is missing the required CRLF separator after the boundary

## Assumptions

? boundary is the raw string (no leading `--`); this rune adds those
? body is held in memory
