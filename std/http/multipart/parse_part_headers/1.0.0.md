# std.http.multipart.parse_part_headers

Splits one multipart part into its header block and body bytes.

## Signature

(part: bytes) -> result[header_body, string]

## Behavior

Scans for the first CRLF CRLF separator. Parses the bytes before it via `parse_block` into a headers map. Returns headers + remaining body bytes.

## Tests

+ returns the headers map and body for a well-formed part
+ accepts a part with no body (empty body bytes)
- returns error when CRLF CRLF is missing
- returns error when header parsing fails

## Assumptions

? part bytes do not include the boundary lines themselves
