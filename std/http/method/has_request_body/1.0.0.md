# std.http.method.has_request_body

Returns true when an HTTP method typically carries a request body.

## Signature

(method: string) -> bool

## Behavior

Returns true for `POST`, `PUT`, `PATCH`. Returns false for all others. Note: the RFC permits bodies on GET etc., but it is not idiomatic.

## Tests

+ returns true for POST
+ returns true for PUT
+ returns true for PATCH
- returns false for GET
- returns false for HEAD
- returns false for DELETE

## Assumptions

? conservative policy matching common client/server behavior, not the permissive spec
