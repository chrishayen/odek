# std.http.method.is_idempotent

Returns true when an HTTP method is idempotent (repeating the request has the same effect as making it once) per RFC 7231.

## Signature

(method: string) -> bool

## Behavior

Returns true for `GET`, `HEAD`, `OPTIONS`, `TRACE`, `PUT`, `DELETE`. Returns false for `POST`, `PATCH`, and anything else.

## Tests

+ returns true for every safe method
+ returns true for PUT
+ returns true for DELETE
- returns false for POST
- returns false for PATCH

## Assumptions

? idempotent method set is RFC 7231 + RFC 5789 (PATCH)
