# std.http.method.is_safe

Returns true when an HTTP method is "safe" (does not intend to modify server state) per RFC 7231.

## Signature

(method: string) -> bool

## Behavior

Returns true for `GET`, `HEAD`, `OPTIONS`, `TRACE`. Returns false for all others.

## Tests

+ returns true for GET
+ returns true for HEAD
+ returns true for OPTIONS
+ returns true for TRACE
- returns false for POST
- returns false for PUT
- returns false for DELETE

## Assumptions

? method is already validated as an uppercase token
