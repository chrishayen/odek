# std.http.method.allows_response_body

Returns true when a response to the given method + status can have a body per RFC 7230 section 3.3.3.

## Signature

(method: string, status: i32) -> bool

## Behavior

Returns false when: method is HEAD; status is 1xx; status is 204 or 304. Returns true otherwise.

## Tests

+ returns false for HEAD regardless of status
+ returns false for status 204 (No Content)
+ returns false for status 304 (Not Modified)
+ returns false for status in 100-199 (informational)
+ returns true for a normal GET 200
- returns true for HEAD under no circumstance

## Assumptions

? follows RFC 7230 section 3.3.3 — these cases MUST NOT include a body
