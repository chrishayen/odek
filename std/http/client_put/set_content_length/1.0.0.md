# std.http.client_put.set_content_length

Sets Content-Length on PUT request headers from the body length. Facade over the shared post-side helper.

## Signature

(headers: map[string, string], body: bytes) -> map[string, string]

## Behavior

Delegates to `std.http.client_post.set_content_length`.

## Tests

+ sets Content-Length from the body length
+ returns headers unchanged when Transfer-Encoding is present
- does not mutate the input headers
