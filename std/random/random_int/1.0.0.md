# std.random.random_int

Returns a pseudo-random integer uniformly distributed in the half-open range [min, max).

## Signature

(min: i64, max: i64) -> result[i64, string]

## Behavior

Draws a uniformly distributed integer where `min <= result < max`. The range is half-open: `max` is never returned. Uses a cryptographically strong or system-provided generator so that the result is not predictable between calls.

## Tests

+ returns a value in [min, max) across many invocations
+ returns min when max == min + 1 (singleton range)
+ produces all values in small ranges over a large sample (uniform distribution)
- returns error when min >= max (empty range)
- returns error when the range width exceeds i64 maximum (e.g. min = i64::MIN, max = i64::MAX)

## Assumptions

? draws are independent across calls; no state leak between invocations
? distribution is approximately uniform; bias below 1 ULP is tolerated
