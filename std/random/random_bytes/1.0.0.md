# std.random.random_bytes

Returns a buffer of cryptographically strong pseudo-random bytes of the requested length.

## Signature

(length: i32) -> result[bytes, string]

## Behavior

Fills a freshly allocated buffer of size `length` with cryptographically strong pseudo-random bytes drawn from the host OS entropy source. Appropriate for use as key material, IVs, salts, and tokens.

## Tests

+ returns a buffer of exactly `length` bytes for length > 0
+ returns an empty buffer for length == 0
+ produces byte sequences that do not repeat across calls within a reasonable sample
- returns error when length is negative
- returns error when the host entropy source is unavailable

## Assumptions

? entropy source is the OS CSPRNG (e.g. getrandom on Linux, BCryptGenRandom on Windows)
? callers treat the result as sensitive and do not log it
