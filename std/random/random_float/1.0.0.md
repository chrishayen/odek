# std.random.random_float

Returns a pseudo-random double-precision float uniformly distributed in [0.0, 1.0).

## Signature

() -> f64

## Behavior

Draws a double-precision value where `0.0 <= result < 1.0`. The upper bound is exclusive. Uses a cryptographically strong or system-provided generator so that the result is not predictable between calls.

## Tests

+ returns a value in [0.0, 1.0) across many invocations
+ produces values uniformly distributed across the 53-bit significand over a large sample
+ never returns exactly 1.0
- never returns NaN or any infinity
- never returns a negative value

## Assumptions

? uses 53 bits of significand randomness
? does not guarantee returning every representable float in the range — only uniform distribution over the 2^53 evenly-spaced doubles
