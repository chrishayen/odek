# std.base64.encode

Encodes a byte buffer as a standard base64 string with padding.

## Signature

(data: bytes) -> string

## Behavior

Encodes bytes using the standard base64 alphabet (`A-Z a-z 0-9 + /`). The output length is always a multiple of 4; unused trailing bytes are padded with `=`. An empty input produces an empty output.

## Tests

+ returns "" for empty input
+ returns "Zg==" for input {0x66}
+ returns "Zm8=" for input {0x66, 0x6f}
+ returns "Zm9v" for input {0x66, 0x6f, 0x6f}
+ pads the final 4-char block with "=" so output length is a multiple of 4
- emits no line breaks in the output

## Assumptions

? alphabet is the standard base64 set (RFC 4648 section 4); use url-safe variant for URL contexts
? padding is always emitted; unpadded variant would be a separate function
