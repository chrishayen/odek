# std.base64.encode.encode_triplet

Encodes three input bytes as four base64 alphabet characters.

## Signature

(a: u8, b: u8, c: u8) -> string

## Behavior

Combines the three bytes into a 24-bit group, splits it into four 6-bit indices, and maps each index to a character from the standard base64 alphabet (`A-Z a-z 0-9 + /`).

## Tests

+ returns "AAAA" for bytes (0, 0, 0)
+ returns "////" for bytes (0xFF, 0xFF, 0xFF)
+ returns "Zm9v" for bytes (0x66, 0x6F, 0x6F) ("foo")
- returns no output character outside the standard base64 alphabet

## Assumptions

? alphabet is the standard (non-URL) variant per RFC 4648 section 4
