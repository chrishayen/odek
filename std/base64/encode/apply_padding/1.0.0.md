# std.base64.encode.apply_padding

Replaces trailing alphabet characters with `=` so the output length is a multiple of 4.

## Signature

(encoded: string, input_len: i64) -> string

## Behavior

When `input_len % 3` is 1, replaces the last two encoded characters with `==`. When it is 2, replaces the last one with `=`. When it is 0, returns `encoded` unchanged.

## Tests

+ returns the input unchanged when input_len is a multiple of 3
+ replaces one trailing character with "=" when input_len % 3 == 2
+ replaces two trailing characters with "==" when input_len % 3 == 1
- never adds more than two "=" characters

## Assumptions

? output length is always a multiple of 4
