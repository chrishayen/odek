# std.base64.decode.decode_quartet

Decodes four base64 alphabet characters into three bytes.

## Signature

(a: u8, b: u8, c: u8, d: u8) -> bytes

## Behavior

Calls `decode_char` on each input, combines the four 6-bit values into a 24-bit group, then splits the group into three bytes in order.

## Tests

+ returns (0, 0, 0) for input "AAAA"
+ returns (0xFF, 0xFF, 0xFF) for input "////"
+ returns ("f", "o", "o") for input "Zm9v"
- returns exactly three bytes for every valid input

## Assumptions

? padding ("=") in the last group is handled by a separate wrapper, not here
