# std.base64.decode

Decodes a standard base64-encoded string back into its original byte buffer.

## Signature

(s: string) -> result[bytes, string]

## Behavior

Decodes bytes from the standard base64 alphabet. Accepts padded input (length multiple of 4). Rejects unpadded input. Ignores no characters — whitespace embedded in the input is treated as invalid.

## Tests

+ returns empty bytes for empty input
+ returns {0x66} for input "Zg=="
+ returns {0x66, 0x6f} for input "Zm8="
+ returns {0x66, 0x6f, 0x6f} for input "Zm9v"
- returns error when the input length is not a multiple of 4
- returns error on characters outside the base64 alphabet
- returns error on padding in the middle of the input

## Assumptions

? alphabet is standard base64 (RFC 4648 section 4)
? whitespace is NOT skipped; callers that need whitespace tolerance must pre-strip
