# std.base64.decode.decode_char

Maps one base64 alphabet character to its 6-bit index value.

## Signature

(c: u8) -> result[u8, string]

## Behavior

Returns 0-25 for `A-Z`, 26-51 for `a-z`, 52-61 for `0-9`, 62 for `+`, and 63 for `/`. Returns error for any other byte, including `=` (which is handled at the group boundary by the caller).

## Tests

+ returns 0 for "A"
+ returns 25 for "Z"
+ returns 26 for "a"
+ returns 52 for "0"
