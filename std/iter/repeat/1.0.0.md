# std.iter.repeat

Returns an iterator that yields the given value `count` times.

## Signature

(value: T, count: i64) -> iter[T]

## Behavior

Yields `value` exactly `count` times, then terminates. Returns an empty iterator when `count` is zero or negative.

## Tests

+ yields value exactly count times for positive count
+ yields an empty iterator when count is 0
+ yields an empty iterator when count is negative
+ always yields the same value on each yield
- yields no value other than the input

## Assumptions

? this is the bounded variant; an infinite repeat is a separate rune
