# std.iter.drop_while

Returns an iterator that skips elements as long as a predicate holds, yielding the remainder starting with the first failure.

## Signature

(xs: iter[T], pred: func(T) -> bool) -> iter[T]

## Behavior

Skips leading elements while `pred(x)` is true. From the first element where `pred` returns false (inclusive), yields every remaining element without further filtering.

## Tests

+ yields elements starting at the first pred-false element
+ yields the whole input when pred is false for the first element
+ yields an empty iterator when pred is always true
+ yields an empty iterator when the input is empty
- does not re-test pred after the first false; subsequent trues still pass through

## Assumptions

? once drop_while stops filtering, no element is skipped regardless of pred
