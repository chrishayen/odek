# std.iter.count

Consumes an iterator completely and returns the number of elements yielded.

## Signature

(xs: iter[T]) -> i64

## Behavior

Advances the iterator to exhaustion, incrementing a counter for each element. Returns the final count.

## Tests

+ returns 0 for an empty iterator
+ returns the input's length for a finite iterator
+ consumes every element
- returns a negative value under no circumstance

## Assumptions

? for infinite iterators, this diverges; callers combine with take first
