# std.iter.any

Returns true if any element of an iterator satisfies a predicate, false otherwise.

## Signature

(xs: iter[T], pred: func(T) -> bool) -> bool

## Behavior

Short-circuits: returns true on the first element where `pred(x)` is true. Returns false when the iterator is exhausted without a true result.

## Tests

+ returns true when at least one element satisfies pred
+ returns false when no element satisfies pred
+ returns false for an empty iterator
+ stops advancing after the first true
- returns true for an empty iterator under no circumstance

## Assumptions

? short-circuits; elements after the first true are not evaluated
