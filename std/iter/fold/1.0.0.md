# std.iter.fold

Consumes an iterator left-to-right, threading an accumulator through `f`, and returns the final accumulator value.

## Signature

(xs: iter[T], initial: U, f: func(U, T) -> U) -> U

## Behavior

Starts with `acc = initial`. For each element `x` of the iterator, updates `acc = f(acc, x)`. Returns `acc` after the iterator is exhausted.

## Tests

+ returns initial when the iterator is empty
+ returns f(initial, x) for a single-element iterator
+ folds left-to-right: f is called with elements in input order
+ sums [1, 2, 3] with initial 0 and f = add to 6
- never evaluates f right-to-left (distinct from fold_right)

## Assumptions

? traversal order is left-to-right
? f is called exactly once per element
