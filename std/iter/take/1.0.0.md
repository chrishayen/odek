# std.iter.take

Returns an iterator that yields at most the first `n` elements of the input.

## Signature

(xs: iter[T], n: i64) -> iter[T]

## Behavior

Yields `xs[0]`, `xs[1]`, ..., `xs[n-1]`. Stops when `n` elements have been yielded or the input is exhausted, whichever comes first.

## Tests

+ yields the first 3 elements for (xs of length >= 3, 3)
+ yields the entire input when n exceeds its length
+ yields an empty iterator when n is 0
+ yields an empty iterator when n is negative

## Assumptions

? negative n clamps to 0 rather than raising an error
