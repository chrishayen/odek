# std.iter.reduce

Consumes an iterator left-to-right, combining elements pairwise with `f`, and returns the result (or none on empty input).

## Signature

(xs: iter[T], f: func(T, T) -> T) -> optional[T]

## Behavior

Takes the first element as the initial accumulator. For each subsequent element, updates `acc = f(acc, x)`. Returns `some(acc)` after the iterator is exhausted, or `none` when the input has no elements.

## Tests

+ returns none for an empty iterator
+ returns some(x) for a single-element iterator
+ combines left-to-right: reduce([1,2,3], add) returns some(6)
- returns no value for an empty input

## Assumptions

? differs from fold by not requiring a seed; empty input returns none
