# std.iter.filter

Returns an iterator that yields only the elements for which a predicate returns true.

## Signature

(xs: iter[T], pred: func(T) -> bool) -> iter[T]

## Behavior

Advances the input; yields each element for which `pred(x)` is true, skipping those for which it is false. Preserves input order. The returned iterator's length is 0 <= n <= input length.

## Tests

+ yields only elements for which pred is true
+ yields an empty iterator when pred is false for every input element
+ yields the entire input when pred is true for every element
+ preserves the order of surviving elements
- yields no element for which pred returned false

## Assumptions

? pred is called exactly once per input element
