# std.iter.drop

Returns an iterator that skips the first `n` elements and yields the rest.

## Signature

(xs: iter[T], n: i64) -> iter[T]

## Behavior

Advances the input past its first `n` elements, then yields every remaining element in order. When `n` exceeds the input's length, yields nothing.

## Tests

+ yields elements from index n onward when n is within the input's length
+ yields an empty iterator when n exceeds the input's length
+ yields the whole input when n is 0
+ yields the whole input when n is negative
- yields no element whose original index is below n

## Assumptions

? negative n clamps to 0 rather than raising an error
