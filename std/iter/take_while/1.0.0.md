# std.iter.take_while

Returns an iterator that yields elements from the input as long as a predicate holds, stopping at the first failure.

## Signature

(xs: iter[T], pred: func(T) -> bool) -> iter[T]

## Behavior

Yields `xs[0]`, `xs[1]`, ... each passed through `pred`. Stops (and does not yield) at the first element for which `pred` returns false.

## Tests

+ yields every element while pred returns true
+ stops at the first element where pred returns false
+ yields an empty iterator when pred is false for the first element
+ yields the whole input when pred is always true
- yields no element after the first false result

## Assumptions

? the stopping element is consumed from the underlying iterator but not yielded
