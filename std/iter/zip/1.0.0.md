# std.iter.zip

Returns an iterator that pairs corresponding elements of two iterators, stopping at the shorter one.

## Signature

(xs: iter[A], ys: iter[B]) -> iter[pair[A, B]]

## Behavior

Yields `(xs[0], ys[0])`, `(xs[1], ys[1])`, ... until either input is exhausted. Never continues past the shorter input; the remaining elements of the longer input are discarded.

## Tests

+ yields (1, "a"), (2, "b") for inputs [1,2] and ["a","b"]
+ stops at the shorter input when lengths differ
+ yields an empty iterator when either input is empty
- yields no element whose index exceeds the shorter input's length

## Assumptions

? fairness: even when one iterator has side effects, the longer one is not advanced past the stopping point
