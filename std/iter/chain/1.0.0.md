# std.iter.chain

Returns an iterator that yields every element of one iterator followed by every element of another.

## Signature

(xs: iter[T], ys: iter[T]) -> iter[T]

## Behavior

Drives `xs` to exhaustion, then drives `ys` to exhaustion, yielding every element in order. Does not advance `ys` until `xs` is done.

## Tests

+ yields 1, 2, 3, 4 for inputs [1, 2] and [3, 4]
+ yields the contents of ys when xs is empty
+ yields the contents of xs when ys is empty
+ yields an empty iterator when both inputs are empty
- does not interleave elements from ys before xs is exhausted

## Assumptions

? lazy: elements are consumed on demand
