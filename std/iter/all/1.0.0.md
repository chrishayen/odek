# std.iter.all

Returns true if every element of an iterator satisfies a predicate, false otherwise.

## Signature

(xs: iter[T], pred: func(T) -> bool) -> bool

## Behavior

Short-circuits: returns false on the first element where `pred(x)` is false. Returns true when the iterator is exhausted without a false result. An empty iterator returns true (vacuously).

## Tests

+ returns true when every element satisfies pred
+ returns false when any element fails pred
+ returns true for an empty iterator (vacuous truth)
+ stops advancing after the first false
- returns false for an empty iterator under no circumstance

## Assumptions

? vacuous truth is intentional and matches standard set theory / most stdlibs
