# std.iter.range

Returns an iterator over integers from `start` (inclusive) to `end` (exclusive) with the given step.

## Signature

(start: i64, end: i64, step: i64) -> iter[i64]

## Behavior

Yields `start`, `start + step`, ..., stopping at the first value that reaches or passes `end`. When `step` is positive the range is `[start, end)`; when negative it iterates downward and stops when the value is at or below `end`.

## Tests

+ yields 0, 1, 2 for (0, 3, 1)
+ yields 0, 2, 4 for (0, 5, 2)
+ yields 3, 2, 1 for (3, 0, -1)
+ yields an empty iterator when start equals end
+ yields an empty iterator when step is positive but start > end
- returns error (or empty) when step is zero

## Assumptions

? step of zero is treated as empty iterator (no error); callers can check explicitly
