# std.iter.find

Returns the first element of an iterator for which a predicate returns true, or none if no element matches.

## Signature

(xs: iter[T], pred: func(T) -> bool) -> optional[T]

## Behavior

Advances the input; returns `some(x)` on the first element where `pred(x)` is true. Returns `none` when the iterator is exhausted without a match. Short-circuits: does not evaluate `pred` on elements after the match.

## Tests

+ returns some(x) for the first matching element
+ returns none when no element matches
+ returns none for an empty iterator
+ stops advancing after the first match
- does not evaluate pred on elements after the match

## Assumptions

? the returned element is consumed from the underlying iterator
