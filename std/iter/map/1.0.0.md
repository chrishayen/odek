# std.iter.map

Returns an iterator that yields `f(x)` for each element of the input.

## Signature

(xs: iter[T], f: func(T) -> U) -> iter[U]

## Behavior

Lazily applies `f` to each element as the returned iterator is consumed. Preserves element order and count.

## Tests

+ yields f(x) in order for each element of xs
+ yields an empty iterator when the input is empty
+ preserves the input's length
+ is lazy: f is not called until the returned iterator is advanced
- yields no extra elements beyond the input's length

## Assumptions

? f is called exactly once per yielded element; caching is the caller's responsibility
