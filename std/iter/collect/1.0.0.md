# std.iter.collect

Consumes an iterator completely and returns its elements as a list in order.

## Signature

(xs: iter[T]) -> list[T]

## Behavior

Advances the iterator to exhaustion, pushing each yielded element onto a list. Returns the list. For an infinite iterator, this never returns.

## Tests

+ returns an empty list for an empty iterator
+ returns the elements in the same order they were yielded
+ returns the correct element count for a finite iterator
- returns no list with duplicate elements unless the iterator yielded duplicates

## Assumptions

? for infinite iterators, this diverges; combine with take first
