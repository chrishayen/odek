# std.iter.enumerate

Returns an iterator that pairs each element of the input with its zero-based index.

## Signature

(xs: iter[T]) -> iter[pair[i64, T]]

## Behavior

Yields `(0, xs[0])`, `(1, xs[1])`, ... until the input is exhausted.

## Tests

+ yields (0, "a"), (1, "b"), (2, "c") for input ["a", "b", "c"]
+ yields an empty iterator when the input is empty
+ preserves the input's element order
- yields no index greater than the input's length

## Assumptions

? index starts at 0; a `start` variant is a separate rune
