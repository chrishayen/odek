# std.hex.encode

Encodes a byte buffer as a lowercase hexadecimal string.

## Signature

(data: bytes) -> string

## Behavior

Produces two lowercase hex digits (`0-9`, `a-f`) for each input byte. The output is always exactly `2 * len(data)` characters. An empty input produces the empty string.

## Tests

+ returns "" for empty input
+ returns "00" for input {0x00}
+ returns "ff" for input {0xff}
+ returns "deadbeef" for input {0xde, 0xad, 0xbe, 0xef}
+ always emits lowercase hex digits
- emits no separator between bytes

## Assumptions

? always emits lowercase; use a separate function or case conversion for uppercase output
