# std.hex.decode

Decodes a hexadecimal string into its byte buffer.

## Signature

(s: string) -> result[bytes, string]

## Behavior

Reads pairs of hex digits left-to-right and emits one byte per pair. Accepts both lowercase and uppercase digits. Rejects inputs with an odd length.

## Tests

+ returns empty bytes for empty input
+ returns {0x00} for input "00"
+ returns {0xff} for input "ff"
+ returns {0xde, 0xad, 0xbe, 0xef} for input "deadbeef"
+ accepts uppercase hex digits ("DEADBEEF" -> same)
- returns error when the input length is odd
- returns error on a non-hex character
- returns error on a separator like ":" or "-"

## Assumptions

? no separators are tolerated; pre-strip them if needed
