# std.hex.decode.pair_to_byte

Decodes a pair of hex characters into one byte.

## Signature

(hi: u8, lo: u8) -> result[u8, string]

## Behavior

Calls `hex_char_to_nibble(hi)` and `hex_char_to_nibble(lo)`. Returns `(hi_nibble << 4) | lo_nibble`.

## Tests

+ returns 0x00 for input ("0", "0")
+ returns 0xFF for input ("f", "f")
+ returns 0xDE for input ("d", "e")
- returns error when either character is not a valid hex digit

## Assumptions

? both characters must be valid hex; mixed case is tolerated
