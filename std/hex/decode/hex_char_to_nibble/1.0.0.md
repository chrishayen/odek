# std.hex.decode.hex_char_to_nibble

Converts one ASCII hex character to its 4-bit nibble value.

## Signature

(c: u8) -> result[u8, string]

## Behavior

Returns 0-9 for `'0'-'9'`, 10-15 for `'a'-'f'`, 10-15 for `'A'-'F'`. Returns error for any other byte.

## Tests

+ returns 0 for "0" and 9 for "9"
+ returns 10 for "a" and "A"
+ returns 15 for "f" and "F"
- returns error for "g"
- returns error for "G"
- returns error for space

## Assumptions

? accepts both lowercase and uppercase hex letters
