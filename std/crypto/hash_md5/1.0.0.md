# std.crypto.hash_md5

Computes the MD5 digest of a byte buffer (16 bytes). Legacy; not cryptographically secure.

## Signature

(data: bytes) -> bytes

## Behavior

Returns a 16-byte digest per RFC 1321. MD5 is broken for collision resistance; provided only for legacy interop (e.g. ETag computation, checksum of known-non-adversarial data).

## Tests

+ returns 16 bytes for any input
+ returns d41d8cd98f00b204e9800998ecf8427e for empty input
+ produces a deterministic digest for the same input
- not suitable for security-sensitive purposes

## Assumptions

? included for interop only; new code should use SHA-256 or SHA-3
