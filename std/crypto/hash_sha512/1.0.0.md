# std.crypto.hash_sha512

Computes the SHA-512 digest of a byte buffer (64 bytes).

## Signature

(data: bytes) -> bytes

## Behavior

Returns a 64-byte digest per FIPS 180-4.

## Tests

+ returns 64 bytes for any input
+ returns the canonical empty-input digest for empty input
+ produces a deterministic digest for the same input
- never returns fewer than 64 bytes

## Assumptions

? output is 64 bytes (512 bits)
