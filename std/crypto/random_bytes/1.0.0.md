# std.crypto.random_bytes

Returns a buffer of cryptographically strong random bytes suitable for key material.

## Signature

(length: i32) -> result[bytes, string]

## Behavior

Fills a freshly allocated buffer of size `length` with bytes drawn from the host OS cryptographically secure random source. Suitable for keys, IVs, nonces, tokens, and salts.

## Tests

+ returns a buffer of exactly `length` bytes for length > 0
+ returns an empty buffer for length == 0
+ produces byte sequences that do not repeat across calls within a reasonable sample
+ passes standard statistical randomness tests (monobit, runs)
- returns error when length is negative
- returns error when the host entropy source is unavailable

## Assumptions

? entropy source is the OS CSPRNG; never a non-crypto PRNG
? distinct from std.random.random_bytes in intent (key material vs statistical draws) even if implementation overlaps
