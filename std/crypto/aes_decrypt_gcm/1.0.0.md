# std.crypto.aes_decrypt_gcm

Decrypts and authenticates an AES-GCM ciphertext.

## Signature

(key: bytes, nonce: bytes, ciphertext_and_tag: bytes, associated_data: bytes) -> result[bytes, string]

## Behavior

Expects ciphertext concatenated with the 16-byte GCM tag as the last 16 bytes. Verifies the tag in constant time before returning plaintext. Returns an error on authentication failure (without revealing partial plaintext).

## Tests

+ returns plaintext for a valid ciphertext+tag
+ verifies associated_data matches what was used during encryption
- returns error when the tag fails verification
- returns error when key length is not 16/24/32
- returns error when nonce is not exactly 12 bytes
- returns error when ciphertext+tag is shorter than 16 bytes

## Assumptions

? must not return partial plaintext on tag failure
? constant-time tag compare
