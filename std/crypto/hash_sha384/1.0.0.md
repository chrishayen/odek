# std.crypto.hash_sha384

Computes the SHA-384 digest of a byte buffer (48 bytes).

## Signature

(data: bytes) -> bytes

## Behavior

Returns a 48-byte digest per FIPS 180-4. SHA-384 is a truncation of SHA-512 with a different IV, offering a middle ground between SHA-256 and SHA-512.

## Tests

+ returns 48 bytes for any input
+ returns the canonical empty-input digest for empty input
+ produces a deterministic digest
- never returns fewer than 48 bytes

## Assumptions

? output is 48 bytes (384 bits)
