# std.crypto.sign_ed25519

Signs a message with an Ed25519 private key, producing a 64-byte signature.

## Signature

(private_key: bytes, message: bytes) -> result[bytes, string]

## Behavior

Requires a 32-byte private key per RFC 8032. Produces a 64-byte signature over the entire message. Signing is deterministic — the same (key, message) always produces the same signature.

## Tests

+ returns 64 bytes for any input
+ produces the same signature for the same (key, message) pair
+ different messages produce different signatures
- returns error when private_key is not exactly 32 bytes

## Assumptions

? Ed25519 signing is deterministic (no RNG needed)
