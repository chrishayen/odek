# std.crypto.verify_hmac_sha256

Verifies an HMAC-SHA-256 tag against a byte buffer under a secret key, in constant time.

## Signature

(key: bytes, data: bytes, tag: bytes) -> bool

## Behavior

Recomputes the HMAC-SHA-256 of `data` under `key` and compares the result against `tag` using a constant-time byte comparison. Returns true when the computed and supplied tags are equal, false otherwise. Tags of the wrong length always return false.

## Tests

+ returns true when tag matches the computed HMAC-SHA-256
+ returns false when the data differs from what was signed
+ returns false when the key differs from what was used to sign
+ returns false when the supplied tag is not exactly 32 bytes
- returns false in constant time with respect to which byte mismatches

## Assumptions

? comparison is constant-time; timing does not leak which byte mismatched
? accepts only 32-byte tags; no truncated-tag support
