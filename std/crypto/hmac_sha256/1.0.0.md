# std.crypto.hmac_sha256

Computes an HMAC-SHA-256 tag over a byte buffer under a secret key.

## Signature

(key: bytes, data: bytes) -> bytes

## Behavior

Produces a 32-byte message authentication code per RFC 2104 using SHA-256 as the underlying hash. Keys longer than the hash block size (64 bytes) are pre-hashed; shorter keys are zero-padded.

## Tests

+ returns 32 bytes for any input
+ returns the RFC 4231 test vector 1 tag for (key={0x0b * 20}, data="Hi There")
+ returns a deterministic tag for the same (key, data) pair
+ produces different tags for data that differs by a single bit
+ handles keys longer than the block size by pre-hashing
- emits no bits that depend on the key through timing side channels

## Assumptions

? hash function is SHA-256 (produces 32-byte tag)
? caller compares tags using constant-time compare, not byte equality
