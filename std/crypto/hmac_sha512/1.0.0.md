# std.crypto.hmac_sha512

Computes an HMAC-SHA-512 tag (64 bytes).

## Signature

(key: bytes, data: bytes) -> bytes

## Behavior

Returns a 64-byte tag per RFC 2104 with SHA-512 as the hash.

## Tests

+ returns 64 bytes for any input
+ produces deterministic output for the same (key, data)
+ different keys produce different tags
- never returns fewer than 64 bytes

## Assumptions

? output is 64 bytes
