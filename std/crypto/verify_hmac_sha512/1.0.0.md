# std.crypto.verify_hmac_sha512

Verifies an HMAC-SHA-512 tag in constant time.

## Signature

(key: bytes, data: bytes, tag: bytes) -> bool

## Behavior

Recomputes HMAC-SHA-512 of `data` under `key` and constant-time-compares against `tag`.

## Tests

+ returns true when tag matches
+ returns false when any input differs
+ returns false when tag is not exactly 64 bytes
- returns false in constant time

## Assumptions

? tag must be 64 bytes exactly; truncated tags are a separate primitive
