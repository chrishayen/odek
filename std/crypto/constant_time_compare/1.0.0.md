# std.crypto.constant_time_compare

Compares two byte buffers for equality in time independent of the position of the first mismatch.

## Signature

(a: bytes, b: bytes) -> bool

## Behavior

Returns true iff `a` and `b` have the same length and every byte is equal. The comparison loop visits every byte of the shorter buffer regardless of where the first difference occurs, so execution time does not leak information about the content.

## Tests

+ returns true for identical byte slices
+ returns true for two empty slices
+ returns false for slices of different lengths
+ returns false for slices of the same length differing in one byte
+ executes in time proportional to the input length, independent of where mismatches lie
- returns true only when lengths match and every byte matches

## Assumptions

? length mismatch is an early-return false; the timing leak of length is considered acceptable
? the function assumes both inputs are already in memory; no streaming variant
