# std.crypto.aes_encrypt_gcm

Encrypts plaintext under AES-GCM with the given key and nonce, returning ciphertext concatenated with the 16-byte authentication tag.

## Signature

(key: bytes, nonce: bytes, plaintext: bytes, associated_data: bytes) -> result[bytes, string]

## Behavior

Requires a 16, 24, or 32-byte key (AES-128/192/256). Requires a 12-byte nonce. Produces ciphertext of length `len(plaintext) + 16` (16-byte GCM tag appended). Authenticates both plaintext and `associated_data`; the associated data is not encrypted.

## Tests

+ returns ciphertext of length plaintext + 16
+ accepts associated_data of any length including empty
+ accepts key lengths 16, 24, 32
- returns error when key length is not 16/24/32
- returns error when nonce is not exactly 12 bytes
- returns error when plaintext is 2^39-256 bytes or larger (GCM limit)

## Assumptions

? nonce must be unique per (key, message); nonce reuse destroys confidentiality
? tag is appended to ciphertext, not returned separately
