# std.crypto.verify_ed25519

Verifies an Ed25519 signature against a message and public key.

## Signature

(public_key: bytes, message: bytes, signature: bytes) -> bool

## Behavior

Requires a 32-byte public key and a 64-byte signature. Returns true iff the signature is a valid Ed25519 signature for `message` under `public_key`. Returns false for any length mismatch or verification failure.

## Tests

+ returns true for a valid signature from the corresponding private key
+ returns false for a signature of a different message
+ returns false for a signature with any byte flipped
+ returns false when public_key is not exactly 32 bytes
+ returns false when signature is not exactly 64 bytes

## Assumptions

? checks are constant-time on the signature bytes
