# std.crypto.hash_sha1

Computes the SHA-1 digest of a byte buffer (20 bytes).

## Signature

(data: bytes) -> bytes

## Behavior

Returns a 20-byte digest per FIPS 180-4. SHA-1 is cryptographically broken for collision resistance; use only for legacy compatibility, not new security constructs.

## Tests

+ returns 20 bytes for any input
+ returns da39a3ee5e6b4b0d3255bfef95601890afd80709 for empty input
+ produces different digests for inputs differing by one bit
- not suitable for collision-resistant cryptography (documented)

## Assumptions

? provided for legacy compatibility (Git objects, TLS hash variants, etc.)
