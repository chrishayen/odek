# std.crypto.verify_hmac_sha1

Verifies an HMAC-SHA-1 tag in constant time.

## Signature

(key: bytes, data: bytes, tag: bytes) -> bool

## Behavior

Recomputes HMAC-SHA-1 of `data` under `key` and constant-time-compares against `tag`. Returns false when lengths differ or any bit differs.

## Tests

+ returns true when tag matches the computed HMAC
+ returns false when data, key, or tag differs
+ returns false when tag is not exactly 20 bytes
- returns false in constant time

## Assumptions

? legacy; prefer HMAC-SHA-256 for new constructs
