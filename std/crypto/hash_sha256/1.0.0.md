# std.crypto.hash_sha256

Computes the SHA-256 digest of a byte buffer.

## Signature

(data: bytes) -> bytes

## Behavior

Returns a 32-byte digest produced by the SHA-256 algorithm (FIPS 180-4) over the input bytes. Empty input produces the well-known digest `e3b0c44298fc1c149afbf4c8996fb92427ae41e4649b934ca495991b7852b855`.

## Tests

+ returns 32 bytes for any input
+ returns the SHA-256 of the empty byte string for empty input
+ returns a deterministic digest for the same input across invocations
+ produces different digests for inputs that differ by a single bit
+ handles large inputs (multi-megabyte) without error
- emits no side channels through timing; constant-time per byte

## Assumptions

? output length is always exactly 32 bytes
? caller converts the digest to hex or base64 as needed
