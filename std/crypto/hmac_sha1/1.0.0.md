# std.crypto.hmac_sha1

Computes an HMAC-SHA-1 tag (20 bytes). Legacy; prefer HMAC-SHA-256 for new designs.

## Signature

(key: bytes, data: bytes) -> bytes

## Behavior

Returns a 20-byte tag per RFC 2104 with SHA-1 as the hash.

## Tests

+ returns 20 bytes for any input
+ matches the RFC 2202 test vector 1 tag
+ produces deterministic output for the same (key, data)
- never returns fewer than 20 bytes

## Assumptions

? provided for legacy interop (OAuth 1.0a, AWS Signature V2, etc.)
