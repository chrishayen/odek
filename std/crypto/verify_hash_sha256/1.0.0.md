# std.crypto.verify_hash_sha256

Recomputes the SHA-256 of data and compares against an expected digest in constant time.

## Signature

(data: bytes, expected_digest: bytes) -> bool

## Behavior

Returns true iff the expected digest is exactly 32 bytes AND matches the computed digest bit-for-bit. Uses constant-time compare to avoid timing leaks.

## Tests

+ returns true when expected matches the computed digest
+ returns false when data differs
+ returns false when expected is not exactly 32 bytes
- returns false in constant time with respect to mismatch position

## Assumptions

? comparison is constant-time with respect to bit positions
