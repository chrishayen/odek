# std.url.format.format_path_query_fragment

Builds the path, query, and fragment tail of a URL.

## Signature

(u: url) -> string

## Behavior

Emits the path verbatim. Appends `?` + query when the query is present. Appends `#` + fragment when the fragment is present. Returns the concatenated tail. Components absent from the URL are omitted entirely.

## Tests

+ emits the path only when query and fragment are absent
+ emits "?query" immediately after the path when present
+ emits "#fragment" last
+ emits "" when path, query, and fragment are all empty/absent
- omits "?" when the query component is absent

## Assumptions

? none of the components are re-encoded; they are emitted as stored
