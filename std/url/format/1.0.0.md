# std.url.format

Re-assembles a url value into its canonical string form.

## Signature

(u: url) -> string

## Behavior

Concatenates the components in canonical order: `scheme://[userinfo@]host[:port][path][?query][#fragment]`. Components that were not present on the input url are omitted. The path is emitted with its leading slash when the url has an authority. Port is omitted when it matches the scheme default (80 for http, 443 for https).

## Tests

+ formats the components back into the original string for a simple URL
+ omits the port when equal to the scheme's default
+ omits the query when empty
+ omits the fragment when empty
+ round-trips through parse -> format for any canonically-formed URL
- emits no "?" when the query component is empty

## Assumptions

? scheme defaults table covers http (80) and https (443) only
? percent-encoding already present on the structured components is preserved verbatim
