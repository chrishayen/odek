# std.url.format.format_authority

Builds the `//[userinfo@]host[:port]` authority section of a URL.

## Signature

(u: url) -> string

## Behavior

Emits `//`. If userinfo is present, emits `user[:password]@`. Emits the host verbatim. Emits `:port` only when the port is present and not equal to the scheme default (80 for http, 443 for https). Returns the assembled authority.

## Tests

+ emits "//example.com" when host only
+ emits "//user@example.com" when userinfo has no password
+ emits "//user:pass@example.com:8080" when all parts are present
+ omits the port when it equals the scheme default
- emits no characters before the leading "//"

## Assumptions

? scheme-default port table covers http and https only
? userinfo is emitted verbatim; percent-encoding is up to the caller
