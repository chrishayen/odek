# std.url.encode_component

Percent-encodes a string so it can be safely used as a single URL path or query component.

## Signature

(s: string) -> string

## Behavior

Encodes every byte outside the RFC 3986 `unreserved` set (`A-Z a-z 0-9 - _ . ~`) as `%XX` where `XX` is the uppercase hex of the byte value. The input is first UTF-8 encoded. Reserved characters like `/`, `?`, `#`, `&`, `=`, and space are all encoded, matching JavaScript's `encodeURIComponent` behavior.

## Tests

+ leaves "abc-_.~" unchanged
+ encodes space as "%20" (not "+")
+ encodes "&" as "%26"
+ encodes "/" as "%2F"
+ encodes non-ASCII characters via their UTF-8 byte sequences (e.g. "é" -> "%C3%A9")
- emits uppercase hex digits, never lowercase

## Assumptions

? follows RFC 3986 unreserved set, matching encodeURIComponent exactly
? space is always encoded as %20, never as "+"
