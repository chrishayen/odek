# std.url.encode_component.is_unreserved

Returns true when a byte is in the RFC 3986 unreserved character set.

## Signature

(b: u8) -> bool

## Behavior

Returns true for bytes `A-Z`, `a-z`, `0-9`, `-`, `_`, `.`, or `~`. Returns false for everything else.

## Tests

+ returns true for every letter A-Z and a-z
+ returns true for every digit 0-9
+ returns true for "-", "_", ".", and "~"
- returns false for space
- returns false for "+"
- returns false for "/"

## Assumptions

? the unreserved set is strictly RFC 3986 (no additional characters)
