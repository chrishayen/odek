# std.url.encode_component.encode_byte

Encodes one byte as either itself (unreserved) or its `%XX` percent-escape.

## Signature

(b: u8) -> string

## Behavior

Calls `is_unreserved(b)`. Returns the byte as a one-character string when true; returns `%` followed by two uppercase hex digits when false.

## Tests

+ returns "A" for 0x41
+ returns "%20" for space (0x20)
+ returns "%2F" for "/"
+ always uses uppercase hex for the percent-escape
- never returns more than three characters

## Assumptions

? hex digits are always uppercase
