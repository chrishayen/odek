# std.url.decode_component

Decodes a percent-encoded URL component back into its original string.

## Signature

(s: string) -> result[string, string]

## Behavior

Scans the input for `%XX` escapes and replaces each with the corresponding byte. Other characters pass through unchanged. The resulting byte sequence is validated as UTF-8 and returned as a string.

## Tests

+ returns "hello world" for input "hello%20world"
+ returns "é" for input "%C3%A9"
+ accepts lowercase hex digits in percent escapes
+ returns the input unchanged when it contains no "%" bytes
- returns error on a truncated escape like "%2"
- returns error on invalid hex digits in an escape like "%ZZ"
- returns error when the decoded bytes are not valid UTF-8

## Assumptions

? decoded result must be valid UTF-8; use a byte-variant function for arbitrary binary payloads
? "+" is NOT decoded as space (this is url_decode_component, not form_decode)
