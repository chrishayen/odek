# std.url.decode_component.decode_escape

Decodes one `%XX` escape starting at a cursor and returns the byte value plus the position after the escape.

## Signature

(input: string, pos: i64) -> result[byte_result, string]

## Behavior

Expects `%` at `pos`. Reads the two following hex digits (case-insensitive) and returns the decoded byte and `pos + 3`.

## Tests

+ decodes "%20" at pos 0 to byte 0x20 and pos 3
+ decodes "%ff" and "%FF" identically (case-insensitive)
+ decodes "%C3" at pos 0 to byte 0xC3
- returns error when fewer than 3 bytes remain from pos
- returns error when the two characters after "%" are not hex

## Assumptions

? accepts both uppercase and lowercase hex
