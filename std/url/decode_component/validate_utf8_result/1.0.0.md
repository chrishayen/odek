# std.url.decode_component.validate_utf8_result

Validates that a decoded byte buffer is well-formed UTF-8 and returns it as a string.

## Signature

(bytes: bytes) -> result[string, string]

## Behavior

Calls `std.utf8.valid`; on true returns the buffer viewed as a string, on false returns an error. Used as the final step of component decoding to reject binary payloads.

## Tests

+ returns the input as a string when it is valid UTF-8
+ accepts an empty byte buffer as a valid (empty) string
- returns error when any byte sequence is not valid UTF-8
- returns error on an over-long UTF-8 encoding

## Assumptions

? delegates the validation policy entirely to std.utf8.valid
