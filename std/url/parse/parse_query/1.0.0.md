# std.url.parse.parse_query

Reads the query segment of a URL (the substring between `?` and `#` or end).

## Signature

(input: string, pos: i64) -> query_result

## Behavior

If the byte at `pos` is `?`, returns the substring from `pos + 1` up to but not including the first `#` or end of input. If the byte at `pos` is not `?`, returns none (no query present).

## Tests

+ returns some("q=1&r=2") for input "?q=1&r=2" at pos 0
+ returns some("") for input "?#frag" (empty query, has fragment)
+ returns none when pos is not at "?"
+ stops reading at the first "#"
- returns none for any input that does not begin with "?"

## Assumptions

? does not decode "+" as space (that is form-decoding, handled separately)
? key-value parsing of the query string is a separate rune
