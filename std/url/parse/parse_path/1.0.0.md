# std.url.parse.parse_path

Reads the path segment of a URL from a cursor up to the first `?` or `#`.

## Signature

(input: string, pos: i64) -> path_result

## Behavior

Returns the substring from `pos` up to but not including the first `?`, `#`, or end of input. Does not validate or decode percent-encoded octets — the raw bytes are preserved.

## Tests

+ returns "/a/b" for input "/a/b?q=1" at pos 0
+ returns "/a/b" for input "/a/b#frag" at pos 0
+ returns "/a/b" for input "/a/b" (end-of-input terminator)
+ returns "" when pos is already at a "?" or "#"
- returns no substring containing "?" or "#"

## Assumptions

? percent-encoded octets are left encoded
? empty path (e.g. "http://host?q=1") returns ""
