# std.url.parse.parse_port

Parses the port portion of a URL authority into a 16-bit integer, or none when absent.

## Signature

(token: string) -> result[optional[i32], string]

## Behavior

An empty input returns none. Otherwise parses the input as a decimal integer. Returns `some(n)` when the integer is in [0, 65535].

## Tests

+ returns none for an empty input
+ returns some(8080) for input "8080"
+ returns some(0) for input "0"
- returns error when the input contains non-digit characters
- returns error when the integer exceeds 65535

## Assumptions

? leading zeros are tolerated; "080" parses as 80
? negative values are not representable in the input format
