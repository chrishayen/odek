# std.url.parse.parse_scheme

Reads the scheme of a URL and returns it as a lowercase string plus the position after the trailing `:`.

## Signature

(input: string, pos: i64) -> result[scheme_result, string]

## Behavior

Scans from `pos` for a letter followed by any combination of letters, digits, `+`, `-`, or `.`, terminated by `:`. Returns the scheme lowercased and the position after the `:`.

## Tests

+ returns "https" and pos past ":" for input "https://..."
+ lowercases a mixed-case scheme like "HTTP"
+ accepts schemes containing "+", "-", and "." (e.g. "coap+tcp")
- returns error when the first character is not a letter
- returns error when no ":" is found

## Assumptions

? scheme is always lowercased in the result
? the `:` is consumed from the cursor; the caller sees `//` or path next
