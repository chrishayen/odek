# std.url.parse.parse_authority

Parses the authority component of a URL (optional userinfo, host, optional port).

## Signature

(input: string, pos: i64) -> result[authority_result, string]

## Behavior

Expects `//` at `pos` (consumes it). Splits the remaining authority substring at the next `/`, `?`, `#`, or end. Within that substring, optionally parses userinfo (delimited by `@`), host, and optional port (delimited by `:`). Returns an `authority_result` carrying each component.

## Tests

+ parses "example.com" into host "example.com" and no port
+ parses "user:pass@example.com:8080" into all three components
+ parses "[::1]:443" into an IPv6 host with brackets and port 443
- returns error when "//" is missing at pos
- returns error when host is empty

## Assumptions

? the authority terminator is the first "/", "?", "#", or end of input
? IPv6 addresses must be bracketed, per RFC 3986
