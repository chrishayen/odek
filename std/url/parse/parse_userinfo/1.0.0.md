# std.url.parse.parse_userinfo

Parses the userinfo portion of a URL authority (`user` or `user:pass`).

## Signature

(token: string) -> result[userinfo, string]

## Behavior

Accepts the substring preceding `@` in the authority. Splits on the first `:` into user and password. Returns both components; an absent password is represented as none. Validates that bytes belong to the RFC 3986 userinfo set.

## Tests

+ returns user "alice" for input "alice"
+ returns user "alice" and password "s3cret" for input "alice:s3cret"
+ treats colons after the first as part of the password
- returns error on an empty user component
- returns error when characters outside the userinfo set are present

## Assumptions

? percent-encoded octets are not decoded here; caller decodes later
? password-less form is stored with password = none
