# std.url.parse.parse_fragment

Reads the fragment segment of a URL (the substring after `#`).

## Signature

(input: string, pos: i64) -> fragment_result

## Behavior

If the byte at `pos` is `#`, returns the substring from `pos + 1` to end of input. If the byte at `pos` is not `#`, returns none.

## Tests

+ returns some("frag") for input "#frag" at pos 0
+ returns some("") for input "#" alone
+ returns none when pos is not at "#"
- returns none for any input that does not begin with "#"

## Assumptions

? fragment is always the final URL component
? percent-encoded octets are left encoded
