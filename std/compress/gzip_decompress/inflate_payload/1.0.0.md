# std.compress.gzip_decompress.inflate_payload

Inflates a raw deflate stream starting at a given offset and returns the uncompressed bytes plus end offset.

## Signature

(data: bytes, offset: i64) -> result[inflate_result, string]

## Behavior

Reads deflate-compressed bytes from `data` starting at `offset` until the final block is consumed. Returns the uncompressed output buffer and the offset of the first byte after the deflate stream (where the gzip trailer begins).

## Tests

+ returns the original bytes when given a deflate stream produced by deflate_payload
+ returns an empty buffer when the stream contains only an empty final block
+ advances the returned offset past the entire deflate stream
- returns error on a malformed deflate block type
- returns error on a truncated stream

## Assumptions

? supports all three deflate block types: stored, fixed Huffman, dynamic Huffman
? maximum output size is bounded only by available memory
