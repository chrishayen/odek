# std.compress.gzip_decompress.parse_header

Parses a gzip member header and returns the offset at which the deflate payload starts.

## Signature

(data: bytes) -> result[header_result, string]

## Behavior

Calls `verify_magic`. Reads the compression method byte (must be 8). Reads the flags byte and skips each optional field it enables: FEXTRA, FNAME, FCOMMENT, FHCRC. Returns the position after all header bytes.

## Tests

+ returns offset 10 for a minimal header with flags 0x00
+ skips FEXTRA data when the FEXTRA flag is set
+ skips FNAME up to a NUL terminator when FNAME is set
+ skips FHCRC's 2-byte hcrc when set
- returns error when the compression method byte is not 8
- returns error when reserved flag bits are set

## Assumptions

? reserved flag bits (0xE0) must be zero; nonzero is an error per RFC 1952
