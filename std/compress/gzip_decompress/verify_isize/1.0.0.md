# std.compress.gzip_decompress.verify_isize

Checks that the uncompressed data length matches the gzip trailer's ISIZE field (modulo 2^32).

## Signature

(data: bytes, expected: u32) -> result[void, string]

## Behavior

Returns success when `len(data) & 0xFFFFFFFF == expected`. Returns error otherwise.

## Tests

+ returns success when len(data) mod 2^32 equals expected
+ returns success for empty data when expected is 0
- returns error when len(data) mod 2^32 does not equal expected

## Assumptions

? ISIZE is stored modulo 2^32; buffers larger than 4GB intentionally wrap
