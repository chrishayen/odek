# std.compress.gzip_decompress.verify_magic

Verifies that a byte buffer starts with the two-byte gzip magic number.

## Signature

(data: bytes) -> result[void, string]

## Behavior

Checks that bytes 0 and 1 are `0x1F` and `0x8B`. Returns success on match, error otherwise.

## Tests

+ returns success for bytes starting with 0x1F 0x8B
- returns error for fewer than 2 bytes
- returns error when the first byte is not 0x1F
- returns error when the second byte is not 0x8B

## Assumptions

? magic is strictly the two bytes 0x1F 0x8B in that order
