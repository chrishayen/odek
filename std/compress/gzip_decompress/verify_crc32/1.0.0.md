# std.compress.gzip_decompress.verify_crc32

Recomputes the CRC-32 of an uncompressed buffer and compares it against an expected value.

## Signature

(data: bytes, expected: u32) -> result[void, string]

## Behavior

Computes `compute_crc32(data)` and returns success when the result equals `expected`. Returns error otherwise.

## Tests

+ returns success when the computed CRC matches expected
- returns error when the computed CRC does not match expected
- returns error when data is empty but expected is not 0

## Assumptions

? shares the CRC-32 implementation with the compressor for consistency
