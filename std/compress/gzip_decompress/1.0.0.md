# std.compress.gzip_decompress

Decompresses a gzip-formatted byte stream back into its original payload.

## Signature

(data: bytes) -> result[bytes, string]

## Behavior

Parses the RFC 1952 gzip framing and inflates the deflate-compressed payload. Verifies the trailing CRC32 and ISIZE fields against the decompressed data. Concatenates multiple gzip members into a single output buffer.

## Tests

+ returns the original bytes for any stream produced by gzip_compress
+ handles concatenated gzip members (e.g. gzip -c a b) by concatenating payloads
+ handles an empty (but valid) gzip member
- returns error on a stream with an invalid magic header (not 0x1f 0x8b)
- returns error when the trailing CRC32 does not match
- returns error on a truncated stream
- returns error on unknown or reserved header flags

## Assumptions

? only deflate compression method is accepted (CM=8)
? trailing ISIZE is validated mod 2^32
