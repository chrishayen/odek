# std.compress.gzip_compress.compute_crc32

Computes the CRC-32 checksum of a byte buffer using the IEEE polynomial.

## Signature

(data: bytes) -> u32

## Behavior

Computes CRC-32 with the IEEE 802.3 polynomial `0xEDB88320` (reflected), initial value `0xFFFFFFFF`, and final XOR `0xFFFFFFFF`. Returns the unsigned 32-bit result.

## Tests

+ returns 0x00000000 for an empty input
+ returns 0xCBF43926 for input "123456789"
+ returns a deterministic value for the same input on every call
+ produces different values for inputs that differ by a single bit
- returns no value greater than 0xFFFFFFFF

## Assumptions

? polynomial is the reflected IEEE variant, matching gzip and zlib
