# std.compress.gzip_compress

Compresses a byte buffer into a gzip-formatted stream.

## Signature

(data: bytes, level: i32) -> result[bytes, string]

## Behavior

Produces a gzip stream (RFC 1952) with a single member containing the deflate-compressed payload. `level` selects the deflate compression level: 0 is no compression, 1 is fastest, 9 is smallest. Levels below 0 or above 9 are clamped to the nearest valid value.

## Tests

+ returns a valid gzip stream for any input
+ produces a stream that round-trips through gzip_decompress to the original bytes
+ produces smaller output at level 9 than level 1 for compressible input
+ accepts level 0 (store-only) and still emits valid gzip framing
+ handles empty input and produces a valid minimal gzip stream
- returns error when level is explicitly invalid (e.g. level 10 is clamped, but NaN/overflow inputs error)

## Assumptions

? level is clamped to [0, 9]; arbitrary values never fail
? gzip header carries no filename or timestamp by default
