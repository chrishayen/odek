# std.compress.gzip_compress.write_header

Emits the fixed 10-byte gzip member header with no optional fields.

## Signature

() -> bytes

## Behavior

Returns the 10-byte sequence: magic `0x1F 0x8B`, compression method `0x08` (deflate), flags `0x00`, modification time `0x00 0x00 0x00 0x00`, extra-flags `0x00`, OS `0xFF` (unknown).

## Tests

+ returns exactly 10 bytes
+ starts with magic bytes 0x1F 0x8B
+ sets compression method to 0x08
+ sets flags to 0x00 (no optional fields)
- emits no filename, comment, or extra header fields

## Assumptions

? modification time is 0 (unspecified) to keep outputs deterministic
? OS byte is 0xFF (unknown); producers may override if they need identification
