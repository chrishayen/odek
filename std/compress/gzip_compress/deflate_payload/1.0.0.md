# std.compress.gzip_compress.deflate_payload

Compresses a byte buffer with raw deflate (RFC 1951), with no gzip framing.

## Signature

(data: bytes, level: i32) -> result[bytes, string]

## Behavior

Emits a raw deflate stream of `data` at the chosen compression level. Level is clamped to [0, 9]: 0 for stored blocks, 1 for fastest, 9 for smallest. Does not emit any header, trailer, or checksum.

## Tests

+ returns a deflate stream that round-trips through inflate to the original bytes
+ level 0 produces only stored (uncompressed) blocks
+ level 9 produces smaller output than level 1 on compressible input
+ returns a minimal empty-block sequence for empty input
- returns no gzip magic or trailer bytes

## Assumptions

? level is clamped to [0, 9]; out-of-range values are silently clamped
