# std.log.fatal

Emits a structured log record at the FATAL severity (highest).

## Signature

(message: string, fields: map[string, string]) -> void

## Behavior

Constructs a FATAL-level record with timestamp, message, and fields. Always emitted. FATAL typically precedes process termination; this rune does NOT itself terminate — the caller does.

## Tests

+ writes a record unconditionally (no severity filtering)
+ includes every supplied field
+ produces a record with severity "FATAL"
- does not terminate the process on its own

## Assumptions

? termination is the caller's responsibility; this is a log-only function
