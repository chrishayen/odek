# std.log.debug

Emits a structured log record at the DEBUG severity.

## Signature

(message: string, fields: map[string, string]) -> void

## Behavior

Constructs a structured log record with the current wall-clock timestamp, the severity DEBUG, the supplied message, and every field from `fields`. Writes the record to the process's logging sink. Discarded at runtime when the configured minimum severity is above DEBUG.

## Tests

+ writes a record containing the message when the configured level is DEBUG or lower
+ includes every supplied field verbatim
+ includes a wall-clock timestamp
+ produces a record with severity "DEBUG"
- discards the record when the configured level is INFO or higher

## Assumptions

? default sink is stderr; reconfigurable via a separate setup function
? record format is JSON lines by default
