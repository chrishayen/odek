# std.log.trace

Emits a structured log record at the TRACE severity (below DEBUG).

## Signature

(message: string, fields: map[string, string]) -> void

## Behavior

Constructs a TRACE-level record with timestamp, message, and fields. Discarded by default; emitted only when the configured minimum severity is TRACE.

## Tests

+ writes a record when the configured level is TRACE
+ includes every supplied field
+ produces a record with severity "TRACE"
- discards the record when the configured level is DEBUG or higher

## Assumptions

? TRACE is below DEBUG; typically used for very verbose diagnostic output
