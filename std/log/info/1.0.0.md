# std.log.info

Emits a structured log record at the INFO severity.

## Signature

(message: string, fields: map[string, string]) -> void

## Behavior

Constructs a structured log record with the current wall-clock timestamp, the severity INFO, the supplied message, and every field from `fields`. Writes the record to the process's logging sink. Discarded at runtime when the configured minimum severity is above INFO.

## Tests

+ writes a record containing the message when the configured level is INFO or lower
+ includes every supplied field verbatim
+ includes a wall-clock timestamp
+ produces a record with severity "INFO"
- discards the record when the configured level is WARN or higher

## Assumptions

? default sink is stderr; reconfigurable via a separate setup function
? record format is JSON lines by default
