# std.log.error

Emits a structured log record at the ERROR severity.

## Signature

(message: string, fields: map[string, string]) -> void

## Behavior

Constructs a structured log record with the current wall-clock timestamp, the severity ERROR, the supplied message, and every field from `fields`. Writes the record to the process's logging sink. Never filtered out by severity; ERROR is always emitted.

## Tests

+ writes a record containing the message unconditionally
+ includes every supplied field verbatim
+ includes a wall-clock timestamp
+ produces a record with severity "ERROR"
- never discarded by severity filter (ERROR is the highest default level)

## Assumptions

? default sink is stderr; reconfigurable via a separate setup function
? record format is JSON lines by default
