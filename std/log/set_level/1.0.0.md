# std.log.set_level

Sets the global minimum severity for log output.

## Signature

(level: string) -> result[void, string]

## Behavior

Accepts `"TRACE"`, `"DEBUG"`, `"INFO"`, `"WARN"`, `"ERROR"`, `"FATAL"` (case-insensitive). Records below the given level are discarded by subsequent calls to `debug`, `info`, etc.

## Tests

+ accepts all six standard levels (case-insensitive)
+ changes the global minimum; subsequent records below it are filtered
- returns error on an unrecognized level string

## Assumptions

? global configuration; per-logger configuration is a separate rune
? default level is INFO
