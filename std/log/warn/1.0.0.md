# std.log.warn

Emits a structured log record at the WARN severity.

## Signature

(message: string, fields: map[string, string]) -> void

## Behavior

Constructs a WARN-level record with timestamp, message, and fields. Writes to the process's log sink. Discarded when the configured minimum severity is above WARN (i.e. ERROR or FATAL only).

## Tests

+ writes a record when the configured level is WARN or lower
+ includes every supplied field
+ includes a wall-clock timestamp
+ produces a record with severity "WARN"
- discards the record when the configured level is ERROR

## Assumptions

? default sink is stderr
