# std.int.parse_octal

Parses an unsigned octal integer literal (no `0o` prefix) into an i64.

## Signature

(s: string) -> result[i64, string]

## Behavior

Accepts one or more octal digits (`0`-`7`). Rejects empty strings, the `0o` prefix, and any non-octal digit.

## Tests

+ returns 0 for "0"
+ returns 8 for "10"
+ returns 511 for "777"
- returns error on "0o7" (no prefix accepted)
- returns error on "8" (out of octal digit range)
- returns error on empty input
- returns error on overflow beyond i64 range

## Assumptions

? input has no "0o" or leading "0" prefix
