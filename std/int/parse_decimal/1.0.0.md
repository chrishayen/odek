# std.int.parse_decimal

Parses a signed decimal integer literal into an i64.

## Signature

(s: string) -> result[i64, string]

## Behavior

Accepts an optional leading `+` or `-`, then one or more decimal digits. Rejects empty strings, leading zeros beyond a single `0`, and any non-digit byte after the sign.

## Tests

+ returns 0 for "0"
+ returns 123 for "123"
+ returns -42 for "-42"
+ returns 42 for "+42"
- returns error on empty input
- returns error on a non-digit character
- returns error on overflow beyond i64 range

## Assumptions

? leading zeros like "00" are rejected; "0" alone is fine
? no underscores or thousands separators are accepted
