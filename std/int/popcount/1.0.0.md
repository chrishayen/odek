# std.int.popcount

Returns the number of 1-bits in the i64's two's-complement representation.

## Signature

(n: i64) -> i32

## Behavior

Counts every set bit in the 64-bit binary representation of `n`. Range of the result is 0 to 64.

## Tests

+ returns 0 for 0
+ returns 1 for 1
+ returns 1 for any power of two
+ returns 64 for -1 (all bits set in two's complement)
- returns a value outside 0..64 under no circumstance

## Assumptions

? treats the i64 as its unsigned two's-complement bit pattern
