# std.int.pow

Raises an integer base to a non-negative integer exponent and returns an i64.

## Signature

(base: i64, exponent: i32) -> result[i64, string]

## Behavior

Returns `base ** exponent` using integer arithmetic. `pow(x, 0)` is 1 for every x. Rejects negative exponents. Returns an overflow error when the result cannot fit in i64.

## Tests

+ returns 8 for (2, 3)
+ returns 1 for (x, 0) for any x
+ returns 0 for (0, 1)
+ returns 1 for (1, n) for any n
- returns error when exponent is negative
- returns error when the result overflows i64

## Assumptions

? integer-only; floating-point exponents go through std.math.pow
