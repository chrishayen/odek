# std.int.lcm

Returns the least common multiple of two integers, as a non-negative value.

## Signature

(a: i64, b: i64) -> result[i64, string]

## Behavior

Computes `|a / gcd(a, b) * b|`. Returns 0 when either input is 0. Returns an overflow error when the result cannot fit in i64.

## Tests

+ returns 24 for (8, 12)
+ returns 0 for (0, x)
+ returns |x| for (x, x)
+ always returns a non-negative value
- returns error when the product overflows i64

## Assumptions

? lcm(0, 0) is defined as 0, not an error
