# std.int.trailing_zeros

Returns the number of trailing zero bits in the i64's 64-bit representation.

## Signature

(n: i64) -> i32

## Behavior

Counts zero bits from the least-significant end until the first 1-bit is reached. Returns 64 when the input is 0.

## Tests

+ returns 64 for 0
+ returns 0 for 1
+ returns 3 for 8
+ returns 63 for i64::MIN (only the sign bit set)
- returns a value outside 0..64 under no circumstance

## Assumptions

? treats the i64 as its unsigned two's-complement bit pattern
