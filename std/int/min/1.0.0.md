# std.int.min

Returns the smaller of two i64 values.

## Signature

(a: i64, b: i64) -> i64

## Behavior

Returns `a` when `a <= b`, otherwise `b`.

## Tests

+ returns 1 for (1, 2)
+ returns 1 for (2, 1)
+ returns a when a equals b
- returns a value greater than either input under no circumstance

## Assumptions

? comparison is signed
