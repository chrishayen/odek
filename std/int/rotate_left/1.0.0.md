# std.int.rotate_left

Rotates the bits of an i64 to the left by `bits` positions.

## Signature

(n: i64, bits: i32) -> i64

## Behavior

Shifts bits left by `bits mod 64`, with the overflow bits re-entering on the right. Does not change the overall set of bits — only their positions.

## Tests

+ returns n unchanged when bits is 0
+ returns n unchanged when bits is a multiple of 64
+ rotates 0x8000000000000000 by 1 to 0x0000000000000001
+ rotate_left by n and rotate_right by n are inverse operations
- returns a value with a different set of 1-bits than the input

## Assumptions

? negative bits values are normalized via mod 64
