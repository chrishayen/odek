# std.int.abs

Returns the absolute value of an i64.

## Signature

(n: i64) -> result[i64, string]

## Behavior

Returns `n` when `n >= 0`, otherwise `-n`. The single value `i64::MIN` cannot be represented as a positive i64 and returns an overflow error.

## Tests

+ returns 5 for input 5
+ returns 5 for input -5
+ returns 0 for input 0
- returns error for input i64::MIN (overflow)

## Assumptions

? overflow is an error, not a wrap; no silent two's-complement reuse
