# std.int.parse_binary

Parses an unsigned binary integer literal (no `0b` prefix) into an i64.

## Signature

(s: string) -> result[i64, string]

## Behavior

Accepts one or more bits (`0` or `1`). Rejects empty strings, the `0b` prefix, and any non-bit character.

## Tests

+ returns 0 for "0"
+ returns 1 for "1"
+ returns 255 for "11111111"
- returns error on "0b1" (no prefix accepted)
- returns error on empty input
- returns error on any non-bit character
- returns error on overflow beyond i64 range

## Assumptions

? input has no "0b" prefix
