# std.int.format_decimal

Formats an i64 as a signed decimal string.

## Signature

(n: i64) -> string

## Behavior

Returns the shortest decimal representation of `n` with an optional leading `-` for negative values. No thousands separators, no padding.

## Tests

+ returns "0" for 0
+ returns "123" for 123
+ returns "-42" for -42
+ returns "9223372036854775807" for i64 maximum
- emits no leading zeros
- emits no "+" prefix for positive values

## Assumptions

? no thousands separators; use a separate locale-aware formatter for that
