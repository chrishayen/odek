# std.int.format_binary

Formats an i64 as a binary string (no `0b` prefix, no leading zeros, unsigned view).

## Signature

(n: i64) -> string

## Behavior

Interprets the i64 as a 64-bit unsigned value and returns its binary representation using `0` and `1`. No leading zeros (except "0" for zero). No `0b` prefix.

## Tests

+ returns "0" for 0
+ returns "1" for 1
+ returns "11111111" for 255
+ returns a 64-character string for -1 (two's complement view)
- emits no "0b" prefix
- emits no leading zeros (except for "0")

## Assumptions

? unsigned 64-bit view of the i64 bit pattern
