# std.int.leading_zeros

Returns the number of leading zero bits in the i64's 64-bit representation.

## Signature

(n: i64) -> i32

## Behavior

Counts zero bits from the most-significant end until the first 1-bit is reached. Returns 64 when the input is 0.

## Tests

+ returns 64 for 0
+ returns 63 for 1
+ returns 0 for any negative value (MSB set in two's complement)
+ returns 1 for i64 maximum (0x7FFF...)
- returns a value outside 0..64 under no circumstance

## Assumptions

? treats the i64 as its unsigned two's-complement bit pattern
