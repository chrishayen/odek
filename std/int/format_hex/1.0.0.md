# std.int.format_hex

Formats an i64 as a lowercase hexadecimal string (no `0x` prefix, no leading zeros, unsigned view).

## Signature

(n: i64) -> string

## Behavior

Interprets the i64 as a 64-bit unsigned value and returns its hex representation using `0-9a-f`. No leading zeros; the shortest representation. No `0x` prefix.

## Tests

+ returns "0" for 0
+ returns "ff" for 255
+ returns "ffffffffffffffff" for -1 (two's complement view)
+ always emits lowercase digits
- emits no "0x" prefix
- emits no leading zeros

## Assumptions

? treats the i64 as an unsigned 64-bit value for formatting
? lowercase output; uppercase is a separate rune if needed
