# std.int.parse_hex

Parses an unsigned hexadecimal integer literal (no `0x` prefix) into an i64.

## Signature

(s: string) -> result[i64, string]

## Behavior

Accepts one or more hex digits (case-insensitive). Rejects empty strings, the `0x` prefix (the caller strips it), or any non-hex character.

## Tests

+ returns 0 for "0"
+ returns 255 for "ff"
+ returns 255 for "FF"
+ returns the i64 maximum for "7fffffffffffffff"
- returns error on "0xff" (no prefix accepted)
- returns error on empty input
- returns error on a non-hex character
- returns error on overflow beyond i64 range

## Assumptions

? input has no "0x" prefix; caller strips before calling
