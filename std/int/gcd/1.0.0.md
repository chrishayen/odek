# std.int.gcd

Returns the greatest common divisor of two integers, as a non-negative value.

## Signature

(a: i64, b: i64) -> i64

## Behavior

Applies the Euclidean algorithm to `|a|` and `|b|`. Returns the non-negative GCD. `gcd(0, 0)` returns 0. `gcd(x, 0)` returns `|x|`.

## Tests

+ returns 4 for (12, 8)
+ returns 5 for (-15, 10)
+ returns |x| for (x, 0)
+ returns 0 for (0, 0)
- returns a negative value under no circumstance

## Assumptions

? always non-negative even for negative inputs
