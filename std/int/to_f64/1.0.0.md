# std.int.to_f64

Converts an i64 to an f64, with rounding when the magnitude exceeds 2^53.

## Signature

(n: i64) -> f64

## Behavior

Values in [-2^53, 2^53] convert exactly. Larger magnitudes round to the nearest representable f64 under round-to-nearest-even.

## Tests

+ converts 0 to 0.0
+ converts 1 to 1.0
+ converts 2^53 exactly
+ converts values above 2^53 by rounding to nearest
- returns NaN under no circumstance (i64 has no NaN)

## Assumptions

? rounding follows IEEE-754 round-to-nearest-even
