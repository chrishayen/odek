# std.int.rotate_right

Rotates the bits of an i64 to the right by `bits` positions.

## Signature

(n: i64, bits: i32) -> i64

## Behavior

Shifts bits right by `bits mod 64`, with the underflow bits re-entering on the left.

## Tests

+ returns n unchanged when bits is 0
+ returns n unchanged when bits is a multiple of 64
+ rotates 0x0000000000000001 by 1 to 0x8000000000000000
+ rotate_left and rotate_right by the same n are inverses
- returns a value with a different set of 1-bits than the input

## Assumptions

? negative bits values are normalized via mod 64
