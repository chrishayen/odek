# std.int.format_octal

Formats an i64 as an octal string (no prefix, no leading zeros, unsigned view).

## Signature

(n: i64) -> string

## Behavior

Interprets the i64 as a 64-bit unsigned value and returns its octal representation using `0-7`. No leading zeros (except "0" for zero). No `0o` prefix.

## Tests

+ returns "0" for 0
+ returns "10" for 8
+ returns "777" for 511
- emits no "0o" prefix
- emits no leading zeros

## Assumptions

? unsigned 64-bit view of the i64 bit pattern
