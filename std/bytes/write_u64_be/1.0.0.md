# std.bytes.write_u64_be

Returns an 8-byte buffer holding the big-endian encoding of a u64.

## Signature

(value: u64) -> bytes

## Behavior

Emits bytes MSB-first.

## Tests

+ returns [0x08..0x01] reversed for 0x0807060504030201
+ returns eight zero bytes for 0
+ returns exactly 8 bytes
- does not emit little-endian order

## Assumptions

? always 8 bytes
