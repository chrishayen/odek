# std.bytes.read_u32_le

Reads a little-endian unsigned 32-bit integer from a byte buffer at the given offset.

## Signature

(b: bytes, offset: i64) -> result[u32, string]

## Behavior

Reads four bytes starting at `offset` and assembles them LSB-first into a u32.

## Tests

+ returns 0x04030201 for bytes [0x01, 0x02, 0x03, 0x04]
+ returns 0x00000000 for four zero bytes
+ returns 0xFFFFFFFF for four 0xFF bytes
- returns error when offset is negative
- returns error when offset + 4 exceeds length

## Assumptions

? result is unsigned 32-bit
