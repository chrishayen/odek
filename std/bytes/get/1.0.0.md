# std.bytes.get

Returns the byte at a given index, or an error when out of range.

## Signature

(b: bytes, i: i64) -> result[u8, string]

## Behavior

Returns the byte at index `i` when `0 <= i < length(b)`. Returns an error otherwise.

## Tests

+ returns the byte at index 0 for a non-empty buffer
+ returns the byte at index length-1
- returns error when i is negative
- returns error when i is equal to or greater than length

## Assumptions

? no negative-index-wraps-from-end convention; negative is always an error
