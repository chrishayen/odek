# std.bytes.reverse

Returns a new buffer with the bytes of the input in reverse order.

## Signature

(b: bytes) -> bytes

## Behavior

Allocates a fresh buffer of the same length, copying byte `i` of the input to position `length - 1 - i` of the output.

## Tests

+ returns an empty buffer for an empty input
+ returns [0x02, 0x01] for input [0x01, 0x02]
+ returns the original length
+ reverse applied twice returns the original buffer
- does not mutate the input

## Assumptions

? always allocates a new buffer
