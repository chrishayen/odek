# std.bytes.write_u16_be

Returns a 2-byte buffer holding the big-endian encoding of a u16.

## Signature

(value: u16) -> bytes

## Behavior

Emits the high byte first, then the low byte.

## Tests

+ returns [0x02, 0x01] for 0x0201
+ returns [0x00, 0x00] for 0
+ returns exactly 2 bytes
- does not emit little-endian order

## Assumptions

? always 2 bytes
