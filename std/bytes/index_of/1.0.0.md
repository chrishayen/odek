# std.bytes.index_of

Returns the index of the first occurrence of `needle` in `haystack`, or none if not found.

## Signature

(haystack: bytes, needle: bytes) -> optional[i64]

## Behavior

Scans `haystack` left-to-right and returns `some(i)` for the start index of the first match. Returns `none` when `needle` does not appear. Returns `some(0)` when `needle` is empty (empty matches at the start).

## Tests

+ returns some(0) when needle is empty
+ returns some(i) for the first occurrence
+ returns none when needle is not present
+ returns none when needle is longer than haystack
- returns a negative index under no circumstance

## Assumptions

? empty-needle convention matches Python, returning 0
