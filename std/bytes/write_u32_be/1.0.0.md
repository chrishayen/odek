# std.bytes.write_u32_be

Returns a 4-byte buffer holding the big-endian encoding of a u32.

## Signature

(value: u32) -> bytes

## Behavior

Emits bytes MSB-first.

## Tests

+ returns [0x04, 0x03, 0x02, 0x01] for 0x04030201
+ returns four zero bytes for 0
+ returns exactly 4 bytes
- does not emit little-endian order

## Assumptions

? always 4 bytes
