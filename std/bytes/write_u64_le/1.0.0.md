# std.bytes.write_u64_le

Returns an 8-byte buffer holding the little-endian encoding of a u64.

## Signature

(value: u64) -> bytes

## Behavior

Emits bytes LSB-first.

## Tests

+ returns [0x01..0x08] for 0x0807060504030201
+ returns eight zero bytes for 0
+ returns exactly 8 bytes
- does not emit big-endian order

## Assumptions

? always 8 bytes
