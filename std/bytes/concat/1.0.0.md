# std.bytes.concat

Returns a new buffer containing every byte of `a` followed by every byte of `b`.

## Signature

(a: bytes, b: bytes) -> bytes

## Behavior

Allocates a fresh buffer of size `length(a) + length(b)` and copies both inputs in order.

## Tests

+ returns a buffer whose length is sum of input lengths
+ returns b when a is empty
+ returns a when b is empty
+ returns an empty buffer when both inputs are empty
- does not mutate either input

## Assumptions

? always allocates a new buffer; no aliasing with inputs
