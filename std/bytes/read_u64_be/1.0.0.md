# std.bytes.read_u64_be

Reads a big-endian unsigned 64-bit integer from a byte buffer at the given offset.

## Signature

(b: bytes, offset: i64) -> result[u64, string]

## Behavior

Reads eight bytes starting at `offset` and assembles them MSB-first into a u64.

## Tests

+ returns 0x0102030405060708 for bytes [0x01..0x08]
+ returns 0 for eight zero bytes
- returns error when offset is negative
- returns error when offset + 8 exceeds length

## Assumptions

? result is unsigned 64-bit
