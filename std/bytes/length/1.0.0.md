# std.bytes.length

Returns the number of bytes in a buffer.

## Signature

(b: bytes) -> i64

## Behavior

Returns the count of bytes in the buffer. Never negative.

## Tests

+ returns 0 for an empty buffer
+ returns 3 for a three-byte buffer
+ returns the correct count for buffers of any length
- never returns a negative value

## Assumptions

? length is computed in O(1); buffers carry their own length
