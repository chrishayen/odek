# std.bytes.write_u32_le

Returns a 4-byte buffer holding the little-endian encoding of a u32.

## Signature

(value: u32) -> bytes

## Behavior

Emits bytes LSB-first.

## Tests

+ returns [0x01, 0x02, 0x03, 0x04] for 0x04030201
+ returns four zero bytes for 0
+ returns four 0xFF bytes for u32 max
+ returns exactly 4 bytes
- does not emit big-endian order

## Assumptions

? always 4 bytes
