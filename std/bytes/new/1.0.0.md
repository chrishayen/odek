# std.bytes.new

Allocates a byte buffer of the given length, zero-initialized.

## Signature

(length: i64) -> bytes

## Behavior

Returns a buffer whose length is exactly `length` and whose every byte is zero. A length of 0 returns an empty buffer.

## Tests

+ returns a buffer of exactly `length` bytes
+ returns an empty buffer for length 0
+ every byte in the returned buffer is zero
- length cannot be negative; negative input returns an empty buffer or errors per platform

## Assumptions

? negative length is treated as 0 rather than an error
