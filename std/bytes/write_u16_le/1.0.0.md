# std.bytes.write_u16_le

Returns a 2-byte buffer holding the little-endian encoding of a u16.

## Signature

(value: u16) -> bytes

## Behavior

Emits the low byte first, then the high byte.

## Tests

+ returns [0x01, 0x02] for 0x0201
+ returns [0x00, 0x00] for 0
+ returns [0xFF, 0xFF] for 0xFFFF
+ returns exactly 2 bytes
- does not emit big-endian order

## Assumptions

? always 2 bytes; no variable-length encoding
