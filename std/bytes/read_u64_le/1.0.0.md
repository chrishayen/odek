# std.bytes.read_u64_le

Reads a little-endian unsigned 64-bit integer from a byte buffer at the given offset.

## Signature

(b: bytes, offset: i64) -> result[u64, string]

## Behavior

Reads eight bytes starting at `offset` and assembles them LSB-first into a u64.

## Tests

+ returns 0x0807060504030201 for bytes [0x01..0x08]
+ returns 0 for eight zero bytes
+ returns u64 max for eight 0xFF bytes
- returns error when offset is negative
- returns error when offset + 8 exceeds length

## Assumptions

? result is unsigned 64-bit
