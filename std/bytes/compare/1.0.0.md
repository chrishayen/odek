# std.bytes.compare

Compares two byte buffers lexicographically and returns -1, 0, or 1.

## Signature

(a: bytes, b: bytes) -> i32

## Behavior

Walks both buffers in parallel. Returns -1 when the first differing byte in `a` is less than in `b`, 1 when greater. When one buffer is a prefix of the other, the shorter is "less". Equal buffers return 0.

## Tests

+ returns 0 for two identical buffers
+ returns -1 when a is a proper prefix of b
+ returns 1 when b is a proper prefix of a
+ returns -1 for ([0x01, 0x02], [0x01, 0x03])
- returns exactly one of {-1, 0, 1} for any input pair

## Assumptions

? lexicographic byte comparison, unsigned
