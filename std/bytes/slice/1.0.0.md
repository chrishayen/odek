# std.bytes.slice

Returns the subrange of a byte buffer from `start` (inclusive) to `end` (exclusive).

## Signature

(b: bytes, start: i64, end: i64) -> result[bytes, string]

## Behavior

Returns bytes from index `start` up to (but not including) `end`. Valid range is `0 <= start <= end <= length(b)`. Returns an empty buffer when `start == end`.

## Tests

+ returns the full buffer for (b, 0, length(b))
+ returns an empty buffer for (b, k, k)
+ returns the middle portion for valid sub-range
- returns error when start is negative
- returns error when end exceeds length
- returns error when start > end

## Assumptions

? slice is a fresh allocation; callers cannot mutate `b` through it
