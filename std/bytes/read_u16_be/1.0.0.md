# std.bytes.read_u16_be

Reads a big-endian unsigned 16-bit integer from a byte buffer at the given offset.

## Signature

(b: bytes, offset: i64) -> result[u16, string]

## Behavior

Reads two bytes starting at `offset` and combines them as `(b[offset] << 8) | b[offset+1]`.

## Tests

+ returns 0x0102 for bytes [0x01, 0x02] at offset 0
+ returns 0xFFFF for bytes [0xFF, 0xFF]
- returns error when offset is negative
- returns error when offset + 2 exceeds length

## Assumptions

? result is unsigned 16-bit
