# std.bytes.equal

Returns true when two byte buffers have the same length and identical contents.

## Signature

(a: bytes, b: bytes) -> bool

## Behavior

Returns true iff `length(a) == length(b)` and every byte is equal. Non-constant-time; for cryptographic use, callers should prefer `std.crypto.constant_time_compare`.

## Tests

+ returns true for two empty buffers
+ returns true for two buffers with identical contents
+ returns false when lengths differ
+ returns false when contents differ
- does not have constant-time guarantees

## Assumptions

? short-circuits on length check and first differing byte
