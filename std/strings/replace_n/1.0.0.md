# std.strings.replace_n

Returns a copy of the string with at most `n` occurrences of `old` replaced by `new`, left-to-right.

## Signature

(s: string, old: string, new: string, n: i64) -> string

## Behavior

Scans left-to-right. Replaces the first `n` non-overlapping matches, leaving any further matches untouched. Negative `n` means unlimited (equivalent to `replace`).

## Tests

+ replaces exactly n matches when at least n are present
+ replaces every match when n exceeds the count
+ returns s unchanged when n is 0
+ treats negative n as unlimited
- does not continue after the n-th replacement

## Assumptions

? matching is byte-exact
