# std.strings.trim

Returns a copy of the input with leading and trailing whitespace removed.

## Signature

(s: string) -> string

## Behavior

Scans forward from the start and backward from the end of the string, advancing each cursor past any code point classified as whitespace by Unicode. The substring between the two cursors is returned. When the entire string is whitespace, the empty string is returned.

## Tests

+ returns "hello" for input "  hello  "
+ returns "a b" for input "\t a b \n"
+ returns "" for an all-whitespace string
+ returns "" for an empty string
+ preserves internal whitespace between non-whitespace runs
- returns the same string when neither end has whitespace

## Assumptions

? whitespace is the Unicode White_Space property (includes space, tab, CR, LF, NBSP, and others)
? only leading and trailing whitespace are removed; internal whitespace is untouched
