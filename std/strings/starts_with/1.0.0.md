# std.strings.starts_with

Returns true when the input string begins with the given prefix.

## Signature

(s: string, prefix: string) -> bool

## Behavior

Compares `prefix` against the first `len(prefix)` bytes of `s`. Returns true iff every byte matches. The empty prefix is considered a prefix of every string.

## Tests

+ returns true for input ("hello world", "hello")
+ returns true when prefix equals s
+ returns true when prefix is the empty string
+ returns true for input ("", "") — empty prefix of empty string
- returns false when prefix is longer than s
- returns false when the first bytes of s differ from prefix
- returns false when prefix matches only a middle or suffix portion of s

## Assumptions

? matching is byte-exact and case-sensitive
