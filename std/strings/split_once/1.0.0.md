# std.strings.split_once

Splits the string at the first occurrence of a separator and returns the two halves.

## Signature

(s: string, sep: string) -> optional[pair[string, string]]

## Behavior

Finds the first occurrence of `sep`. Returns `some((before, after))`. Returns `none` when the separator is absent.

## Tests

+ returns some(("a", "b,c")) for ("a,b,c", ",")
+ returns some(("", "bc")) when sep matches at start
+ returns some(("abc", "")) when sep matches at end
+ returns none when sep is absent
- does not return multiple-way split (that is strings.split)

## Assumptions

? only the first occurrence is used; later occurrences are included in the right half
