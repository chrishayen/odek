# std.strings.trim_right

Returns a copy of the input with trailing whitespace removed.

## Signature

(s: string) -> string

## Behavior

Scans from the end backward past every Unicode whitespace code point and returns the substring from start up to that cursor.

## Tests

+ returns "  hello" for "  hello  "
+ returns "" for an all-whitespace string
+ returns "" for an empty string
+ preserves leading whitespace
- does not strip leading whitespace

## Assumptions

? whitespace is the Unicode White_Space property
