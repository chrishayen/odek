# std.strings.index_of

Returns the byte index of the first occurrence of a substring, or none if not present.

## Signature

(s: string, substr: string) -> optional[i64]

## Behavior

Scans `s` left-to-right for the first position where `substr` appears. Returns `some(i)` on match, `none` otherwise. Returns `some(0)` when `substr` is the empty string.

## Tests

+ returns some(0) when substr matches at the start
+ returns some(i) for the first occurrence
+ returns none when substr is absent
+ returns some(0) when substr is empty
- returns a negative index under no circumstance

## Assumptions

? byte-exact matching; for code-point indices use a utf8 variant
