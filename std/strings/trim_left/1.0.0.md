# std.strings.trim_left

Returns a copy of the input with leading whitespace removed.

## Signature

(s: string) -> string

## Behavior

Advances from the start past every Unicode whitespace code point and returns the substring from that cursor to end. Does not touch trailing whitespace.

## Tests

+ returns "hello  " for "  hello  "
+ returns "" for an all-whitespace string
+ returns "" for an empty string
+ preserves trailing whitespace
- does not strip trailing whitespace

## Assumptions

? whitespace is the Unicode White_Space property
