# std.strings.replace

Returns a copy of the string with every non-overlapping occurrence of `old` replaced by `new`.

## Signature

(s: string, old: string, new: string) -> string

## Behavior

Scans left-to-right for non-overlapping occurrences of `old`. Each match is removed from the output and the `new` value is emitted in its place. Matches do not cascade: characters contributed by `new` are not re-scanned. When `old` is the empty string, `new` is inserted between every adjacent pair of code points and at both ends.

## Tests

+ returns "hXllX" for input ("hello", "e", "X") and ("hello", "l", "X") -> "heXXo"
+ returns the original string when `old` is not present
+ returns "" for an empty input
+ replaces all non-overlapping matches, not just the first
+ does not re-scan inserted replacement text (old="aa", new="a", input="aaaa" -> "aa", not "a")
- returns `new` inserted between every character when `old` is empty

## Assumptions

? matching is byte-exact; no regex or case-folding
? replacement is greedy left-to-right and non-overlapping
