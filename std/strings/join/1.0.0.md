# std.strings.join

Concatenates a list of strings into a single string with the separator placed between each pair of adjacent elements.

## Signature

(parts: list[string], sep: string) -> string

## Behavior

Concatenates elements in order. The separator appears between elements but never at the start or end. An empty list returns the empty string. A single-element list returns that element unchanged regardless of separator.

## Tests

+ returns "a,b,c" for input (["a", "b", "c"], ",")
+ returns "abc" for input (["a", "b", "c"], "")
+ returns "" for an empty list
+ returns "hello" for input (["hello"], ",")
+ handles empty strings inside the list, producing adjacent separators
- returns "" when every element is the empty string and the separator is non-empty
  (e.g. ([ "", "", "" ], ",") -> ",,")

## Assumptions

? parts are concatenated in the order provided; order is preserved
