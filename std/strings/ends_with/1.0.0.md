# std.strings.ends_with

Returns true when the input string ends with the given suffix.

## Signature

(s: string, suffix: string) -> bool

## Behavior

Compares `suffix` against the last `len(suffix)` bytes of `s`. Returns true iff every byte matches. The empty suffix is considered a suffix of every string.

## Tests

+ returns true for input ("hello world", "world")
+ returns true when suffix equals s
+ returns true when suffix is the empty string
- returns false when suffix is longer than s
- returns false when the last bytes of s differ from suffix
- returns false when suffix matches only a middle or prefix portion

## Assumptions

? matching is byte-exact and case-sensitive
