# std.strings.to_upper

Returns a copy of the input string with every Unicode letter mapped to its upper-case form.

## Signature

(s: string) -> string

## Behavior

Walks the string one Unicode code point at a time and substitutes each letter with its upper-case mapping as defined by the Unicode character database. Non-letter code points pass through unchanged. The returned string is newly allocated; the input is not mutated.

## Tests

+ returns "HELLO" for input "hello"
+ returns "ABC123" for input "abc123"
+ returns "" for an empty string
+ preserves non-letter characters like punctuation and digits
+ upper-cases non-ASCII letters such as "ß" -> "SS" where defined by Unicode
- returns the same string when every code point is already upper-case

## Assumptions

? case mapping follows the default Unicode full case mapping (no locale-specific variants such as Turkish dotted-I)
? surrogate pairs in malformed UTF-16 input are passed through unchanged
