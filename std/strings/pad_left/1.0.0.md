# std.strings.pad_left

Returns a string padded on the left with `pad` (repeated) until its byte length reaches `total_width`.

## Signature

(s: string, total_width: i64, pad: string) -> string

## Behavior

When `len(s) >= total_width`, returns `s` unchanged. Otherwise prepends copies of `pad`, truncating the final copy if needed, so the output is exactly `total_width` bytes.

## Tests

+ returns s unchanged when already at total_width
+ pads " " to the left to reach the target width
+ allows multi-byte pad strings
+ truncates the last pad copy if needed
- does not pad when s is longer than total_width

## Assumptions

? pad must be non-empty; empty pad is an error at the caller's level
