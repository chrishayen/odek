# std.strings.repeat

Returns a string that is `s` concatenated with itself `count` times.

## Signature

(s: string, count: i64) -> string

## Behavior

Returns an empty string for `count <= 0`. Returns `s` for `count == 1`. Returns `s + s + ... + s` (count copies) for `count > 1`.

## Tests

+ returns "" for count 0
+ returns "abc" for ("abc", 1)
+ returns "abcabc" for ("abc", 2)
+ returns "" for ("", any count)
- returns any string other than the repetition under no circumstance

## Assumptions

? negative count returns empty; no error
