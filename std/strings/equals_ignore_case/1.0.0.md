# std.strings.equals_ignore_case

Returns true when two strings are equal after applying Unicode lowercase mapping to both.

## Signature

(a: string, b: string) -> bool

## Behavior

Applies full Unicode lowercase mapping to both strings and compares byte-for-byte. Returns true on match.

## Tests

+ returns true for ("hello", "HELLO")
+ returns true for ("Hello", "heLLO")
+ returns true for two empty strings
+ returns true for non-ASCII casing where Unicode defines it
- returns false when the strings differ beyond case

## Assumptions

? uses Unicode default case mapping; no locale-specific variants
