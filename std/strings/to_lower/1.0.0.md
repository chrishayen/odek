# std.strings.to_lower

Returns a copy of the input string with every Unicode letter mapped to its lower-case form.

## Signature

(s: string) -> string

## Behavior

Walks the string one Unicode code point at a time and substitutes each letter with its lower-case mapping as defined by the Unicode character database. Non-letter code points pass through unchanged. The returned string is newly allocated.

## Tests

+ returns "hello" for input "HELLO"
+ returns "abc123" for input "ABC123"
+ returns "" for an empty string
+ preserves non-letter characters like punctuation and digits
+ lower-cases non-ASCII letters such as "Ä" -> "ä"
- returns the same string when every code point is already lower-case

## Assumptions

? case mapping follows the default Unicode full case mapping (no locale-specific variants)
? code points with no defined lower-case mapping are emitted unchanged
