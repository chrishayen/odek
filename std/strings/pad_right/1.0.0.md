# std.strings.pad_right

Returns a string padded on the right with `pad` until its byte length reaches `total_width`.

## Signature

(s: string, total_width: i64, pad: string) -> string

## Behavior

When `len(s) >= total_width`, returns `s` unchanged. Otherwise appends copies of `pad`, truncating the final copy if needed.

## Tests

+ returns s unchanged when already at total_width
+ pads to the right with space or other pad
+ truncates the last pad copy if needed
- does not pad when s is already long enough

## Assumptions

? pad must be non-empty
