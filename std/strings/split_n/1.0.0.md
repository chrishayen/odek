# std.strings.split_n

Splits the string into at most `n` parts on the separator; the last part contains any remaining separators.

## Signature

(s: string, sep: string, n: i64) -> list[string]

## Behavior

When `n <= 0`, equivalent to `split` (no limit). When `n == 1`, returns `[s]`. When `n > 1`, splits up to `n - 1` times; the last element contains the unsplit remainder.

## Tests

+ returns ["a", "b,c,d"] for ("a,b,c,d", ",", 2)
+ returns ["a", "b", "c,d"] for ("a,b,c,d", ",", 3)
+ returns ["a,b,c,d"] for ("a,b,c,d", ",", 1)
+ returns the full split when n is 0 or negative
- does not exceed n elements in the result

## Assumptions

? negative n is treated as "no limit"
