# std.strings.replace_last

Returns a copy of the string with only the last occurrence of `old` replaced by `new`.

## Signature

(s: string, old: string, new: string) -> string

## Behavior

Finds the last occurrence of `old`, replaces it with `new`. When `old` is absent, returns `s` unchanged. When `old` is empty, returns `s + new` (appended at end).

## Tests

+ replaces only the last occurrence
+ returns s unchanged when old is absent
+ appends new when old is empty
- does not replace earlier occurrences

## Assumptions

? matching is byte-exact
