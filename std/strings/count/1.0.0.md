# std.strings.count

Returns the number of non-overlapping occurrences of `substr` in `s`.

## Signature

(s: string, substr: string) -> i64

## Behavior

Scans left-to-right, counting non-overlapping matches. Returns `len(s) + 1` when `substr` is empty (matches at every position).

## Tests

+ returns 2 for ("ababa", "ab")
+ returns 0 when substr is absent
+ returns len(s) + 1 when substr is empty
+ counts non-overlapping: ("aaaa", "aa") returns 2
- returns a negative value under no circumstance

## Assumptions

? non-overlapping count; advances past each match by its length
