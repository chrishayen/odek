# std.strings.length

Returns the number of bytes in a string.

## Signature

(s: string) -> i64

## Behavior

Returns the byte count of the string's UTF-8 encoding. For code-point or grapheme counts, use a separate rune under utf8.

## Tests

+ returns 0 for an empty string
+ returns 5 for "hello"
+ returns the UTF-8 byte length (not code-point count) for multi-byte characters
- never returns a negative value

## Assumptions

? byte length, not code-point count
