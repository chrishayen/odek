# std.strings.replace_first

Returns a copy of the string with only the first occurrence of `old` replaced by `new`.

## Signature

(s: string, old: string, new: string) -> string

## Behavior

Finds the first occurrence of `old`, removes it, inserts `new`, and returns the result. Later occurrences are left in place. When `old` is absent, returns `s` unchanged.

## Tests

+ replaces only the first occurrence
+ returns s unchanged when old is absent
+ returns new + s when old is empty (inserted at start)
- does not replace occurrences after the first

## Assumptions

? matching is byte-exact
