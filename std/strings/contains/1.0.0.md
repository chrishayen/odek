# std.strings.contains

Returns true when the input string contains the given substring anywhere within it.

## Signature

(s: string, substr: string) -> bool

## Behavior

Performs a byte-exact search of `s` for an occurrence of `substr`. Returns true on the first match. The empty substring is considered present in every string, including the empty string.

## Tests

+ returns true for input ("hello world", "world")
+ returns true when substr equals s
+ returns true when substr is the empty string
+ returns true when substr is at the start of s
+ returns true when substr is at the end of s
- returns false when substr does not appear in s
- returns false when substr is longer than s

## Assumptions

? matching is byte-exact and case-sensitive
