# std.strings.last_index_of

Returns the byte index of the last occurrence of a substring, or none if not present.

## Signature

(s: string, substr: string) -> optional[i64]

## Behavior

Scans `s` right-to-left for the last position where `substr` appears. Returns `some(i)` on match, `none` otherwise. Returns `some(len(s))` when `substr` is empty (empty matches at end).

## Tests

+ returns some(i) for the last occurrence
+ returns some(len(s)) when substr is empty
+ returns none when substr is absent
- returns a negative index under no circumstance

## Assumptions

? empty-substring convention matches start-of-string index_of; last-index_of returns end
