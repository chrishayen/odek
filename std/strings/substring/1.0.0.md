# std.strings.substring

Returns the byte-index slice of the input from `start` (inclusive) to `end` (exclusive).

## Signature

(s: string, start: i64, end: i64) -> result[string, string]

## Behavior

Requires `0 <= start <= end <= len(s)`. Returns the substring of those bytes. Valid empty slice when `start == end`.

## Tests

+ returns the full string for (s, 0, len(s))
+ returns an empty string for (s, k, k)
+ returns the middle portion for a valid range
- returns error when start is negative
- returns error when end exceeds length
- returns error when start > end

## Assumptions

? byte indices, not code-point indices; may split multi-byte UTF-8 sequences
