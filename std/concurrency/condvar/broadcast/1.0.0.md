# std.concurrency.condvar.broadcast

Wakes every thread currently waiting on the condition variable.

## Signature

(cv: condvar) -> void

## Behavior

Unblocks every waiter. They will each race to re-acquire the associated mutex before returning from `wait`.

## Tests

+ wakes every waiting thread
+ is a no-op when no waiters
+ subsequent waiters (joining after broadcast) are not affected
- does not cause a waiter to return without re-acquiring the mutex

## Assumptions

? used when multiple waiters may benefit from the state change
