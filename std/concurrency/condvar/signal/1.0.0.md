# std.concurrency.condvar.signal

Wakes exactly one thread (if any) waiting on the condition variable.

## Signature

(cv: condvar) -> void

## Behavior

Atomically selects one blocked thread (implementation-defined choice) and unblocks it. No-op when no threads are waiting.

## Tests

+ wakes exactly one waiter when at least one is waiting
+ is a no-op when no waiters
- does not wake more than one waiter in a single signal call

## Assumptions

? the caller does not need to hold the associated mutex, but typically does
