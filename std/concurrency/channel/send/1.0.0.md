# std.concurrency.channel.send

Sends a value on a channel, blocking until a receiver is ready or the buffer has space.

## Signature

(ch: channel[T], x: T) -> result[void, string]

## Behavior

For unbuffered channels, blocks the calling thread until another thread is ready to receive. For buffered channels, blocks only when the buffer is full. Returns an error when the channel has been closed. Values are delivered in FIFO order relative to sender calls.

## Tests

+ delivers the value to the next receiver on an unbuffered channel
+ appends to the buffer without blocking when a buffered channel has space
+ blocks until the buffer drains when a buffered channel is full
+ preserves FIFO order across multiple senders
- returns error when the channel is closed
- returns error when the channel is closed during a blocked send

## Assumptions

? sends are first-in-first-out per sender; no priority
? panics are converted to errors
