# std.concurrency.channel.receive

Receives a value from a channel, blocking until one is available or the channel is closed.

## Signature

(ch: channel[T]) -> result[optional[T], string]

## Behavior

Blocks the caller until a value is available to receive. Returns `some(value)` on success. Returns `none` when the channel is closed and no buffered values remain. Values are received in FIFO order across senders.

## Tests

+ returns some(value) when a sender provides one
+ returns buffered values first before blocking
+ returns none once the channel is closed and drained
+ respects FIFO across multiple senders
- returns error (not none) when an underlying transport failure occurs

## Assumptions

? distinguishes "closed and drained" (none) from "closed with buffered values remaining" (returns values until drained, then none)
? does not support timeout directly; callers compose with a select primitive
