# std.concurrency.once.do

Executes `f` exactly once for the given `once_token`, even under concurrent callers.

## Signature

(once: once_token, f: func() -> void) -> void

## Behavior

The first caller executes `f`. All subsequent calls (including concurrent ones) wait for the first to complete, then return without calling `f` again.

## Tests

+ calls f exactly once across all concurrent callers
+ blocks concurrent callers until the first completes
+ does not call f again on subsequent calls after the first
- produces any observable side effect of f more than once under no circumstance

## Assumptions

? panic/error in f still counts as "done"; subsequent calls do not retry
