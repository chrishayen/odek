# std.concurrency.wait_group.done

Decrements a wait group's counter by 1, typically called by a completing worker.

## Signature

(wg: wait_group) -> void

## Behavior

Atomically decrements the counter. When the counter reaches zero, any threads blocked in `wait` are unblocked.

## Tests

+ decrements the counter by 1
+ unblocks waiters when the counter reaches 0
- produces undefined behavior when called more times than add

## Assumptions

? done is typically called via defer/finally to ensure it runs even on error
