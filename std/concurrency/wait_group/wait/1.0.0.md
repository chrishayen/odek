# std.concurrency.wait_group.wait

Blocks the caller until the wait group's counter reaches zero.

## Signature

(wg: wait_group) -> void

## Behavior

If the counter is already zero, returns immediately. Otherwise blocks until a `done` call drops it to zero.

## Tests

+ returns immediately when the counter is already 0
+ blocks until the counter reaches 0
+ multiple waiters can block concurrently and are all released when the counter hits 0
- never returns early while the counter is non-zero

## Assumptions

? a wait group cannot be reused concurrently; add must complete before the next wait cycle
