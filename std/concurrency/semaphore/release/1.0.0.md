# std.concurrency.semaphore.release

Increments a semaphore's permit count, unblocking one waiter if any.

## Signature

(sem: semaphore) -> void

## Behavior

Increments the count. If one or more threads are blocked in `acquire`, one is unblocked.

## Tests

+ unblocks a single waiting acquirer
+ increments the permit count when no waiters are blocked
- produces undefined behavior when called more times than the max permit count allows

## Assumptions

? release may be called by a different thread than the one that acquired (unlike mutex)
