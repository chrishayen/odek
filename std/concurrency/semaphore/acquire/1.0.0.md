# std.concurrency.semaphore.acquire

Decrements a semaphore's permit count, blocking when the count is zero.

## Signature

(sem: semaphore) -> void

## Behavior

When the permit count is > 0, decrements it and returns. Otherwise blocks until a permit becomes available (via another thread's release).

## Tests

+ returns immediately when a permit is available
+ blocks until a permit is released when none available
+ multiple waiters are served in FIFO order (or implementation-defined fair order)
- never returns with a negative permit count

## Assumptions

? fairness: implementation-defined, typically FIFO
