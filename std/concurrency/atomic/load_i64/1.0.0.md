# std.concurrency.atomic.load_i64

Atomically reads the current value of an atomic i64 cell.

## Signature

(cell: atomic_i64) -> i64

## Behavior

Returns the value most recently stored into the cell. Uses sequentially consistent ordering: any store that happens-before the load is observable. Never produces a torn value.

## Tests

+ returns the value most recently stored
+ returns the initial value before any store
+ returns a consistent snapshot even under concurrent stores
+ is linearizable with respect to other atomic operations on the same cell
- produces no torn read under concurrent access

## Assumptions

? memory order is sequentially consistent
? single-word atomic load is supported on all target platforms
