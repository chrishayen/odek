# std.concurrency.atomic.compare_and_swap

Atomically swaps `new_value` into the cell if and only if its current value equals `expected`; returns whether the swap happened.

## Signature

(cell: atomic_i64, expected: i64, new_value: i64) -> bool

## Behavior

Reads the cell, compares to `expected`. If equal, stores `new_value` and returns true. Otherwise leaves the cell unchanged and returns false. Sequentially consistent.

## Tests

+ returns true and stores new_value when the cell holds expected
+ returns false and leaves the cell alone when the cell differs
+ serializes with other atomic operations on the cell
- is linearizable under concurrent CAS

## Assumptions

? sequentially consistent ordering
? caller loops on false return for "retry on failure"
