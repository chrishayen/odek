# std.concurrency.atomic.store_i64

Atomically writes a value into an atomic i64 cell.

## Signature

(cell: atomic_i64, value: i64) -> void

## Behavior

Writes `value` to the cell. Any subsequent load from any thread sees either this write or a later one. Sequentially consistent.

## Tests

+ stores the value
+ makes the new value visible to all threads after this call
+ serializes with other atomic operations on the cell
- produces no torn write under concurrent access

## Assumptions

? sequentially consistent ordering
