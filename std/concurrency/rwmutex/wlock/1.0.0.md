# std.concurrency.rwmutex.wlock

Acquires an exclusive write lock, blocking until all readers and any other writer have released.

## Signature

(m: rwmutex) -> void

## Behavior

Blocks until no readers hold the lock and no other writer holds it. Grants exclusive access.

## Tests

+ acquires when uncontended
+ blocks while readers hold
+ blocks while another writer holds
+ grants exclusive access once acquired
- deadlocks when the same thread re-locks non-recursively

## Assumptions

? non-recursive; caller must not hold the lock already
