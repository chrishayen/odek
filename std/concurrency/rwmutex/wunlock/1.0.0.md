# std.concurrency.rwmutex.wunlock

Releases an exclusive write lock.

## Signature

(m: rwmutex) -> void

## Behavior

Releases the writer's exclusive hold. Under writer-preference policy, waiting writers are preferred; otherwise waiting readers proceed.

## Tests

+ allows waiting writers or readers to proceed
+ restores the lock to the unlocked state
- produces undefined behavior when the caller does not hold the write lock

## Assumptions

? caller must have previously called wlock
