# std.concurrency.rwmutex.runlock

Releases a read lock on the mutex.

## Signature

(m: rwmutex) -> void

## Behavior

Decrements the reader count. When the last reader releases and a writer is waiting, the writer is unblocked.

## Tests

+ allows another reader to continue while other readers still hold
+ unblocks a waiting writer when the last reader releases
- produces undefined behavior when the caller does not hold the read lock

## Assumptions

? caller must have previously called rlock
