# std.time.format_rfc3339.format_fractional_millis

Formats a millisecond value in [0, 999] as a zero-padded three-digit decimal string.

## Signature

(n: i32) -> string

## Behavior

Returns a string of exactly three decimal digits. Pads with leading zeros for values below 100.

## Tests

+ returns "000" for input 0
+ returns "007" for input 7
+ returns "042" for input 42
+ returns "999" for input 999
- returns a string of length exactly 3 for every valid input

## Assumptions

? caller guarantees inputs are in [0, 999]
