# std.time.format_rfc3339

Formats a Unix-epoch millisecond count as an RFC 3339 UTC timestamp.

## Signature

(epoch_millis: i64) -> string

## Behavior

Converts the given epoch millisecond count to a UTC timestamp in the canonical RFC 3339 form `YYYY-MM-DDTHH:MM:SS.sssZ`. Fractional seconds are always included with three digits (milliseconds). The offset is always `Z`.

## Tests

+ formats 0 as "1970-01-01T00:00:00.000Z"
+ formats positive values with four-digit year, zero-padded month/day/hour/minute/second
+ formats negative values (pre-epoch) with four-digit year
+ always emits three fractional-second digits, even when the milliseconds are 0
+ always emits the offset as "Z"
- emits no timezone offset other than "Z"

## Assumptions

? always emits UTC; no local-time output
? pre-year-0 values extend the ISO 8601 proleptic Gregorian calendar
