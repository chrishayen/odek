# std.time.format_rfc3339.split_components

Splits an epoch-millisecond value into year, month, day, hour, minute, second, and millisecond components in UTC.

## Signature

(epoch_millis: i64) -> rfc3339_components

## Behavior

Divides the input into days-since-epoch and ms-within-day. Converts the day count to a civil date (year/month/day) using the Howard Hinnant algorithm. Splits ms-within-day into hour/minute/second/millisecond components.

## Tests

+ returns 1970-01-01 00:00:00.000 for input 0
+ returns 1970-01-02 00:00:00.000 for 86_400_000
+ returns 2020-02-29 for a leap-day epoch input
+ handles negative epoch inputs (pre-1970 dates)
- returns components whose composition equals the original input

## Assumptions

? output is always UTC; no local-time output
? proleptic Gregorian calendar
