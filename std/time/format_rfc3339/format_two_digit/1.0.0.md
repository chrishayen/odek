# std.time.format_rfc3339.format_two_digit

Formats a non-negative integer in [0, 99] as a zero-padded two-digit decimal string.

## Signature

(n: i32) -> string

## Behavior

Returns a string of exactly two decimal digits. Pads with a leading `0` when the input is a single digit. Inputs above 99 or below 0 are clamped; this rune is only called by internal formatters with already-validated inputs.

## Tests

+ returns "00" for input 0
+ returns "09" for input 9
+ returns "10" for input 10
+ returns "99" for input 99
- returns a string of length exactly 2 for every valid input

## Assumptions

? caller guarantees inputs are in [0, 99]; out-of-range clamps silently
