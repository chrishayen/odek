# std.time.now_seconds

Returns the current wall-clock time as whole seconds since the Unix epoch.

## Signature

() -> i64

## Behavior

Reads the host's real-time clock and returns the integer second count. Sub-second precision is truncated.

## Tests

+ returns a positive value on any modern host
+ returns a monotonically-non-decreasing value under a monotonic host clock
+ differs from now_millis by a factor of 1000 (within rounding)
- returns a value in non-second units under no circumstance

## Assumptions

? truncates sub-second precision, not rounds
