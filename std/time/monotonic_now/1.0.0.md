# std.time.monotonic_now

Returns a monotonic nanosecond count from an arbitrary start point, suitable for measuring elapsed time.

## Signature

() -> i64

## Behavior

Reads a monotonic clock source (not wall-clock) and returns nanoseconds since an arbitrary reference. The reference point may change between process lifetimes; differences between calls within one process are meaningful.

## Tests

+ returns a non-decreasing value across successive calls
+ differences equal the elapsed wall-time within precision
+ never jumps backward even if the system clock is adjusted
- is not comparable across process lifetimes

## Assumptions

? clock source is CLOCK_MONOTONIC on Linux, similar on other OSes
