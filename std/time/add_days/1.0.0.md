# std.time.add_days

Returns `epoch_millis` advanced by `days` (each day = 86_400 seconds).

## Signature

(epoch_millis: i64, days: i64) -> result[i64, string]

## Behavior

Delegates to `add_seconds(epoch_millis, days * 86_400)`. Does NOT account for leap seconds or daylight-saving shifts — every day is exactly 86,400 seconds.

## Tests

+ advances by 86_400_000 ms per day
+ accepts negative days
- returns error on i64 overflow

## Assumptions

? every day is 86,400 seconds; calendar-aware rune is separate
