# std.time.is_equal

Returns true when two epoch-millisecond timestamps are equal.

## Signature

(a_millis: i64, b_millis: i64) -> bool

## Behavior

Returns `a_millis == b_millis`.

## Tests

+ returns true for equal timestamps
+ returns false when they differ by any amount

## Assumptions

? compares at millisecond resolution; sub-millisecond precision is not represented
