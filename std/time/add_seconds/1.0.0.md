# std.time.add_seconds

Returns `epoch_millis + seconds * 1000`, with overflow detection.

## Signature

(epoch_millis: i64, seconds: i64) -> result[i64, string]

## Behavior

Adds `seconds * 1000` to `epoch_millis` with overflow checking. Returns the new epoch millisecond count. Returns an error on i64 overflow.

## Tests

+ adds 1 second (1000 ms) correctly
+ accepts negative seconds to go backward
+ returns 0 for add_seconds(0, 0)
- returns error on i64 overflow

## Assumptions

? no calendar-aware arithmetic; just millisecond math
