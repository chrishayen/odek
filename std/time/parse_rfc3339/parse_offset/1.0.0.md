# std.time.parse_rfc3339.parse_offset

Parses a UTC offset suffix (`Z` or `±HH:MM`) into a signed minute count.

## Signature

(token: string) -> result[i32, string]

## Behavior

`Z` or `z` returns 0. `+HH:MM` or `-HH:MM` returns `(hour * 60 + minute) * sign`. The hour portion must be 00-14; the minute portion 00-59. Any other form is an error.

## Tests

+ returns 0 for "Z" and for "z"
+ returns 120 for "+02:00"
+ returns -300 for "-05:00"
- returns error for a sign without ":"
- returns error when hour is above 14
- returns error when minute is above 59

## Assumptions

? accepted hour range caps at 14 per ISO 8601
? signs other than "+" / "-" are rejected
