# std.time.parse_rfc3339.parse_fractional_seconds

Parses an optional `.fff...` fractional-seconds component and returns milliseconds plus end position.

## Signature

(input: string, pos: i64) -> result[fractional_result, string]

## Behavior

If the byte at `pos` is `.`, reads the following digits (at least one required) and returns the first three as milliseconds, truncating any additional digits. If the byte at `pos` is not `.`, returns 0 milliseconds with `pos` unchanged.

## Tests

+ returns 0 milliseconds and unchanged pos when byte at pos is not "."
+ returns 123 milliseconds for input ".123"
+ truncates additional digits; ".123456" returns 123
+ returns 100 milliseconds for input ".1" (padded)
- returns error on "." followed by no digits
- returns error on "." followed by a non-digit character

## Assumptions

? precision is milliseconds; sub-millisecond digits are discarded
? trailing zeros within the first three digits are preserved
