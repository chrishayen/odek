# std.time.parse_rfc3339.parse_time_of_day

Parses a `HH:MM:SS` time-of-day token into hour/minute/second integers.

## Signature

(token: string) -> result[time_of_day, string]

## Behavior

Accepts an 8-character token. Requires two-digit hour (00-23), `:`, two-digit minute (00-59), `:`, two-digit second (00-59). Returns a `time_of_day` struct on success.

## Tests

+ returns (12, 0, 0) for "12:00:00"
+ accepts (23, 59, 59) as the upper boundary
+ accepts (00, 00, 00) as the lower boundary
- returns error when hour is above 23
- returns error when minute or second is above 59
- returns error when the separators are not ":"

## Assumptions

? leap seconds (second == 60) are rejected
? fractional seconds are handled by a separate rune
