# std.time.parse_rfc3339.days_since_epoch

Returns the number of days between a Gregorian date and 1970-01-01.

## Signature

(d: date) -> i64

## Behavior

Uses the Howard Hinnant civil-date algorithm to compute days since the Unix epoch. Negative values represent dates before the epoch. Returns 0 for 1970-01-01.

## Tests

+ returns 0 for 1970-01-01
+ returns 1 for 1970-01-02
+ returns -1 for 1969-12-31
+ accounts for leap years (2020-02-29 is counted as a valid day)
- returns no negative result for any date on or after the epoch

## Assumptions

? proleptic Gregorian calendar extends indefinitely in both directions
? input has already been validated by parse_date
