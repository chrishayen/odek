# std.time.now_millis

Returns the current wall-clock time as milliseconds since the Unix epoch.

## Signature

() -> i64

## Behavior

Reads the host's system clock and returns the count of whole milliseconds elapsed since 1970-01-01T00:00:00Z. The value can jump backward when the host adjusts its clock (NTP, daylight-saving, manual change).

## Tests

+ returns a value greater than zero on any modern host
+ returns a value greater than or equal to a previously returned value under a monotonic host clock
+ successive calls within the same millisecond may return the same value
- returns a time that is never expressed in a unit other than milliseconds
- returns a result that is not adjusted for the caller's local timezone

## Assumptions

? uses the host's real-time clock; not a monotonic clock
? precision is whole milliseconds; sub-millisecond granularity is dropped
