# std.time.is_after

Returns true when `a_millis` is strictly later than `b_millis`.

## Signature

(a_millis: i64, b_millis: i64) -> bool

## Behavior

Returns `a_millis > b_millis`.

## Tests

+ returns true when a is later than b
+ returns false when a equals b
+ returns false when a is earlier than b
- returns true for equal timestamps under no circumstance

## Assumptions

? strict comparison
