# std.time.now_nanos

Returns the current wall-clock time as nanoseconds since the Unix epoch.

## Signature

() -> i64

## Behavior

Reads the host's real-time clock at nanosecond resolution. Note: i64 nanoseconds ranges ~1678-2262, so overflow is a concern only outside that window.

## Tests

+ returns a positive value on any modern host
+ returns a value consistent with now_millis within 1e6
+ differs between successive calls on typical hardware
- returns a value in non-nanosecond units under no circumstance

## Assumptions

? underlying clock may have coarser resolution; returned value is the best available
