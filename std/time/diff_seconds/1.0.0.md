# std.time.diff_seconds

Returns the whole-second difference between two Unix-epoch millisecond counts.

## Signature

(later_millis: i64, earlier_millis: i64) -> i64

## Behavior

Computes `(later_millis - earlier_millis) / 1000`, truncating toward zero. The result is negative when `earlier_millis` is later than `later_millis`.

## Tests

+ returns 60 for (60000, 0)
+ returns 0 for (999, 0) — sub-second differences truncate to zero
+ returns -60 for (0, 60000) — negative when earlier is actually later
+ returns 0 when both inputs are equal
+ returns the correct value for inputs spanning a clock second boundary
- overflows i64 only when the magnitude of the difference exceeds 9.2e18 milliseconds

## Assumptions

? truncation is toward zero, matching integer division semantics
? millisecond remainders are discarded, never rounded
