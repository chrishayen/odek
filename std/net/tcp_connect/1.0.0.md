# std.net.tcp_connect

Opens a TCP connection to a remote host and port, with a bounded connect timeout.

## Signature

(host: string, port: i32, timeout_millis: i64) -> result[tcp_conn, string]

## Behavior

Resolves `host` via DNS, then attempts to establish a TCP connection to `host:port`. Fails with a timeout error if the three-way handshake does not complete within `timeout_millis`. Returns a connected `tcp_conn` value for reading and writing on success.

## Tests

+ returns a connected tcp_conn for a reachable host and port
+ fails with a timeout error when the handshake exceeds timeout_millis
+ fails with an error containing "refused" when the remote port is closed
+ follows DNS to an IP address automatically
- returns error when host fails to resolve
- returns error when port is outside 1..65535
- returns error when timeout_millis is non-positive

## Assumptions

? TCP_NODELAY is NOT enabled by default; callers opt in
? keep-alive is NOT enabled by default
