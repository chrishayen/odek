# std.net.format_ip

Formats an IP address value as its canonical string form.

## Signature

(ip: ip_address) -> string

## Behavior

IPv4 formats as dotted-decimal (e.g. "192.0.2.1"). IPv6 formats per RFC 5952 (compressed: "::1", lowercase hex, no leading zeros).

## Tests

+ formats IPv4 in dotted-decimal
+ formats IPv6 with RFC 5952 compression rules
+ lowercases IPv6 hex digits
+ round-trips: format then parse yields the original
- never emits uppercase hex

## Assumptions

? RFC 5952 canonicalization for IPv6
