# std.net.udp_send

Sends a datagram to a destination address, returning bytes sent (never partial).

## Signature

(sock: udp_socket, dest_host: string, dest_port: i32, data: bytes) -> result[i64, string]

## Behavior

Sends `data` as a single UDP datagram to `dest_host:dest_port`. Returns `len(data)` on success. UDP does not support partial sends — either the full datagram is sent or an error is returned.

## Tests

+ returns len(data) on success
+ sends the datagram to the specified destination
+ accepts an empty datagram (0 bytes is valid in UDP)
- returns error when the message exceeds the OS datagram size limit
- returns error when the socket is closed
- returns error when the destination is unroutable

## Assumptions

? UDP has no delivery guarantees; success only indicates local transmission
