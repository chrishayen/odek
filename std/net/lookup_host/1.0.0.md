# std.net.lookup_host

Resolves a hostname to a list of IP address strings.

## Signature

(host: string) -> result[list[string], string]

## Behavior

Performs DNS resolution for `host` via the host OS resolver. Returns every A and AAAA record as a string (IPv4 in dotted-decimal, IPv6 in canonical compressed form). An IP literal passed as `host` is returned as a single-element list without DNS traffic.

## Tests

+ returns a non-empty list for a resolvable hostname
+ returns a single-element list for an IPv4 literal (e.g. "1.2.3.4")
+ returns a single-element list for an IPv6 literal (e.g. "::1")
+ returns both A and AAAA entries when both exist
- returns error for a hostname with no A or AAAA records (NXDOMAIN)
- returns error on DNS server timeout
- returns error on an empty hostname

## Assumptions

? order of returned addresses is not guaranteed to be stable between calls
? does not apply happy-eyeballs ordering; callers that care should sort
