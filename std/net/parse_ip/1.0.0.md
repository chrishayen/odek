# std.net.parse_ip

Parses an IP address literal into an IPv4 or IPv6 value.

## Signature

(s: string) -> result[ip_address, string]

## Behavior

Recognizes dotted-decimal IPv4 (e.g. "192.0.2.1") and colon-separated IPv6 (e.g. "::1", "2001:db8::1"). Rejects any other form, including IPv6 with zone identifiers unless explicitly supported.

## Tests

+ parses an IPv4 dotted-decimal
+ parses IPv6 with double-colon abbreviation
+ parses IPv4-mapped IPv6 like "::ffff:1.2.3.4"
- returns error on IPv4 with out-of-range octets (e.g. "256.0.0.0")
- returns error on invalid IPv6 syntax

## Assumptions

? no zone identifier support by default
