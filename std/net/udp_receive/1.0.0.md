# std.net.udp_receive

Receives the next UDP datagram, returning the payload and the source address.

## Signature

(sock: udp_socket) -> result[udp_packet, string]

## Behavior

Blocks until a datagram arrives. Returns a struct with `data`, `source_host`, `source_port`. Truncates if the datagram exceeds the implementation's receive buffer.

## Tests

+ returns the next datagram with its source address
+ blocks until a datagram arrives
- returns error when the socket is closed
- returns an error or truncation indicator when a datagram exceeds the buffer

## Assumptions

? buffer size is implementation-defined
