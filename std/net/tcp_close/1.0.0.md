# std.net.tcp_close

Closes a TCP connection, releasing the socket.

## Signature

(conn: tcp_conn) -> result[void, string]

## Behavior

Sends a TCP FIN to the peer, closes the local socket. Subsequent reads and writes on the connection error.

## Tests

+ closes the socket cleanly
+ makes subsequent reads/writes error
- returns error when already closed

## Assumptions

? caller discards the conn after close; reuse errors
