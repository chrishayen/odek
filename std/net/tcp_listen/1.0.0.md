# std.net.tcp_listen

Opens a TCP listening socket on the given host and port.

## Signature

(host: string, port: i32) -> result[tcp_listener, string]

## Behavior

Creates a TCP socket, binds it to `host:port`, and begins listening for incoming connections. Passing `port == 0` requests an OS-assigned ephemeral port, discoverable through the listener. The host `0.0.0.0` or `::` binds to all interfaces.

## Tests

+ returns a listener bound to the requested port
+ returns a listener on an OS-assigned port when port == 0
+ binds to all interfaces for host "0.0.0.0"
+ returns a listener that accepts incoming TCP connections
- returns error when the port is already in use
- returns error when the port is below 1024 and the process lacks privilege
- returns error when the host is unresolvable

## Assumptions

? SO_REUSEADDR is enabled by default to allow quick rebind
? backlog depth defaults to a sensible value (e.g. 128)
