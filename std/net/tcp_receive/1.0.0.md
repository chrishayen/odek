# std.net.tcp_receive

Reads up to `max_length` bytes from a TCP connection. Returns empty on clean EOF.

## Signature

(conn: tcp_conn, max_length: i64) -> result[bytes, string]

## Behavior

Issues a single recv. Returns up to `max_length` bytes. Returns an empty buffer when the peer has closed the connection cleanly.

## Tests

+ returns up to max_length bytes
+ returns an empty buffer at peer-side EOF
- returns error on network failure (not EOF)
- returns error when the local side has been closed

## Assumptions

? blocks until data is available; non-blocking variant is separate
