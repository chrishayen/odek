# std.net.tcp_send

Writes bytes on a TCP connection. May write fewer than requested; returns the count.

## Signature

(conn: tcp_conn, data: bytes) -> result[i64, string]

## Behavior

Issues a single send. Returns the number of bytes actually written (may be less than `len(data)` due to kernel buffer limits). Callers loop until `data` is fully sent.

## Tests

+ returns the byte count of the send
+ accepts an empty buffer (returns 0)
- returns error when the connection has been closed by the peer or locally
- returns error on network failure

## Assumptions

? short writes are possible; use write_all style loop for exactness
