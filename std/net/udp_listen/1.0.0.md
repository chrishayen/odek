# std.net.udp_listen

Binds a UDP socket to a local address and port, returning a socket handle for send/receive.

## Signature

(host: string, port: i32) -> result[udp_socket, string]

## Behavior

Creates and binds a UDP socket. Passing port 0 requests an OS-assigned ephemeral port. `0.0.0.0` / `::` binds to all interfaces.

## Tests

+ returns a socket bound to the given port
+ uses an OS-chosen port when port == 0
+ allows send and receive on the returned socket
- returns error when the port is in use
- returns error when the port is privileged and the caller lacks privilege

## Assumptions

? SO_REUSEADDR defaults off for UDP unless the platform requires it
