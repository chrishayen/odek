# std.utf8.valid

Returns true when the byte buffer is a well-formed UTF-8 sequence.

## Signature

(data: bytes) -> bool

## Behavior

Scans the buffer and returns true iff every byte sequence decodes to a valid Unicode scalar value with minimum-length encoding. Returns true for an empty buffer. Never allocates intermediate code-point storage.

## Tests

+ returns true for empty input
+ returns true for ASCII-only input
+ returns true for multi-byte sequences representing valid scalar values
- returns false on a truncated multi-byte sequence
- returns false on an over-long encoding
- returns false on a surrogate half
- returns false on a lone continuation byte

## Assumptions

? validation is strict per the UTF-8 spec; no lenient "CESU-8" acceptance
