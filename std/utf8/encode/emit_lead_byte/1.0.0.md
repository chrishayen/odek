# std.utf8.encode.emit_lead_byte

Produces the first UTF-8 byte for a code point given its total byte count.

## Signature

(code_point: i32, byte_count: i32) -> u8

## Behavior

For byte_count=1, returns the code point itself. For 2/3/4, returns the high bits masked with `0xC0` / `0xE0` / `0xF0` and shifted down by `(byte_count - 1) * 6` bits.

## Tests

+ returns 0x41 for (0x41, 1)
+ returns 0xC3 for (0x00E9, 2)
+ returns 0xE2 for (0x20AC, 3)
+ returns 0xF0 for (0x1F600, 4)
- always returns a byte whose high bits match the byte_count's lead pattern

## Assumptions

? caller has already validated the code point via byte_count_for_code_point
