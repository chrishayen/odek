# std.utf8.encode.byte_count_for_code_point

Returns the number of UTF-8 bytes required to encode a given Unicode scalar value.

## Signature

(code_point: i32) -> result[i32, string]

## Behavior

Returns 1 for code points below 0x80, 2 for 0x80-0x7FF, 3 for 0x800-0xFFFF (excluding surrogates), 4 for 0x10000-0x10FFFF. Returns error for negative, surrogate (0xD800-0xDFFF), or out-of-range code points.

## Tests

+ returns 1 for 0x41 ("A")
+ returns 2 for 0x00E9 ("é")
+ returns 3 for 0x20AC ("€")
+ returns 4 for 0x1F600 (emoji)
- returns error for a negative code point
- returns error for a surrogate (0xD800-0xDFFF)
