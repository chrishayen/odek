# std.utf8.encode.emit_continuation_byte

Produces the `byte_index`-th UTF-8 continuation byte for a code point (1 <= byte_index < byte_count).

## Signature

(code_point: i32, byte_index: i32, byte_count: i32) -> u8

## Behavior

Computes `0x80 | ((code_point >> ((byte_count - 1 - byte_index) * 6)) & 0x3F)`. The high two bits are always `10`; the low six bits carry the relevant slice of the code point.

## Tests

+ returns 0xA9 for code_point 0x00E9, byte_index 1, byte_count 2
+ returns 0x82 for code_point 0x20AC, byte_index 1, byte_count 3
+ returns 0x9F for code_point 0x1F600, byte_index 1, byte_count 4
- always returns a byte whose top two bits are "10"

## Assumptions

? caller enforces 1 <= byte_index < byte_count
