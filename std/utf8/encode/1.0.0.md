# std.utf8.encode

Encodes a single Unicode code point into its UTF-8 byte sequence.

## Signature

(code_point: i32) -> result[bytes, string]

## Behavior

Returns the 1-to-4-byte UTF-8 encoding of the code point. Code points below 0x80 produce 1 byte; 0x80-0x7FF produce 2 bytes; 0x800-0xFFFF (excluding surrogates) produce 3 bytes; 0x10000-0x10FFFF produce 4 bytes.

## Tests

+ encodes 0x41 as single byte {0x41}
+ encodes 0x00E9 ("é") as two bytes {0xC3, 0xA9}
+ encodes 0x20AC ("€") as three bytes {0xE2, 0x82, 0xAC}
+ encodes 0x1F600 as four bytes {0xF0, 0x9F, 0x98, 0x80}
- returns error for a negative code point
- returns error for a code point above 0x10FFFF
- returns error for a surrogate code point (0xD800-0xDFFF)

## Assumptions

? surrogates are not valid Unicode scalar values and are rejected
? output is the minimum-length UTF-8 sequence (no over-long encodings)
