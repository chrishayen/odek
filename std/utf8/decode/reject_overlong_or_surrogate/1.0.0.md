# std.utf8.decode.reject_overlong_or_surrogate

Rejects UTF-8 decodings that produce an over-long encoding or a surrogate code point.

## Signature

(code_point: i32, byte_count: i32) -> result[void, string]

## Behavior

Returns error when byte_count == 2 and code_point < 0x80, or byte_count == 3 and code_point < 0x800, or byte_count == 4 and code_point < 0x10000 (all over-long). Returns error when code_point is in 0xD800-0xDFFF (surrogates). Returns success otherwise.

## Tests

+ returns success for a legitimate 3-byte code point like 0x20AC
- returns error for code_point 0x41 decoded as 2 bytes (over-long)
- returns error for code_point 0x800 decoded as 2 bytes (over-long)
- returns error for code_point 0xD800 (high surrogate)
- returns error for code_point 0xDFFF (low surrogate)

## Assumptions

? the caller has already decoded the code point from bytes; this rune only validates it
