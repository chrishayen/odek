# std.utf8.decode.byte_count_from_lead

Returns the total UTF-8 byte count for a sequence based on its leading byte.

## Signature

(lead: u8) -> result[i32, string]

## Behavior

Returns 1 when `lead < 0x80`, 2 when in `[0xC2, 0xDF]`, 3 when in `[0xE0, 0xEF]`, 4 when in `[0xF0, 0xF4]`. Returns error for any other byte (continuation bytes, invalid leads, or over-long-only leads `0xC0` / `0xC1`).

## Tests

+ returns 1 for 0x41
+ returns 2 for 0xC3
+ returns 3 for 0xE2
+ returns 4 for 0xF0
- returns error for 0x80 (a continuation byte, not a lead)
- returns error for 0xC0 (over-long only)
- returns error for 0xF8 (5-byte lead, not allowed)

## Assumptions

? 0xC0 and 0xC1 always indicate over-long encodings and are rejected as leads
