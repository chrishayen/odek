# std.utf8.decode

Decodes a UTF-8 byte buffer into a list of Unicode code points.

## Signature

(data: bytes) -> result[list[i32], string]

## Behavior

Reads the buffer left-to-right, consuming 1-to-4 bytes per code point. Validates that every multi-byte sequence uses the shortest form and produces a scalar value in [0x0000, 0xD7FF] or [0xE000, 0x10FFFF]. Rejects over-long encodings and surrogate halves.

## Tests

+ decodes {0x41, 0x42, 0x43} as [0x41, 0x42, 0x43]
+ decodes "é" bytes {0xC3, 0xA9} as [0x00E9]
+ decodes "€" bytes {0xE2, 0x82, 0xAC} as [0x20AC]
+ decodes empty input as an empty list
- returns error on a truncated multi-byte sequence
- returns error on an over-long encoding (e.g. 0xC0 0x80 for NUL)
- returns error on a lone continuation byte
- returns error on a byte that encodes a surrogate half

## Assumptions

? strict validation; never silently replaces bad bytes with U+FFFD
