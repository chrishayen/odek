# std.utf8.decode.validate_continuation

Validates that a byte is a valid UTF-8 continuation byte (10xxxxxx) and returns its 6-bit payload.

## Signature

(b: u8) -> result[u8, string]

## Behavior

Returns `b & 0x3F` when `b & 0xC0 == 0x80`. Returns error otherwise.

## Tests

+ returns 0 for 0x80
+ returns 63 for 0xBF
+ returns the low 6 bits of any valid continuation byte
- returns error for 0x7F (missing the 10 high bits)
- returns error for 0xC0 (lead-byte pattern)

## Assumptions

? only validates the bit pattern; over-long detection is a separate rune
