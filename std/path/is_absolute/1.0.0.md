# std.path.is_absolute

Returns true when the path is absolute on the host platform.

## Signature

(path: string) -> bool

## Behavior

On POSIX, returns true iff the path begins with `/`. On Windows, returns true iff the path begins with a drive letter and backslash (e.g. `C:\`) or a UNC prefix (e.g. `\\server\share`). Otherwise returns false.

## Tests

+ returns true for "/a/b" on POSIX
+ returns true for "C:\\a" on Windows
+ returns true for "\\\\server\\share\\a" on Windows (UNC)
+ returns false for "a/b" on any platform
+ returns false for "./a" on any platform
- returns false for the empty string

## Assumptions

? platform is detected at runtime from the host OS
? does not perform filesystem lookups; purely syntactic
