# std.path.ext

Returns the file extension of a path, including the leading dot.

## Signature

(path: string) -> string

## Behavior

Returns the substring from the last `.` in the final path segment to the end. When the final segment has no `.`, or the `.` is the first character of the segment (dotfile), returns the empty string.

## Tests

+ returns ".md" for input "a/b/c.md"
+ returns ".gz" for input "a/b/c.tar.gz" (last dot only)
+ returns "" for input "a/b/c" (no dot)
+ returns "" for input ".bashrc" (leading dot is not an extension)
+ returns "" for an empty input
- does not return an extension that came from a directory segment (e.g. "a.dir/b")

## Assumptions

? returns only the last extension; ".tar.gz" is considered just ".gz"
? dotfiles (segments starting with ".") have no extension
