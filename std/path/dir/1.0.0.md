# std.path.dir

Returns the parent-directory portion of a path.

## Signature

(path: string) -> string

## Behavior

Trims any trailing separators, then returns the substring before the final remaining separator. If the path has no separators, returns `.` to represent the current directory. Root paths (`/` on POSIX) return themselves.

## Tests

+ returns "a/b" for input "a/b/c"
+ returns "a/b" for input "a/b/c/"
+ returns "." for input "c" (no separators)
+ returns "/" for input "/"
+ returns "/" for input "/c"
- returns nothing containing the final segment

## Assumptions

? returns "." for input with no separators; never returns empty string for a non-empty input
? does not resolve symbolic links or ".." segments
