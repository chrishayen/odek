# std.path.with_extension

Returns `path` with its final extension replaced by `ext`. If `ext` is empty, strips the extension.

## Signature

(path: string, ext: string) -> string

## Behavior

Drops any existing extension (the part starting at the last `.` of the final segment, if not a dotfile). Appends `.` + `ext` when `ext` is non-empty.

## Tests

+ returns "a/b/c.md" for ("a/b/c.txt", "md")
+ returns "a/b/c" for ("a/b/c.txt", "")
+ returns "a/b/c.md" for ("a/b/c", "md") (adds extension)
+ preserves the path's directory portion
- does not treat a dotfile's leading "." as an extension

## Assumptions

? ext does NOT include the leading "." (function adds it)
