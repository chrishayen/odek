# std.path.normalize

Returns a syntactically equivalent path with redundant segments collapsed.

## Signature

(path: string) -> string

## Behavior

Collapses duplicate separators, `./` components, and `../` components (when they don't cross the root). Preserves leading `/` on absolute paths. Returns `.` for an otherwise-empty path.

## Tests

+ collapses "a//b" to "a/b"
+ removes "./" components
+ resolves "../" by backing up one component
+ preserves leading "/" for absolute paths
+ returns "." for an empty path
- does not consult the filesystem; purely syntactic

## Assumptions

? ".." at root of an absolute path is preserved (cannot back out)
? platform separator is honored
