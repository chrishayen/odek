# std.path.base

Returns the final segment of a path.

## Signature

(path: string) -> string

## Behavior

Trims any trailing separators, then returns the substring after the final remaining separator. If the path has no separators, returns the path unchanged. If the path is just separators, returns the separator.

## Tests

+ returns "c" for input "a/b/c"
+ returns "c" for input "a/b/c/" (trailing separators trimmed)
+ returns "c" for input "c" (no separators)
+ returns "" for an empty input
+ returns "/" for input "/"
- returns no path containing any separators

## Assumptions

? uses the host's path separator
? does not resolve symbolic links or ".." segments
