# std.path.split_ext

Splits a path into a (base, ext) pair, where ext includes the leading dot.

## Signature

(path: string) -> pair[string, string]

## Behavior

Finds the last `.` in the final segment (not a leading dotfile dot). Returns (path-without-ext, ext-with-dot). If no extension, returns (path, "").

## Tests

+ returns ("a/b/c", ".md") for "a/b/c.md"
+ returns ("a/b/c", "") when no extension
+ returns (".bashrc", "") for a dotfile
+ returns ("a/b/c.tar", ".gz") for "a/b/c.tar.gz"
- does not return ext without a leading dot when ext is non-empty

## Assumptions

? ext includes the leading "."; callers strip if they want just the name
