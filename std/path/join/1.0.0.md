# std.path.join

Joins path segments with the platform-appropriate separator, collapsing duplicates.

## Signature

(parts: list[string]) -> string

## Behavior

Concatenates segments in order with the platform's path separator (`/` on POSIX, `\` on Windows). Duplicate separators between segments are collapsed. An absolute segment in the middle of the list discards everything before it. Empty segments are skipped except when they would leave the result empty.

## Tests

+ returns "a/b/c" on POSIX for input ["a", "b", "c"]
+ collapses duplicate separators at segment boundaries (["a/", "/b"] -> "a/b")
+ discards preceding segments when a segment is absolute (["a", "/b", "c"] -> "/b/c")
+ returns "" for an empty list
+ returns a single segment unchanged when the list has one element
- does not normalize ".." or "."; those segments are preserved verbatim

## Assumptions

? uses the host's path separator
? no normalization beyond separator collapsing; use std.path.normalize for that
