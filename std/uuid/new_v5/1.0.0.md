# std.uuid.new_v5

Generates a name-based version-5 UUID (SHA-1 hash of namespace + name) in canonical string form.

## Signature

(namespace: bytes, name: bytes) -> result[string, string]

## Behavior

Computes SHA-1(namespace || name), truncates to 16 bytes, sets the version nibble to 5 and the variant bits to 10, then formats canonically.

## Tests

+ returns a 36-character UUID string
+ the version nibble is "5"
+ identical (namespace, name) inputs produce the same UUID
+ different namespaces with the same name produce different UUIDs
- returns error when namespace is not exactly 16 bytes

## Assumptions

? deterministic — not random
? uses SHA-1 (legacy but required by RFC 4122 for v5)
