# std.uuid.is_nil

Returns true when a 16-byte UUID is the all-zeros nil UUID.

## Signature

(uuid: bytes) -> bool

## Behavior

Returns true iff the input is exactly 16 bytes AND every byte is zero. Returns false for any other length or content.

## Tests

+ returns true for 16 zero bytes
+ returns false for any non-zero byte
+ returns false when input is not exactly 16 bytes

## Assumptions

? nil UUID is the canonical "unset" sentinel per RFC 4122
