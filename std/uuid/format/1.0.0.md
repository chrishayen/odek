# std.uuid.format

Formats a 16-byte UUID into its canonical lowercase hyphenated string form.

## Signature

(uuid: bytes) -> result[string, string]

## Behavior

Validates the input is exactly 16 bytes. Emits `xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx` with lowercase hex.

## Tests

+ returns a 36-character string
+ emits lowercase hex
+ emits hyphens at positions 8, 13, 18, 23
- returns error when input is not exactly 16 bytes

## Assumptions

? canonical form only; braced and URN forms are separate
