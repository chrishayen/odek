# std.uuid.new_v4

Generates a random version-4 UUID in canonical string form.

## Signature

() -> string

## Behavior

Produces a random UUID per RFC 4122 section 4.4: 122 bits of entropy with version (0100) and variant (10xx) bits set appropriately. Returns the canonical lowercase hyphenated form `xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx` where `y` is one of `8`, `9`, `a`, or `b`.

## Tests

+ returns a string of length 36
+ returns a string matching the canonical UUID v4 pattern
+ has the version nibble (13th character) set to "4"
+ has the variant nibble (17th character) set to "8", "9", "a", or "b"
+ produces values that do not collide across many invocations
- emits no uppercase hex digits

## Assumptions

? entropy source is std.crypto.random_bytes (OS CSPRNG)
? output is always lowercase
