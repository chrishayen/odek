# std.uuid.parse.decode_group

Decodes one hyphen-separated hex group from a UUID string into its raw bytes.

## Signature

(s: string) -> result[bytes, string]

## Behavior

Accepts a group of even length (8, 4, 4, 4, or 12 hex characters). Pairs the hex digits and decodes each pair via `std.hex.decode.pair_to_byte`. Returns the concatenated bytes.

## Tests

+ returns 4 bytes for an 8-character group
+ returns 2 bytes for a 4-character group
+ returns 6 bytes for a 12-character group
- returns error when the group length is odd
- returns error when any character is not a valid hex digit

## Assumptions

? caller ensures group length matches 8/4/4/4/12 per the UUID layout
