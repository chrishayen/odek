# std.uuid.parse.validate_hyphens

Checks that a candidate UUID string has the canonical 8-4-4-4-12 hyphen layout.

## Signature

(s: string) -> result[void, string]

## Behavior

Returns success when the input is exactly 36 characters and hyphens sit at positions 8, 13, 18, and 23. Returns error otherwise.

## Tests

+ returns success for a correctly hyphenated 36-character input
- returns error when the input is not 36 characters
- returns error when a hyphen is missing at position 8
- returns error when a hyphen is missing at position 13, 18, or 23
- returns error when a hyphen appears at a wrong position

## Assumptions

? only the canonical form is accepted; braced and URN forms are rejected upstream
