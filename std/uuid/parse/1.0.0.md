# std.uuid.parse

Parses a UUID string into its 16 raw bytes.

## Signature

(s: string) -> result[bytes, string]

## Behavior

Accepts the canonical hyphenated form (`8-4-4-4-12` hex digits). Accepts both lowercase and uppercase hex. Rejects the un-hyphenated 32-character form, braced forms like `{...}`, and URN forms like `urn:uuid:...`. Returns a 16-byte buffer in big-endian UUID byte order.

## Tests

+ returns 16 bytes for a valid canonical UUID
+ accepts uppercase hex digits
+ accepts lowercase hex digits
+ produces bytes in big-endian UUID byte order
- returns error on the un-hyphenated 32-character form
- returns error on braced or URN wrapped forms
- returns error when the hyphen positions are wrong
- returns error on any non-hex character

## Assumptions

? only the canonical 36-character form is accepted
? version and variant bits are NOT validated; the function returns whatever bytes decode
