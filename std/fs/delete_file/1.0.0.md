# std.fs.delete_file

Removes a file at the given path.

## Signature

(path: string) -> result[void, string]

## Behavior

Unlinks the file. Returns an error when the path does not exist, is a directory, or cannot be removed.

## Tests

+ removes an existing file
+ makes a subsequent exists() return false
- returns error when the path does not exist
- returns error when the path names a directory
- returns error when permission denied

## Assumptions

? refuses to follow through to a directory deletion; use delete_dir for that
