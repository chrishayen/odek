# std.fs.write_file

Writes a byte buffer to a file, creating or truncating it.

## Signature

(path: string, data: bytes) -> result[void, string]

## Behavior

Opens the file at `path` for writing, creating it if absent and truncating it to zero bytes if present. Writes the full buffer, then closes the file. Does not flush OS-level caches unless the host guarantees durability on close.

## Tests

+ creates a new file with the given contents when no file exists at path
+ truncates and overwrites an existing file
+ writes binary data without any encoding transformation
+ closes the file descriptor before returning
- returns error when the parent directory does not exist
- returns error when the path is not writable (permission denied)
- returns error when the path names an existing directory

## Assumptions

? writes are not atomic; a crash mid-write may leave a partial file
? does not fsync unless the host default is to fsync on close
