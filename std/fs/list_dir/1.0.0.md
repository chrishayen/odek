# std.fs.list_dir

Returns the names of every entry directly inside a directory.

## Signature

(path: string) -> result[list[string], string]

## Behavior

Reads the directory at `path` and returns each immediate child's name (without the leading path). Does not recurse into subdirectories. Does not return the special `.` and `..` entries. Order is unspecified.

## Tests

+ returns every immediate child name for a populated directory
+ returns an empty list for an empty directory
+ does not include "." or ".." entries
+ does not recurse into subdirectories
- returns error when the path does not exist
- returns error when the path is not a directory
- returns error on permission denied

## Assumptions

? order of entries is not guaranteed; callers sort if they need deterministic order
? hidden files (dotfiles on Unix) are included
