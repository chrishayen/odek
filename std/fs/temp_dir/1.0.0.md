# std.fs.temp_dir

Creates a new unique empty directory in the system's temp directory, returns its path.

## Signature

(prefix: string) -> result[string, string]

## Behavior

Creates a directory whose name starts with `prefix` plus a unique suffix. Sets restrictive permissions (0700 on Unix).

## Tests

+ returns a path that exists and is a directory
+ applies the prefix to the directory name
+ creates a distinct directory on each call
+ creates an empty directory
- returns error when the temp directory is not writable

## Assumptions

? permissions default to 0700
