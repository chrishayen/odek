# std.fs.read_file

Reads the entire contents of a file into a byte buffer.

## Signature

(path: string) -> result[bytes, string]

## Behavior

Opens the file at `path` for reading, reads all bytes to end-of-file, and returns the complete buffer. The file is closed before the function returns, whether the read succeeds or fails.

## Tests

+ returns the exact bytes of an existing file
+ returns empty bytes for a zero-length file
+ reads binary content without any encoding transformation
+ closes the file descriptor before returning
- returns error when the file does not exist
- returns error when the file is not readable (permission denied)
- returns error when the path names a directory

## Assumptions

? entire contents are loaded into memory; use a streaming API for very large files
? path is interpreted relative to the process's current working directory when not absolute
