# std.fs.is_dir

Returns true when the path exists and is a directory.

## Signature

(path: string) -> bool

## Behavior

Calls `stat`. Returns true when the kind is directory. Returns false for any other kind, non-existence, or permission errors.

## Tests

+ returns true for an existing directory
+ returns false for a file
+ returns false for a non-existent path
+ returns false for a broken symlink

## Assumptions

? follows symbolic links to their targets
