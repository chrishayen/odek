# std.fs.make_dir

Creates a directory at the given path, optionally creating intermediate parents.

## Signature

(path: string, recursive: bool) -> result[void, string]

## Behavior

Creates a directory at `path`. When `recursive` is true, missing intermediate parent directories are also created. When `recursive` is true and the final directory already exists, the call succeeds. When `recursive` is false and the final directory already exists, the call fails.

## Tests

+ creates a single directory when the parent exists
+ creates intermediate parents when recursive is true
+ succeeds when the directory already exists and recursive is true
- returns error when recursive is false and the directory already exists
- returns error when recursive is false and the parent does not exist
- returns error on permission denied

## Assumptions

? default permissions follow the process umask
? succeeds silently on idempotent recursive mkdir (matches `mkdir -p`)
