# std.fs.temp_file

Creates a new unique empty file in the system's temp directory, returns its path.

## Signature

(prefix: string) -> result[string, string]

## Behavior

Creates a file whose name starts with `prefix` and ends with an implementation-chosen unique suffix. Ensures the name doesn't collide with existing files. Sets restrictive permissions (0600 on Unix).

## Tests

+ returns a path that exists and is a regular file
+ applies the prefix to the filename
+ creates a distinct file on each call
+ creates an empty (zero-byte) file
- returns error when the temp directory is not writable

## Assumptions

? temp directory is located via the OS convention (TMPDIR, /tmp, etc.)
? permissions default to 0600
