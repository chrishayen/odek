# std.fs.exists

Returns true when a filesystem entry exists at the given path.

## Signature

(path: string) -> bool

## Behavior

Performs a stat of `path`. Returns true for any entry type — file, directory, symbolic link (to an existing target), named pipe, or socket. Broken symlinks return false because the stat follows the link target.

## Tests

+ returns true for an existing file
+ returns true for an existing directory
+ returns true for a symlink whose target exists
+ returns false for a symlink whose target does not exist
+ returns false for a path whose parent directory does not exist
- returns false for any path the process cannot stat (permission denied returns false, not an error)

## Assumptions

? follows symbolic links; broken symlinks return false
? permission-denied during stat is reported as "not exists" rather than surfaced as an error
