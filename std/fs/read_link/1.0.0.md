# std.fs.read_link

Returns the target path that a symbolic link refers to.

## Signature

(path: string) -> result[string, string]

## Behavior

Reads the immediate target of the symlink at `path` (does not resolve further). Returns the target as-is (may be relative or absolute).

## Tests

+ returns the target of a symlink
+ returns relative targets verbatim
+ does not recurse into nested symlinks
- returns error when the path is not a symlink
- returns error when the path does not exist

## Assumptions

? returns the raw target; caller resolves if needed
