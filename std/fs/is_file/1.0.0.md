# std.fs.is_file

Returns true when the path exists and is a regular file.

## Signature

(path: string) -> bool

## Behavior

Calls `stat`. Returns true when the kind is regular file. Returns false for directories, other kinds, non-existence, or permission errors.

## Tests

+ returns true for a regular file
+ returns false for a directory
+ returns false for a non-existent path

## Assumptions

? follows symbolic links to their targets
