# std.fs.symlink

Creates a symbolic link at `link_path` pointing at `target`.

## Signature

(target: string, link_path: string) -> result[void, string]

## Behavior

Creates a new symlink whose content is `target`. Does not require `target` to exist. `link_path` must not already exist.

## Tests

+ creates a symlink with the given target
+ allows a relative or absolute target
+ allows a non-existent target (dangling symlink)
- returns error when link_path already exists
- returns error when the caller lacks privilege to create symlinks (Windows)

## Assumptions

? dangling symlinks are allowed
? on Windows requires developer mode or admin privilege
