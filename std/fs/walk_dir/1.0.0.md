# std.fs.walk_dir

Returns every descendant path under `root` in depth-first pre-order.

## Signature

(root: string) -> result[list[string], string]

## Behavior

Traverses subdirectories recursively. Returns the list of absolute (or root-relative) paths for every file and subdirectory discovered. Does not follow symbolic links.

## Tests

+ returns every descendant path
+ returns subdirectories before their contents (pre-order) or after (implementation choice)
+ does not follow symbolic links
+ returns an empty list for an empty directory
- returns error when root does not exist
- returns error when root is not a directory

## Assumptions

? symlinks are not followed to avoid cycles
? order is deterministic within a single platform; cross-platform order may vary
