# std.fs.append_file

Appends bytes to a file, creating it if absent.

## Signature

(path: string, data: bytes) -> result[void, string]

## Behavior

Opens the file at `path` for appending. Writes every byte of `data` at the end. Creates the file when absent. Closes before returning.

## Tests

+ appends to an existing file, preserving prior contents
+ creates the file when absent
+ closes before returning
- returns error when the parent directory does not exist
- returns error when the path is not writable

## Assumptions

? append is atomic only up to PIPE_BUF bytes on POSIX; larger writes may interleave with concurrent appenders
