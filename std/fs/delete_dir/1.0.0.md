# std.fs.delete_dir

Removes a directory; optionally removes its contents recursively.

## Signature

(path: string, recursive: bool) -> result[void, string]

## Behavior

When `recursive` is false, removes an empty directory only. When true, removes the directory and every entry inside.

## Tests

+ removes an empty directory when recursive is false
+ removes a non-empty directory when recursive is true
+ removes subdirectories recursively
- returns error when recursive is false and the directory is non-empty
- returns error when the path does not exist
- returns error when the path names a file

## Assumptions

? follows symbolic links only within the given path, not the target
