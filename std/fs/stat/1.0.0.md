# std.fs.stat

Returns metadata (size, mtime, kind, permissions) for a filesystem entry.

## Signature

(path: string) -> result[file_metadata, string]

## Behavior

Follows symbolic links. Returns a struct carrying byte size, modification time (epoch seconds), kind (file / dir / symlink / other), and Unix-style permission bits. Use a separate rune for lstat-style (no link following).

## Tests

+ returns a struct with size, mtime, and kind
+ follows symbolic links to their targets
+ returns kind=file for a regular file
+ returns kind=dir for a directory
- returns error when the path does not exist
- returns error when permission denied during stat

## Assumptions

? follows symlinks; lstat variant is separate
? mtime is seconds since epoch
