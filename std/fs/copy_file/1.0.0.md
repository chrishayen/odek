# std.fs.copy_file

Copies a file from `src` to `dst`, overwriting any existing destination. Returns bytes copied.

## Signature

(src: string, dst: string) -> result[i64, string]

## Behavior

Opens `src` for reading, creates or truncates `dst`, and streams bytes end-to-end. Returns the total byte count on success. Permissions of the destination default to the umask; mode preservation is not guaranteed.

## Tests

+ copies the contents of src to dst
+ overwrites an existing dst
+ returns the correct byte count
- returns error when src does not exist
- returns error when dst's parent directory does not exist

## Assumptions

? does not preserve permissions, timestamps, or extended attributes by default
