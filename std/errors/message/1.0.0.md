# std.errors.message

Returns the top-level message of an error, without walking the cause chain.

## Signature

(err: error) -> string

## Behavior

Returns the message string passed when the error was constructed. Does not include causes or stack information.

## Tests

+ returns the message for an error built via errors.new
+ returns the wrap-level message for an error built via errors.wrap (not the cause's message)
+ returns an empty string for an error built from an empty message
- returns no characters contributed by the cause chain

## Assumptions

? only the immediate message is returned; callers walk the chain themselves if needed
