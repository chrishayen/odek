# std.errors.new

Constructs a new error value carrying a human-readable message and no cause chain.

## Signature

(message: string) -> error

## Behavior

Returns a freshly allocated error value whose message is the input string and whose cause is none.

## Tests

+ returns an error whose message equals the input
+ returns an error with no cause
+ returns a distinct error value on every call, even for identical messages
- returns no value with a cause already attached

## Assumptions

? message is stored verbatim; no implicit prefix or suffix
