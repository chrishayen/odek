# std.errors.is

Returns true when an error or any error in its cause chain equals a target error value.

## Signature

(err: error, target: error) -> bool

## Behavior

Walks the cause chain starting at `err`. Returns true on the first error that equals `target` by value. Returns false if the chain ends without a match.

## Tests

+ returns true when err itself equals target
+ returns true when a wrapped cause of err equals target
+ returns true when target is reached through multiple wrap levels
- returns false when target appears nowhere in the chain
- returns false when target is not a valid error reference

## Assumptions

? equality is value equality on the error identifier, not message string comparison
