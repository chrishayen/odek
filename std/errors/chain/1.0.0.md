# std.errors.chain

Returns the messages of an error and every cause in its chain, innermost last.

## Signature

(err: error) -> list[string]

## Behavior

Starts with `message(err)` as the first element. Calls `unwrap` repeatedly, appending each cause's message until unwrap returns none. The returned list is top-level first, innermost cause last.

## Tests

+ returns a one-element list for an error with no cause
+ returns a two-element list for a single wrap
+ returns a list ordered top-level first, innermost last
+ returns an empty string in the list when an error's message is empty
- returns no cycle expansion — callers with cyclic chains receive bounded output via a safety cap

## Assumptions

? cyclic chains are capped at a default depth (e.g. 64) to prevent runaway
