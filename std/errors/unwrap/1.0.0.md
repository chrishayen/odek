# std.errors.unwrap

Returns the immediate cause of a wrapped error, or none for an error built without a cause.

## Signature

(err: error) -> optional[error]

## Behavior

Returns `some(cause)` when the error was built via `wrap`. Returns `none` when the error was built via `new`. Does not recurse — returns only the single-step cause.

## Tests

+ returns some(cause) for an error built via errors.wrap
+ returns none for an error built via errors.new
+ returns none after reaching the root of a multi-level chain
- returns no deeper cause — only the immediate one

## Assumptions

? callers walk the chain themselves by calling unwrap repeatedly
