# std.errors.wrap

Returns a new error that carries a message and points at a prior error as its cause.

## Signature

(cause: error, message: string) -> error

## Behavior

Returns a fresh error whose message is the input string and whose cause is the supplied `cause` error. Does not mutate the original cause.

## Tests

+ returns an error whose message equals the input
+ returns an error whose cause is the supplied error
+ does not mutate the cause error
- returns no error whose cause is a different value than the supplied one

## Assumptions

? wrapping is non-destructive; the original cause remains unchanged
