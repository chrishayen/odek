# std.float.format_exponent

Formats an f64 in scientific-notation form with a fixed number of fractional digits.

## Signature

(n: f64, digits: i32) -> string

## Behavior

Rounds `n` to `digits` fractional digits in the significand and emits `d.ddd...e±N` notation. Exponent is printed with at least one digit and always has a sign.

## Tests

+ returns "3.14e+00" for (3.14, 2)
+ returns "1.00e+03" for (1000.0, 2)
+ returns "-1.00e-03" for (-0.001, 2)
+ always emits a sign in the exponent
- never uses fixed notation

## Assumptions

? digits is clamped to [0, 17]
? exponent is always decimal
