# std.float.is_nan

Returns true when an f64 is any NaN value (quiet or signaling).

## Signature

(n: f64) -> bool

## Behavior

Returns true iff `n != n` (the only value that is not equal to itself). Covers both quiet and signaling NaNs.

## Tests

+ returns true for a NaN
+ returns false for 0.0
+ returns false for positive and negative infinity
+ returns false for any finite value
- returns true for 0.0 under no circumstance

## Assumptions

? no distinction between quiet and signaling NaN
