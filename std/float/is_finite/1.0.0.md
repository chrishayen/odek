# std.float.is_finite

Returns true when an f64 is neither NaN nor infinity.

## Signature

(n: f64) -> bool

## Behavior

Returns true when `n` is a finite value (including subnormals and zero), false for NaN and both infinities.

## Tests

+ returns true for 0.0 and -0.0
+ returns true for any normal finite value
+ returns true for subnormal values
+ returns false for positive and negative infinity
+ returns false for NaN
- returns true for any non-finite value under no circumstance

## Assumptions

? subnormals count as finite
