# std.float.format_fixed

Formats an f64 in fixed-point notation with a fixed number of fractional digits.

## Signature

(n: f64, digits: i32) -> string

## Behavior

Rounds `n` to `digits` fractional digits under round-to-nearest-even and emits the decimal representation. Negative values have a leading `-`. Never uses exponent notation.

## Tests

+ returns "3.14" for (3.14159, 2)
+ returns "3.00" for (3.0, 2) — always emits digits trailing zeros
+ returns "0" for (0.0, 0)
+ rounds ties to even (e.g. 2.5 with 0 digits → "2")
+ emits "-0.00" when the input is negative zero and digits > 0
- never uses exponent notation

## Assumptions

? digits is clamped to [0, 17]
? rounding is round-to-nearest-even
