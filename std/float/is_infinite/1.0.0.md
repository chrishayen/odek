# std.float.is_infinite

Returns true when an f64 is positive or negative infinity.

## Signature

(n: f64) -> bool

## Behavior

Returns true when `n` is `+Infinity` or `-Infinity`, false for NaN and all finite values.

## Tests

+ returns true for positive infinity
+ returns true for negative infinity
+ returns false for NaN
+ returns false for 0.0
+ returns false for any finite value
- returns true for any finite value under no circumstance

## Assumptions

? both infinities return true; sign is not distinguished
