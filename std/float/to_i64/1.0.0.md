# std.float.to_i64

Converts an f64 to an i64 by truncating toward zero. Fails on non-finite inputs or values outside the i64 range.

## Signature

(n: f64) -> result[i64, string]

## Behavior

Truncates `n` toward zero (drops fractional part) and returns the integer. Returns an error when `n` is NaN, infinite, or falls outside the inclusive range of i64 after truncation.

## Tests

+ returns 3 for 3.7
+ returns -3 for -3.7
+ returns 0 for 0.0, -0.0, and values in (-1, 1)
- returns error for NaN
- returns error for positive or negative infinity
- returns error for values above i64::MAX
- returns error for values below i64::MIN

## Assumptions

? truncation is toward zero, not toward negative infinity
