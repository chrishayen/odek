# std.float.from_bits

Constructs an f64 from a raw IEEE-754 64-bit pattern.

## Signature

(bits: u64) -> f64

## Behavior

Interprets `bits` as the binary representation of a double-precision float. Returns the corresponding f64, including NaN values with their payload preserved.

## Tests

+ returns 0.0 for 0
+ returns -0.0 for 0x8000000000000000
+ returns positive infinity for 0x7FF0000000000000
+ returns negative infinity for 0xFFF0000000000000
+ round-trips with to_bits
- returns a value other than NaN for a NaN bit pattern under no circumstance

## Assumptions

? bit layout is IEEE-754 double precision
