# std.float.to_bits

Returns the IEEE-754 64-bit binary representation of an f64 as a u64.

## Signature

(n: f64) -> u64

## Behavior

Returns the raw 64-bit pattern: 1 sign bit, 11 exponent bits, 52 significand bits. Preserves NaN payload bits.

## Tests

+ returns 0 for 0.0
+ returns 0x8000000000000000 for -0.0
+ returns 0x7FF0000000000000 for positive infinity
+ returns 0xFFF0000000000000 for negative infinity
+ round-trips: from_bits(to_bits(n)) equals n (or is NaN when n is)
- does not lose NaN payload bits

## Assumptions

? bit layout is IEEE-754 double precision, big-endian logical bit positions
