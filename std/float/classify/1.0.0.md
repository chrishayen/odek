# std.float.classify

Returns a tag categorizing an f64 as one of: nan, infinite, zero, subnormal, or normal.

## Signature

(n: f64) -> float_class

## Behavior

Inspects the IEEE-754 fields of `n` and returns the appropriate variant. NaN is checked first; then infinity; then zero; then subnormal (exponent 0, non-zero significand); else normal.

## Tests

+ returns nan for any NaN value
+ returns infinite for positive and negative infinity
+ returns zero for 0.0 and -0.0
+ returns subnormal for a subnormal value
+ returns normal for any normal finite value
- returns normal for 0.0 under no circumstance

## Assumptions

? sign is not exposed through this classification; callers check separately
