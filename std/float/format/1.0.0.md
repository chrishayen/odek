# std.float.format

Formats an f64 into the shortest round-trippable decimal representation.

## Signature

(n: f64) -> string

## Behavior

Returns the shortest decimal string that parses back to the same f64 under round-to-nearest-even. Uses fixed notation for values in a reasonable range; exponent notation otherwise. Emits "Infinity", "-Infinity", "NaN" for special values.

## Tests

+ returns "3.14" for 3.14
+ returns "0" for 0.0
+ returns "-0" for -0.0
+ returns "Infinity" for positive infinity
+ returns "NaN" for a NaN
+ returns a string that parses back to the same f64
- does not emit trailing zeros

## Assumptions

? round-trippable is the Grisu/Ryu shortest-decimal algorithm
