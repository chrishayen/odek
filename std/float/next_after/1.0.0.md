# std.float.next_after

Returns the next representable f64 value from `x` in the direction of `y`.

## Signature

(x: f64, y: f64) -> f64

## Behavior

Returns the adjacent f64 toward `y`. Returns `y` when `x == y`. Returns NaN when either input is NaN. Steps by one ULP in the requested direction.

## Tests

+ returns a value larger than x when y > x
+ returns a value smaller than x when y < x
+ returns y when x equals y
+ returns NaN when either input is NaN
- returns x unchanged when stepping in the specified direction is possible

## Assumptions

? steps across 0 handle the sign change
? stepping above +Infinity or below -Infinity returns the infinity
