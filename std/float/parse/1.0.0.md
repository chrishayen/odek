# std.float.parse

Parses a decimal or exponent-form float literal into an f64.

## Signature

(s: string) -> result[f64, string]

## Behavior

Accepts optional sign, integer part, optional fraction (`.`+digits), optional exponent (`e`/`E` + optional sign + digits). Rejects empty input and malformed literals. Parses "inf", "infinity", "nan" (case-insensitive) into the corresponding IEEE-754 values.

## Tests

+ returns 3.14 for "3.14"
+ returns 1e3 for "1e3"
+ returns 0.0 for "0"
+ returns positive infinity for "inf" and "Infinity"
+ returns a NaN for "nan"
- returns error on empty input
- returns error on a lone "."
- returns error on a trailing "e" with no exponent digits

## Assumptions

? rounding follows IEEE-754 round-to-nearest-even
? "inf" and "nan" are accepted in any case
