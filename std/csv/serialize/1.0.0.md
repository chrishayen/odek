# std.csv.serialize

Serializes a list of rows into RFC 4180 CSV text.

## Signature

(rows: list[list[string]]) -> string

## Behavior

Emits each row as a comma-separated line terminated by CRLF. Fields that contain a comma, double-quote, CR, or LF are wrapped in double quotes; embedded double-quotes are escaped by doubling them. Fields with no special characters are emitted unquoted. An empty rows list produces an empty string.

## Tests

+ serializes [["a","b"],["c","d"]] to "a,b\r\nc,d\r\n"
+ quotes a field containing a comma
+ quotes a field containing a double-quote and doubles the inner quote
+ quotes a field containing a newline
+ emits an empty string for an empty list
- emits no trailing newline for an empty rows list

## Assumptions

? line terminator is CRLF per RFC 4180
? field delimiter is comma
