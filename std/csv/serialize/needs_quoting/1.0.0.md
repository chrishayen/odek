# std.csv.serialize.needs_quoting

Returns true when a CSV field value must be wrapped in double-quotes to round-trip.

## Signature

(field: string) -> bool

## Behavior

Returns true if the field contains `,`, `"`, CR, or LF. Returns false otherwise.

## Tests

+ returns true when the field contains a comma
+ returns true when the field contains a double-quote
+ returns true when the field contains CR or LF
+ returns false for a field containing only letters and digits
- returns false for an empty string

## Assumptions

? delimiter is assumed to be comma; other delimiters require a parameterized variant
