# std.csv.serialize.serialize_row

Serializes a list of field values into one CSV row terminated by CRLF.

## Signature

(fields: list[string]) -> string

## Behavior

For each field, emits `quote_field(field)` when `needs_quoting(field)` is true, otherwise the field verbatim. Separates fields with `,`. Appends `\r\n` at the end.

## Tests

+ serializes ["a","b","c"] as "a,b,c\r\n"
+ quotes fields that contain commas or quotes
+ emits unquoted fields verbatim
+ always terminates with CRLF
- does not quote fields that do not require it

## Assumptions

? line terminator is CRLF per RFC 4180
? no header row is emitted; caller decides
