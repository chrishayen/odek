# std.csv.serialize.quote_field

Returns a double-quoted CSV field with embedded double-quotes doubled.

## Signature

(field: string) -> string

## Behavior

Emits `"`, then the input with every `"` replaced by `""`, then `"`. Always emits both surrounding quotes, even for an empty field.

## Tests

+ returns "\"\"" for the empty string
+ returns "\"a,b\"" for "a,b"
+ returns "\"he said \"\"hi\"\"\"" for 'he said "hi"'
+ preserves embedded CR and LF verbatim inside the quotes
- emits no unescaped `"` inside the quoted region

## Assumptions

? caller has already decided this field needs quoting (via needs_quoting)
