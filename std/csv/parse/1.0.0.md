# std.csv.parse

Parses RFC 4180 CSV text into a list of rows, where each row is a list of string fields.

## Signature

(text: string) -> result[list[list[string]], string]

## Behavior

Splits the input on unquoted newlines (CR, LF, or CRLF) and fields on unquoted commas. Double-quoted fields may contain embedded commas, newlines, and escaped double-quotes (`""`). The field delimiter is a literal comma. Trailing newlines do not produce an extra empty row.

## Tests

+ parses "a,b,c" into [["a","b","c"]]
+ parses "a,b\nc,d" into [["a","b"],["c","d"]]
+ parses a quoted field containing a comma: "\"a,b\",c" into [["a,b","c"]]
+ parses an escaped quote: "\"he said \"\"hi\"\"\"" into [["he said \"hi\""]]
+ accepts CRLF line endings and normalizes them to row boundaries
- returns error on an unterminated quoted field
- returns error when a quote appears mid-field outside a quoted context

## Assumptions

? delimiter is fixed at comma; use a configurable reader for other delimiters
? no header-row inference; caller decides whether the first row is a header
