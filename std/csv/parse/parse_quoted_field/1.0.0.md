# std.csv.parse.parse_quoted_field

Reads one double-quoted CSV field and returns its decoded value and end position.

## Signature

(input: string, pos: i64) -> result[field_result, string]

## Behavior

Expects `"` at `pos`. Reads characters until the next unescaped `"`. Decodes `""` as a literal `"`. Returns the decoded field value (without the surrounding quotes) and the index immediately after the closing quote.

## Tests

+ decodes "\"hello\"" to "hello"
+ decodes "\"a,b\"" to "a,b" (embedded comma preserved)
+ decodes "\"he said \"\"hi\"\"\"" to 'he said "hi"'
+ decodes "\"\"" to the empty string
- returns error on an unterminated quoted field
- returns error when a non-`"` byte appears immediately after the closing quote (must be `,`, CR, LF, or EOF)

## Assumptions

? newlines (CR, LF, CRLF) embedded inside a quoted field are preserved verbatim
