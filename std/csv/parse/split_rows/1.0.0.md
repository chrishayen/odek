# std.csv.parse.split_rows

Parses a full CSV document into a list of rows by repeatedly calling `parse_row`.

## Signature

(input: string) -> result[list[list[string]], string]

## Behavior

Calls `parse_row` from position 0, then again from the returned end position, until end of input. Returns the accumulated list of rows. A trailing row terminator does not produce an extra empty row.

## Tests

+ parses "a,b\nc,d\n" into [["a","b"],["c","d"]]
+ does not produce a trailing empty row for a CRLF-terminated document
+ returns an empty list for an empty input
- returns error when any row fails to parse

## Assumptions

? an input without a final row terminator still yields its final row
? mixed CRLF and LF line endings within one document are tolerated
