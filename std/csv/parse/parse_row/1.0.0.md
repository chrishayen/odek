# std.csv.parse.parse_row

Reads one CSV row as a list of field values and returns the list and end position.

## Signature

(input: string, pos: i64) -> result[row_result, string]

## Behavior

Loops: look at the byte at `pos`; dispatch to `parse_quoted_field` if it is `"`, else `parse_field`. Consume the following `,` to continue or CR/LF/CRLF to end the row. Returns the accumulated list of field values and the index after the row terminator (or end of input).

## Tests

+ parses "a,b,c" into ["a", "b", "c"]
+ parses "a,\"b,c\",d" into ["a", "b,c", "d"]
+ returns a single-element list when the row has no commas
+ consumes CRLF as a single row terminator
- returns error when a field parser rejects its input

## Assumptions

? empty rows (a bare CRLF) return [""], not []
? field delimiter is comma; other delimiters require a separate rune
