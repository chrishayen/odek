# std.csv.parse.parse_field

Reads one unquoted CSV field starting at a cursor and returns its value and end position.

## Signature

(input: string, pos: i64) -> result[field_result, string]

## Behavior

Scans bytes until the next `,`, CR, LF, or end of input. Returns the bytes read (exclusive of the terminator) as the field value. Returns the index of the terminator (or end of input).

## Tests

+ returns "a" for input "a,b" at pos 0
+ returns "a" for input "a\nb" at pos 0
+ returns the full remainder when no terminator appears
+ returns "" for an empty field (terminator immediately at pos)
- returns error when a `"` appears inside an unquoted field

## Assumptions

? unquoted fields cannot contain `"`, comma, CR, or LF
? surrounding whitespace is preserved, not trimmed
