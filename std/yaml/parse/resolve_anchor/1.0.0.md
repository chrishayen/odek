# std.yaml.parse.resolve_anchor

Looks up a previously-defined YAML anchor by name and returns the value it binds.

## Signature

(anchor_name: string, anchors: map[string, yaml_value]) -> result[yaml_value, string]

## Behavior

Returns the value bound to `anchor_name` in the anchors map. Returns an error when no such anchor exists. The caller has already populated the map with every anchor seen earlier in the document.

## Tests

+ returns the bound value when the anchor name is present
- returns error when the anchor name is unknown
- returns error on an empty anchor name

## Assumptions

? anchor scope is document-wide; callers manage the map across one document
? alias cycles are prevented by the caller, not here
