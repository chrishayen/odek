# std.yaml.parse.parse_value

Parses any single YAML value — the dispatcher that selects block or flow form based on the context.

## Signature

(input: string, pos: i64, indent: i32) -> result[value_result, string]

## Behavior

Inspects the byte at `pos`: `{` dispatches to flow mapping, `[` to flow sequence, `- ` at line start to block sequence, plain or quoted content followed by `:` at the line's end to block mapping, everything else to `parse_scalar`. Returns the parsed value and the position after it.

## Tests

+ dispatches to parse_flow_mapping for a leading "{"
+ dispatches to parse_block_sequence when the line begins with "- "
+ dispatches to parse_scalar for a plain inline token
- returns error when no rule applies at the current position
- returns error when recursion depth exceeds the implementation limit

## Assumptions

? recursion depth is bounded (default 64)
