# std.yaml.parse.scan_indent

Counts the leading spaces at the start of a YAML line.

## Signature

(input: string, pos: i64) -> result[i32, string]

## Behavior

Assumes `pos` is at the start of a line. Counts spaces until a non-space byte or end-of-input. Returns the count. Returns an error if the indentation includes a tab — YAML forbids tab indentation in block context.

## Tests

+ returns 0 when the first byte is not a space
+ returns 4 when four spaces precede the first non-space
+ returns the full run length when the entire line is spaces
- returns error when the indentation contains a tab
- returns 0 at end-of-input

## Assumptions

? only ASCII space (0x20) counts as indentation
? tabs in indentation are always rejected per YAML 1.2
