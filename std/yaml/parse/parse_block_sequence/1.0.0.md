# std.yaml.parse.parse_block_sequence

Parses a block-style YAML sequence (`- item` lines at a common indent).

## Signature

(input: string, pos: i64, base_indent: i32) -> result[sequence_result, string]

## Behavior

Reads lines while their indent equals `base_indent` and they start with `- `. Each line's remainder (after the `- `) is parsed as a value via `parse_value`. A bare `-` on its own line introduces a deeper-indented value.

## Tests

+ parses "- a\n- b\n- c\n" as a three-element sequence
+ parses nested mappings under sequence entries
+ returns when the next line's indent drops below base_indent
- returns error when a line does not start with "- " at the expected indent

## Assumptions

? two spaces between `-` and the value content are the convention; one space is also accepted
