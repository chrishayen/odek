# std.yaml.parse

Parses a YAML 1.2 document into a tagged value tree.

## Signature

(raw: string) -> result[yaml_value, string]

## Behavior

Consumes a YAML 1.2 core-schema document. Produces a `yaml_value` that can be a null, bool, int, float, string, list, or map. Accepts block-style, flow-style, and mixed inputs. When the input contains multiple documents, only the first is returned.

## Tests

+ parses a single key/value mapping like "a: 1"
+ parses a sequence like "- a\n- b\n"
+ parses nested structures with mixed block and flow style
+ parses "~" and "null" as null
+ parses "true"/"false" as bool
- returns error on tab indentation in block mappings
- returns error on inconsistent indentation
- returns only the first document when the input contains "---" separators

## Assumptions

? uses the YAML 1.2 core schema; not the YAML 1.1 "bool-ish" schema (no "yes/no/on/off")
? anchors and aliases are resolved during parsing
