# std.yaml.parse.parse_flow_sequence

Parses a flow-style YAML sequence (JSON-compatible bracket form).

## Signature

(input: string, pos: i64) -> result[sequence_result, string]

## Behavior

Expects `[` at `pos`. Reads values separated by `,`, until `]`. Inside the brackets, newlines are treated as whitespace. Empty `[]` returns an empty sequence.

## Tests

+ parses "[]" as an empty sequence
+ parses "[1, 2, 3]" as a three-element sequence
+ allows newlines between elements
- returns error on a trailing comma
- returns error on an unclosed bracket

## Assumptions

? flow style is a strict JSON-compatible subset
