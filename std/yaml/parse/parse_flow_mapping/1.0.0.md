# std.yaml.parse.parse_flow_mapping

Parses a flow-style YAML mapping (JSON-compatible brace form).

## Signature

(input: string, pos: i64) -> result[mapping_result, string]

## Behavior

Expects `{` at `pos`. Reads pairs `key: value` separated by `,`, until `}`. Inside the braces, newlines are treated as whitespace. Empty `{}` returns an empty mapping.

## Tests

+ parses "{}" as an empty mapping
+ parses "{a: 1, b: 2}" as a two-entry mapping
+ allows arbitrary whitespace (including newlines) between tokens
- returns error on a missing colon
- returns error on a trailing comma
- returns error on an unclosed brace

## Assumptions

? flow style accepts JSON-compatible inputs as a strict subset
