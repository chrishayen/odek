# std.yaml.parse.parse_block_mapping

Parses a block-style YAML mapping (`key: value` lines at a common indent).

## Signature

(input: string, pos: i64, base_indent: i32) -> result[mapping_result, string]

## Behavior

Reads lines while their indent equals `base_indent`. Each line is `key: value` — the key is a plain scalar up to `:`, the value is either the rest of the line (via `parse_scalar`) or a nested block/flow structure starting on the next line at deeper indent.

## Tests

+ parses "a: 1\nb: 2\n" as a two-entry mapping
+ parses a nested mapping when a value line is empty and the next line is deeper-indented
+ returns when the next line's indent drops below base_indent
- returns error when a line is missing ":"
- returns error when two keys in the same mapping are equal

## Assumptions

? duplicate keys are rejected, not last-wins (YAML 1.2 rule)
? comments are stripped before this rune is called
