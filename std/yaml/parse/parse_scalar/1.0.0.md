# std.yaml.parse.parse_scalar

Resolves a plain (unquoted) YAML scalar token into a typed value per the core schema.

## Signature

(token: string) -> yaml_value

## Behavior

Recognizes `null` / `Null` / `NULL` / `~` / empty → null; `true`/`True`/`TRUE`/`false`/`False`/`FALSE` → bool; decimal, hex, octal, or float literals → number; everything else → string (the token verbatim).

## Tests

+ returns null for "null", "Null", "NULL", "~", and empty
+ returns bool true for "true" / "True" / "TRUE"
+ returns int 42 for "42"
+ returns float 3.14 for "3.14"
+ returns string "hello" for "hello"
- does not recognize "yes" / "no" / "on" / "off" as booleans (those are YAML 1.1)

## Assumptions

? uses the YAML 1.2 core schema, not the YAML 1.1 bool-ish schema
? quoted scalars are handled by a separate rune
