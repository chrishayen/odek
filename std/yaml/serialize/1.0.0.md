# std.yaml.serialize

Serializes a tagged value tree into a YAML 1.2 block-style document.

## Signature

(value: yaml_value) -> string

## Behavior

Emits a YAML document in block style with two-space indentation. Strings that are unambiguous bare scalars emit unquoted; ambiguous strings (e.g. "true", "null", anything with leading whitespace, anything containing `: ` or `#`) are emitted double-quoted. Map keys are emitted in insertion order. The output ends with a trailing newline.

## Tests

+ emits a top-level map with each key on its own line
+ emits a top-level list with "- " entries
+ quotes strings that look like booleans, nulls, or numbers
+ indents nested children by two spaces per level
+ appends a trailing newline
- emits no "---" document marker for single-document output

## Assumptions

? indentation is fixed at two spaces
? flow style (JSON-compatible) is never emitted; always block style
