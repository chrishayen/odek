# std.yaml.serialize.serialize_scalar

Renders a scalar YAML value into its string form, double-quoting when needed for disambiguation.

## Signature

(value: yaml_value) -> string

## Behavior

Null → `~`; bool → `true`/`false`; number → its shortest decimal form; string → the raw string if safe, else double-quoted with `"` escapes. Uses `quote_if_ambiguous` to decide on quoting.

## Tests

+ returns "~" for null
+ returns "true" / "false" for bool
+ returns "hello" unquoted for a safe string
+ returns "\"true\"" (quoted) for the string "true"
- emits no quotes on unambiguous strings

## Assumptions

? single-quote style is not used; double-quote is always preferred
? number formatting is round-trippable
