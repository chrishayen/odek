# std.yaml.serialize.serialize_block_sequence

Renders a list as a block-style YAML sequence at a given indentation depth.

## Signature

(values: list[yaml_value], depth: i32) -> string

## Behavior

For each element: emits `depth * 2` spaces, `- `, and either the scalar-serialized element or (for nested mappings/sequences) a newline followed by the nested serialization at `depth + 1`. Each element ends with a newline.

## Tests

+ emits "- a\n- b\n- c\n" at depth 0 for a three-string list
+ indents nested sequences by 2 spaces per depth level
+ preserves element order
- returns no flow-style bracket representation

## Assumptions

? indentation is always 2 spaces per level
? a `-` entry with a nested mapping uses indented children, not inline
