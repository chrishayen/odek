# std.yaml.serialize.serialize_block_mapping

Renders a map as a block-style YAML mapping at a given indentation depth.

## Signature

(entries: map[string, yaml_value], depth: i32) -> string

## Behavior

For each (key, value) pair: emits `depth * 2` spaces, the scalar-serialized key, `: `, and either the scalar-serialized value (when the value is a scalar) or a newline followed by the nested serialization at `depth + 1` (when the value is a mapping or sequence). Each entry ends with a newline.

## Tests

+ emits "a: 1\nb: 2\n" at depth 0 for a two-entry numeric map
+ indents nested mappings by 2 spaces per depth level
+ preserves map insertion order
- returns no representation with flow-style braces
- does not emit a document marker "---"

## Assumptions

? indentation is always 2 spaces per level
? no empty-line separators between keys
