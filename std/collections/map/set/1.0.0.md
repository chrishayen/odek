# std.collections.map.set

Returns a new map with the given key set to the given value, replacing any prior value.

## Signature

(m: map[K, V], k: K, v: V) -> map[K, V]

## Behavior

Produces a new map containing every entry from `m` except that `k` is bound to `v`. When `k` was absent the new map has one more entry; when `k` was present the new map has the same size. The original map is not mutated.

## Tests

+ adds a new entry when k is absent
+ replaces the value when k is already present
+ does not mutate the input map
+ preserves all other entries verbatim
- does not merge values; set fully replaces

## Assumptions

? functions as a persistent/value-semantic operation; mutable variants are separate
