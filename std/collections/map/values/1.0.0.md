# std.collections.map.values

Returns a list of every value in the map.

## Signature

(m: map[K, V]) -> list[V]

## Behavior

Returns a freshly allocated list containing each value of `m`. Order follows the map's iteration order (insertion order for insertion-ordered maps).

## Tests

+ returns a list of length equal to map size
+ returns every value exactly once
+ returns an empty list for an empty map
- does not return duplicate values when values happen to repeat under different keys (both appear)

## Assumptions

? order follows map iteration order
