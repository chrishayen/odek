# std.collections.map.filter

Returns a new map containing only entries for which `pred(key, value)` is true.

## Signature

(m: map[K, V], pred: func(K, V) -> bool) -> map[K, V]

## Behavior

Calls `pred` on each entry. Retains entries where the result is true.

## Tests

+ keeps only entries where pred returns true
+ returns an empty map when pred is always false
+ returns a copy when pred is always true
- does not mutate the input

## Assumptions

? pred is called once per entry
