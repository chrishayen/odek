# std.collections.map.entries

Returns a list of (key, value) pairs for every entry in the map.

## Signature

(m: map[K, V]) -> list[pair[K, V]]

## Behavior

Returns a freshly allocated list of pairs. Order follows map iteration order.

## Tests

+ returns a list of pairs, one per entry
+ returns an empty list for an empty map
+ pairs are (key, value) in that order
- does not duplicate any entry

## Assumptions

? iteration order is insertion order
