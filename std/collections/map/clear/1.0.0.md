# std.collections.map.clear

Returns a new empty map with the same key/value types as the input.

## Signature

(m: map[K, V]) -> map[K, V]

## Behavior

Returns an empty map. The input is not mutated.

## Tests

+ returns an empty map
+ returns a map of size 0
+ does not mutate the input
- never returns a map with any entries

## Assumptions

? value-semantics clear; original retains its entries
