# std.collections.map.is_empty

Returns true when the map has no entries.

## Signature

(m: map[K, V]) -> bool

## Behavior

Returns true iff `size(m) == 0`. O(1).

## Tests

+ returns true for an empty map
+ returns false when any entry exists
- returns true for a non-empty map under no circumstance

## Assumptions

? O(1) via size check
