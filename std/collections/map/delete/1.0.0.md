# std.collections.map.delete

Returns a new map with the given key removed, if present.

## Signature

(m: map[K, V], k: K) -> map[K, V]

## Behavior

Produces a new map containing every entry from `m` except the one keyed by `k`. When `k` is absent, returns a map with the same contents. The original map is not mutated.

## Tests

+ removes the entry for k when present
+ returns an equivalent map when k is absent
+ returns an empty map when deleting the only entry
+ does not mutate the input map
- never returns a map larger than the input

## Assumptions

? functions as a persistent/value-semantic operation
? returns the input reference when k is absent (implementations may reuse to save allocation)
