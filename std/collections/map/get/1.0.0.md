# std.collections.map.get

Returns the value associated with a key, or none when the key is absent.

## Signature

(m: map[K, V], k: K) -> optional[V]

## Behavior

Looks up `k` in `m` and returns `some(value)` on hit, `none` on miss. Key equality is determined by the key type's hash and equality operators. Runs in expected O(1) time for hash-based maps.

## Tests

+ returns some(v) when k is present
+ returns none when k is absent
+ returns none for an empty map
+ treats equal keys (by the type's equality) as the same entry
- returns none (not error) for a missing key

## Assumptions

? backed by a hash map; key type must be hashable
? expected O(1) lookup; worst-case O(n) under adversarial hash collisions
