# std.collections.map.has

Returns true when the map contains an entry for the given key.

## Signature

(m: map[K, V], k: K) -> bool

## Behavior

Looks up `k` in `m` and returns true iff an entry exists. Distinct from checking whether `get` returns `some(null)` — this function reports presence of the entry regardless of the value.

## Tests

+ returns true when k is present
+ returns false when k is absent
+ returns false for an empty map
+ returns true even when the associated value is itself a null/none sentinel
- returns true for a missing key under no circumstance

## Assumptions

? backed by a hash map; expected O(1)
