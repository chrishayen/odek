# std.collections.map.map_values

Returns a new map with the same keys but each value replaced by `f(value)`.

## Signature

(m: map[K, V], f: func(V) -> U) -> map[K, U]

## Behavior

Preserves keys unchanged. Applies `f` to each value.

## Tests

+ returns a map with the same keys and transformed values
+ returns an empty map for an empty input
+ calls f once per entry
- does not mutate the input

## Assumptions

? keys stay unchanged; only values are transformed
