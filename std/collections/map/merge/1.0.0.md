# std.collections.map.merge

Returns a new map containing every entry from `a` and `b`, with `b` winning on key collisions.

## Signature

(a: map[K, V], b: map[K, V]) -> map[K, V]

## Behavior

Copies entries from `a`, then overwrites with every entry from `b`. Duplicates resolved last-wins (b wins).

## Tests

+ returns a copy of a when b is empty
+ returns a copy of b when a is empty
+ merges both when neither is empty
+ b's value wins on key collision
- does not mutate either input

## Assumptions

? b-wins semantics matches most stdlibs
