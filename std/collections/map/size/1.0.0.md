# std.collections.map.size

Returns the number of entries in the map.

## Signature

(m: map[K, V]) -> i64

## Behavior

Returns the count of (key, value) entries. O(1).

## Tests

+ returns 0 for an empty map
+ returns the number of unique keys
+ returns the same value on successive calls when the map is unchanged
- never returns a negative value

## Assumptions

? size is tracked by the underlying map, queried in O(1)
