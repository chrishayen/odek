# std.collections.map.keys

Returns a list of every key in the map.

## Signature

(m: map[K, V]) -> list[K]

## Behavior

Returns a freshly allocated list containing each key of `m` exactly once. Order is unspecified and may differ between calls even on the same map. Runs in O(n) time.

## Tests

+ returns a list of length equal to map size
+ returns every key in the map exactly once
+ returns an empty list for an empty map
+ returns a fresh list that can be mutated without affecting the map
- does not return any key twice

## Assumptions

? order of keys is unspecified; sort the result if you need deterministic ordering
