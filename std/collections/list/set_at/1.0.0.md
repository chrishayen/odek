# std.collections.list.set_at

Returns a new list with index `i` replaced by `x`.

## Signature

(xs: list[T], i: i64, x: T) -> result[list[T], string]

## Behavior

Validates `0 <= i < length(xs)`. Returns a copy with the element at index `i` replaced by `x`. The original is not mutated.

## Tests

+ replaces the element at index i
+ does not mutate the input
+ returns an equivalent list with same length
- returns error when i is negative
- returns error when i is at or beyond length
