# std.collections.list.map

Returns a new list containing `f(x)` for every element `x` of the input.

## Signature

(xs: list[T], f: func(T) -> U) -> list[U]

## Behavior

Applies `f` eagerly to each element in order. Produces a new list with the same length as the input. Does not mutate the input.

## Tests

+ returns [f(x1), f(x2), f(x3)] for [x1, x2, x3]
+ preserves length
+ returns an empty list for an empty input
- does not mutate the input

## Assumptions

? eager; for lazy semantics use iter.map
? f is called exactly once per element, in order
