# std.collections.list.flatten

Concatenates a list of lists into a single flat list, preserving order.

## Signature

(xss: list[list[T]]) -> list[T]

## Behavior

Returns a new list containing every inner element, from inner list 0 through inner list n-1, in order.

## Tests

+ returns [] for []
+ returns [] for [[], []]
+ returns [1, 2, 3, 4] for [[1, 2], [3, 4]]
+ returns [1, 2] for [[], [1, 2], []]
- does not mutate the input

## Assumptions

? flattens exactly one level; nested-nested lists require repeated calls
