# std.collections.list.remove_at

Returns a new list with the element at index `i` removed, shifting later elements left.

## Signature

(xs: list[T], i: i64) -> result[list[T], string]

## Behavior

Requires `0 <= i < length(xs)`. Returns a new list without the element at `i`. Later elements shift left by one.

## Tests

+ removes the first element when i = 0
+ removes the last element when i = length - 1
+ removes a middle element, shifting the rest left
+ does not mutate the input
- returns error when i is negative or out of range
