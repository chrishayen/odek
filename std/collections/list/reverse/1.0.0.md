# std.collections.list.reverse

Returns a new list with the elements of the input in reverse order.

## Signature

(xs: list[T]) -> list[T]

## Behavior

Produces a new list whose `i`-th element is the `(len(xs) - 1 - i)`-th element of `xs`. The original list is not mutated. Runs in O(n) time.

## Tests

+ returns [3, 2, 1] for input [1, 2, 3]
+ returns [] for an empty list
+ returns [x] for a single-element list
+ preserves element identity (no copy of elements, just a new list)
+ does not mutate the input list
- reverses the list without duplicating or losing any element

## Assumptions

? new list allocation is O(n); element copies are shallow
