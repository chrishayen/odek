# std.collections.list.any

Returns true if any element satisfies `pred`, false otherwise.

## Signature

(xs: list[T], pred: func(T) -> bool) -> bool

## Behavior

Short-circuits on the first true. Returns false for an empty list.

## Tests

+ returns true when at least one element satisfies pred
+ returns false when no element satisfies pred
+ returns false for an empty list
+ short-circuits after the first true
- returns true for an empty list under no circumstance

## Assumptions

? short-circuit is observable; pred may have side effects
