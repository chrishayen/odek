# std.collections.list.contains

Returns true when the list contains the given element.

## Signature

(xs: list[T], x: T) -> bool

## Behavior

Scans the list left-to-right and returns true on the first element equal to `x` by the type's equality. Returns false when no element matches. Runs in O(n) time.

## Tests

+ returns true when x is present in xs
+ returns false when x is absent from xs
+ returns false for an empty list regardless of x
+ returns true when x is the first element
+ returns true when x is the last element
- returns false when xs has elements deeply equal to x but not equal under the type's equality

## Assumptions

? equality is the type T's natural equality (structural for value types, pointer for reference types unless overridden)
