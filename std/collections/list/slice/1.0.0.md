# std.collections.list.slice

Returns a new list containing elements from `start` (inclusive) to `end` (exclusive).

## Signature

(xs: list[T], start: i64, end: i64) -> result[list[T], string]

## Behavior

Requires `0 <= start <= end <= length(xs)`. Returns a new list with the slice. Empty slice when `start == end`.

## Tests

+ returns the full list for (xs, 0, length)
+ returns an empty list for (xs, k, k)
+ returns the middle portion for a valid range
- returns error when start is negative
- returns error when end exceeds length
- returns error when start > end

## Assumptions

? half-open interval (standard slice convention)
