# std.collections.list.unique

Returns a new list with duplicate elements removed, preserving the first occurrence of each.

## Signature

(xs: list[T]) -> list[T]

## Behavior

Walks left-to-right, keeping the first occurrence of each distinct element under the type's equality. Later duplicates are dropped.

## Tests

+ returns an empty list for an empty input
+ returns the input when already unique
+ preserves the first occurrence of each element
+ drops subsequent duplicates
- does not reorder elements other than by dropping

## Assumptions

? equality is the type's natural equality
? uses an auxiliary set; T must be hashable
