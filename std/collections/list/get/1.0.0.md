# std.collections.list.get

Returns the element at index `i`, or none when out of range.

## Signature

(xs: list[T], i: i64) -> optional[T]

## Behavior

Returns `some(xs[i])` when `0 <= i < length(xs)`. Returns `none` for negative `i` or `i` at or beyond length.

## Tests

+ returns some(xs[0]) for index 0
+ returns some(xs[length-1]) for the last index
+ returns none for an empty list
+ returns none for a negative index
+ returns none for an index at or beyond length
- returns no value from out of bounds as some

## Assumptions

? never panics; always returns an optional
