# std.collections.list.find_index

Returns the index of the first element for which `pred` returns true, or none.

## Signature

(xs: list[T], pred: func(T) -> bool) -> optional[i64]

## Behavior

Scans left-to-right. Returns `some(i)` on the first matching index.

## Tests

+ returns some(0) when xs[0] matches
+ returns some(i) for the first matching index
+ returns none for no match
+ returns none for an empty list
- never returns a negative index

## Assumptions

? short-circuits after the first match
