# std.collections.list.sort

Returns a new list with the elements of the input sorted by a total-order comparator.

## Signature

(xs: list[T], less: func(T, T) -> bool) -> list[T]

## Behavior

Produces a new list whose elements are the elements of `xs` ordered by `less`. Sorting is stable: elements for which `less(a, b) == false && less(b, a) == false` retain their original relative order. Runs in O(n log n) time and O(n) extra space.

## Tests

+ returns a list sorted ascending when less is the usual less-than
+ returns the input unchanged when the input is already sorted
+ preserves relative order of elements with equal sort keys (stable)
+ returns [] for an empty input
+ handles lists with duplicate elements
- does not mutate the input list

## Assumptions

? the comparator defines a strict weak ordering; otherwise behavior is undefined
? the sort is stable
