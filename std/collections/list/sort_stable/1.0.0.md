# std.collections.list.sort_stable

Returns a new list sorted by `less`, preserving the relative order of equal elements. Facade over `std.sort.sort_slice_stable`.

## Signature

(xs: list[T], less: func(T, T) -> bool) -> list[T]

## Behavior

Delegates to `std.sort.sort_slice_stable`.

## Tests

+ returns a sorted list
+ preserves relative order of equivalent elements (stable)
+ returns an empty list for an empty input
- does not mutate the input
