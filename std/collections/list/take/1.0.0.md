# std.collections.list.take

Returns a new list with the first at most `n` elements.

## Signature

(xs: list[T], n: i64) -> list[T]

## Behavior

When `n <= 0`, returns empty. When `n >= length(xs)`, returns a copy of `xs`. Otherwise returns the first `n` elements.

## Tests

+ returns the first n elements for n within bounds
+ returns the whole input when n >= length
+ returns an empty list for n = 0
+ returns an empty list for negative n
- returns more than n elements under no circumstance

## Assumptions

? negative n clamps to 0
