# std.collections.list.all

Returns true if every element satisfies `pred`, true for an empty list (vacuous truth).

## Signature

(xs: list[T], pred: func(T) -> bool) -> bool

## Behavior

Short-circuits on the first false. Returns true for an empty list.

## Tests

+ returns true when every element satisfies pred
+ returns false on the first element that fails
+ returns true for an empty list (vacuous truth)
+ short-circuits after the first false
- returns false for an empty list under no circumstance
