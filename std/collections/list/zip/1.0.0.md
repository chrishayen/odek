# std.collections.list.zip

Returns a new list of pairs of corresponding elements, truncated to the shorter input.

## Signature

(xs: list[A], ys: list[B]) -> list[pair[A, B]]

## Behavior

For each `i` in `[0, min(length(xs), length(ys)))`, emits `(xs[i], ys[i])`. Stops at the shorter input's end.

## Tests

+ returns [(1,"a"), (2,"b")] for ([1, 2], ["a", "b"])
+ stops at the shorter input
+ returns an empty list when either input is empty
- does not mutate either input

## Assumptions

? fair zip: longer input's remaining elements are discarded
