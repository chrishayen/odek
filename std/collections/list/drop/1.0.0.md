# std.collections.list.drop

Returns a new list with the first `n` elements removed.

## Signature

(xs: list[T], n: i64) -> list[T]

## Behavior

When `n <= 0`, returns a copy of `xs`. When `n >= length(xs)`, returns empty. Otherwise returns elements from index `n` onward.

## Tests

+ returns xs[n..] for valid n
+ returns the whole input for n = 0 or negative
+ returns an empty list for n >= length
- does not mutate the input

## Assumptions

? negative n clamps to 0
