# std.collections.list.sort_by

Returns a new list sorted by keys extracted from each element. Facade over `std.sort.sort_by_key`.

## Signature

(xs: list[T], key: func(T) -> K, less: func(K, K) -> bool) -> list[T]

## Behavior

Delegates to `std.sort.sort_by_key` unchanged. Exposed under collections.list for discoverability.

## Tests

+ sorts by the extracted key
+ returns an empty list for an empty input
+ calls key exactly once per element
- does not mutate the input
