# std.collections.list.fold

Threads an accumulator through a function, consuming the list left-to-right, and returns the final accumulator.

## Signature

(xs: list[T], initial: U, f: func(U, T) -> U) -> U

## Behavior

Starts with `acc = initial`. For each `x` in `xs` (left-to-right), updates `acc = f(acc, x)`. Returns `acc` at end.

## Tests

+ returns initial for an empty list
+ returns f(initial, x0) for a single-element list
+ sums [1, 2, 3] with initial=0 and f=add to 6
- never applies f right-to-left

## Assumptions

? left-to-right traversal
