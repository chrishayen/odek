# std.collections.list.push

Returns a new list with the given element appended at the end.

## Signature

(xs: list[T], x: T) -> list[T]

## Behavior

Produces a new list whose contents are the elements of `xs` in order followed by `x`. The original list is not mutated. Runs in amortized O(1) time when the implementation supports growth with geometric resizing.

## Tests

+ returns [1, 2, 3, 4] for input ([1, 2, 3], 4)
+ returns [5] when pushing onto an empty list
+ does not mutate the input list
+ preserves the type parameter T across the operation
- rejects no valid value of type T; accepts any T

## Assumptions

? functions as a persistent/value-semantic operation; mutable variants are separate
