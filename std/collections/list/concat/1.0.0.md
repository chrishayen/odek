# std.collections.list.concat

Returns a new list containing every element of `xs` followed by every element of `ys`.

## Signature

(xs: list[T], ys: list[T]) -> list[T]

## Behavior

Preserves order. Returns a fresh list of length `length(xs) + length(ys)`. Neither input is mutated.

## Tests

+ returns [1, 2, 3, 4] for ([1, 2], [3, 4])
+ returns ys when xs is empty
+ returns xs when ys is empty
+ returns an empty list when both are empty
- does not mutate either input

## Assumptions

? always a fresh allocation
