# std.collections.list.length

Returns the number of elements in a list.

## Signature

(xs: list[T]) -> i64

## Behavior

Returns the count of elements in `xs`. Runs in O(1) time for list implementations that track length. Never negative.

## Tests

+ returns 0 for an empty list
+ returns 3 for a list of three elements
+ returns the same value on repeated calls without mutation
+ returns a value equal to the number of elements pushed since creation
- returns a negative value under no circumstance

## Assumptions

? the underlying list implementation tracks length in O(1)
