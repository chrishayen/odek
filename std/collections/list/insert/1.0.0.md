# std.collections.list.insert

Returns a new list with `x` inserted at index `i`, shifting later elements right.

## Signature

(xs: list[T], i: i64, x: T) -> result[list[T], string]

## Behavior

Requires `0 <= i <= length(xs)`. Returns a new list with `x` at position `i`; existing elements at `i` and later shift one position to the right. `i == length(xs)` appends.

## Tests

+ inserts at position 0 (prepend)
+ inserts at length (append)
+ inserts in the middle shifting later elements
+ does not mutate the input
- returns error when i is negative
- returns error when i is greater than length
