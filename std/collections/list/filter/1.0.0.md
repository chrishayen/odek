# std.collections.list.filter

Returns a new list containing only elements for which `pred` returns true.

## Signature

(xs: list[T], pred: func(T) -> bool) -> list[T]

## Behavior

Preserves input order. Length of result is 0 <= n <= input length.

## Tests

+ returns only elements for which pred is true
+ returns an empty list when pred is always false
+ returns the whole input when pred is always true
+ preserves element order
- does not mutate the input

## Assumptions

? eager; for lazy use iter.filter
