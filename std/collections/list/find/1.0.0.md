# std.collections.list.find

Returns the first element for which `pred` returns true, or none.

## Signature

(xs: list[T], pred: func(T) -> bool) -> optional[T]

## Behavior

Scans left-to-right. Returns `some(x)` on the first matching element. Short-circuits: does not call `pred` after a match.

## Tests

+ returns some(x) for the first match
+ returns none when no element matches
+ returns none for an empty list
+ short-circuits after the first match
- does not call pred on elements after the match

## Assumptions

? short-circuiting is observable; pred may have side effects
