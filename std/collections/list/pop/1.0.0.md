# std.collections.list.pop

Returns a new list with the final element removed.

## Signature

(xs: list[T]) -> result[list[T], string]

## Behavior

Produces a new list containing the first `len(xs) - 1` elements of `xs` in order. The original list is not mutated. Runs in O(1) time when the implementation supports capacity-preserving truncation.

## Tests

+ returns [1, 2] for input [1, 2, 3]
+ returns [] for input [1]
+ does not mutate the input list
- returns error for an empty input list

## Assumptions

? does not return the popped element; a separate function handles that
? functions as a persistent/value-semantic operation
