# std.collections.set.is_empty

Returns true when the set has no elements.

## Signature

(s: set[T]) -> bool

## Behavior

Returns true iff `size(s) == 0`. O(1).

## Tests

+ returns true for an empty set
+ returns false when any element is present
- returns true for a non-empty set under no circumstance

## Assumptions

? O(1) via size
