# std.collections.set.contains

Returns true when the set contains the given element.

## Signature

(s: set[T], x: T) -> bool

## Behavior

Tests membership of `x` in `s` using the type's natural equality. Runs in expected O(1) time for hash-based sets.

## Tests

+ returns true when x is in the set
+ returns false when x is not in the set
+ returns false for an empty set
+ treats equal elements (by the type's equality) as the same member
- returns true for a non-member under no circumstance

## Assumptions

? expected O(1); worst-case O(n) under adversarial hash collisions
