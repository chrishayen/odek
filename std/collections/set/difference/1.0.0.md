# std.collections.set.difference

Returns a new set containing elements of `a` that are not in `b`.

## Signature

(a: set[T], b: set[T]) -> set[T]

## Behavior

Copies `a`, then removes every element that is also in `b`.

## Tests

+ returns elements in a that are not in b
+ returns an empty set when a is a subset of b
+ returns a copy of a when b is empty
- does not mutate either input

## Assumptions

? equality is the type's natural equality
