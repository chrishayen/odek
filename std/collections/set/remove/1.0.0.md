# std.collections.set.remove

Returns a new set with `x` removed, or an equivalent set when `x` was absent.

## Signature

(s: set[T], x: T) -> set[T]

## Behavior

Returns a copy of `s` without `x`. When `x` wasn't a member, returns an equivalent set.

## Tests

+ removes x when present
+ returns an equivalent set when x is absent
+ returns an empty set when removing the only element
+ does not mutate the input
