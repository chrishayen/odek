# std.collections.set.union

Returns a new set containing every element from either input set.

## Signature

(a: set[T], b: set[T]) -> set[T]

## Behavior

Produces a new set whose members are the union of `a` and `b`. Neither input is mutated. Runs in O(|a| + |b|) time.

## Tests

+ returns a set containing every element of both inputs
+ returns a copy of a when b is empty
+ returns a copy of b when a is empty
+ returns an empty set when both inputs are empty
+ does not duplicate elements shared by both inputs
- does not mutate either input

## Assumptions

? equality is the type T's natural equality
? no element from either input is dropped
