# std.collections.set.symmetric_difference

Returns a new set containing elements in exactly one of `a` or `b`.

## Signature

(a: set[T], b: set[T]) -> set[T]

## Behavior

Union minus intersection: returns `(a - b) ∪ (b - a)`.

## Tests

+ returns elements in a not in b, plus elements in b not in a
+ returns an empty set when a equals b
+ returns union when a and b are disjoint
- does not mutate either input

## Assumptions

? equality is the type's natural equality
