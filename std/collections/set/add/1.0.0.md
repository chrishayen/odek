# std.collections.set.add

Returns a new set with the given element inserted; a no-op when the element is already present.

## Signature

(s: set[T], x: T) -> set[T]

## Behavior

Produces a new set containing every element of `s` plus `x`. When `x` is already present, the returned set is equivalent to `s`. The original set is not mutated.

## Tests

+ adds a new element when x is absent
+ returns an equivalent set when x is already present
+ does not mutate the input set
+ preserves all other elements verbatim
- never returns a set larger than size(s) + 1

## Assumptions

? equality is the type T's natural equality
? set is backed by a hash structure; T must be hashable
