# std.collections.set.intersection

Returns a new set containing only elements present in both `a` and `b`.

## Signature

(a: set[T], b: set[T]) -> set[T]

## Behavior

Iterates over the smaller input for efficiency, filtering to elements also in the larger. Returns a fresh set.

## Tests

+ returns elements present in both
+ returns an empty set when either input is empty
+ returns a copy when a and b are equal
- does not mutate either input

## Assumptions

? equality is the type's natural equality
