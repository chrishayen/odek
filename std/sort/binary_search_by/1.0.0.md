# std.sort.binary_search_by

Runs binary search over a sorted list using a probe function that returns -1, 0, or 1.

## Signature

(xs: list[T], probe: func(T) -> i32) -> optional[i64]

## Behavior

At each step, calls `probe(xs[mid])`. Narrows left when `probe` returns positive (mid too large), right when negative (mid too small), stops when 0 (match). Returns `some(mid)` on match, `none` on failure.

## Tests

+ returns some(i) where probe returns 0 at index i
+ returns none when probe never returns 0
+ returns none for an empty list
+ calls probe O(log n) times
- never calls probe more than ceil(log2(n+1)) times

## Assumptions

? probe is consistent with a sorted order; otherwise behavior is undefined
