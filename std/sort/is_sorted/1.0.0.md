# std.sort.is_sorted

Returns true when every adjacent pair of elements satisfies the non-strict ordering implied by `less`.

## Signature

(xs: list[T], less: func(T, T) -> bool) -> bool

## Behavior

Returns true iff for every `i`, `!less(xs[i+1], xs[i])` holds. Empty and single-element lists trivially return true.

## Tests

+ returns true for an empty list
+ returns true for a single-element list
+ returns true for a non-decreasing list
+ returns true for a list with equal adjacent elements (non-strict order)
- returns false for a list with a decreasing adjacent pair

## Assumptions

? non-strict order; equal elements are considered "sorted"
