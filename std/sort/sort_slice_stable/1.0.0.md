# std.sort.sort_slice_stable

Returns a new list sorted by a user-supplied less-than comparator, preserving the relative order of equivalent elements.

## Signature

(xs: list[T], less: func(T, T) -> bool) -> list[T]

## Behavior

Like `sort_slice`, but guarantees that elements for which `less(a, b) == false && less(b, a) == false` appear in their original order.

## Tests

+ returns an ascending-sorted list
+ preserves relative order of equivalent elements (stable)
+ returns the input unchanged when already sorted
+ returns an empty list for an empty input
- does not mutate the input list

## Assumptions

? comparator defines a strict weak ordering
? typically O(n log n) time and O(n) space
