# std.sort.binary_search

Returns the index of `target` in a sorted list using binary search, or none if not found.

## Signature

(xs: list[T], target: T, less: func(T, T) -> bool) -> optional[i64]

## Behavior

Runs binary search over a list that is already sorted under `less`. Returns `some(i)` where `xs[i] == target` (neither less than nor greater than). Returns `none` if the target is not present. When multiple matches exist, the index returned is unspecified (one of them).

## Tests

+ returns some(i) for a present target
+ returns none when the target is absent
+ returns none for an empty list
+ runs in O(log n) time
- returns a correct index if the list is not sorted only by coincidence; otherwise behavior is undefined

## Assumptions

? caller guarantees the list is sorted by `less`
? for duplicate elements, the returned index is unspecified
