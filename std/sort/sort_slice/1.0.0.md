# std.sort.sort_slice

Returns a new list sorted by a user-supplied less-than comparator. Order among equal elements is unspecified.

## Signature

(xs: list[T], less: func(T, T) -> bool) -> list[T]

## Behavior

Produces a new list sorted so that `less(xs[i], xs[i+1])` holds (or they are equivalent). Stability is NOT guaranteed — equivalent elements may appear in any order.

## Tests

+ returns an ascending-sorted list for `less = a < b`
+ returns the input unchanged when already sorted
+ handles lists with duplicate elements
+ returns an empty list for an empty input
- does not mutate the input list

## Assumptions

? comparator must define a strict weak ordering
? not stable; use sort_slice_stable for stable sort
