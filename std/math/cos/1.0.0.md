# std.math.cos

Returns the cosine of `x` (x in radians).

## Signature

(x: f64) -> f64

## Behavior

Computes `cos(x)` under IEEE-754 double precision. Returns NaN for NaN or infinite input.

## Tests

+ returns 1.0 for cos(0)
+ returns ~0.0 for cos(π/2)
+ returns -1.0 for cos(π)
- returns NaN for NaN input
- returns NaN for infinite input

## Assumptions

? rounding follows IEEE-754 round-to-nearest-even
