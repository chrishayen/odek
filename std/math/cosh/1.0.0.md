# std.math.cosh

Returns the hyperbolic cosine of `x`.

## Signature

(x: f64) -> f64

## Behavior

Computes `(e^x + e^-x) / 2`. Always >= 1. Returns +Infinity for large |x|.

## Tests

+ returns 1.0 for cosh(0)
+ returns +Infinity for large positive or negative input
+ always returns a value >= 1 for finite input
- returns NaN for NaN input

## Assumptions

? precision bounded within a few ULP
