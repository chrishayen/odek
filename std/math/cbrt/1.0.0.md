# std.math.cbrt

Returns the real cube root of `x`, preserving sign.

## Signature

(x: f64) -> f64

## Behavior

Returns `x^(1/3)` with `cbrt(-x) = -cbrt(x)`. Defined for all finite reals (unlike sqrt for negatives).

## Tests

+ returns 2.0 for cbrt(8)
+ returns -2.0 for cbrt(-8)
+ returns 0.0 for cbrt(0)
+ returns +Infinity for cbrt(+Infinity)
- returns NaN for NaN input

## Assumptions

? defined for negative inputs (returns negative real root)
