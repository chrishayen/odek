# std.math.asin

Returns the arcsine of `x`, in radians, in [-π/2, π/2].

## Signature

(x: f64) -> f64

## Behavior

Returns the angle whose sine is `x`. Valid input range is [-1, 1]. Returns NaN for inputs outside that range or for NaN.

## Tests

+ returns 0.0 for asin(0)
+ returns π/2 for asin(1)
+ returns -π/2 for asin(-1)
- returns NaN for asin(1.1) (out of range)
- returns NaN for NaN input

## Assumptions

? domain is closed [-1, 1]
