# std.math.sqrt

Returns the non-negative square root of a 64-bit floating-point number.

## Signature

(x: f64) -> f64

## Behavior

Returns the principal (non-negative) square root of `x`. For negative `x`, the result is NaN. The square root of `-0.0` is `-0.0`; the square root of `+Infinity` is `+Infinity`.

## Tests

+ returns 2.0 for input 4.0
+ returns 0.0 for input 0.0
+ returns -0.0 for input -0.0
+ returns +Infinity for input +Infinity
+ returns an approximation correct to the last ULP for finite positive inputs
- returns NaN for any negative finite input
- returns NaN for input NaN

## Assumptions

? arithmetic is IEEE-754 double precision
? rounding follows round-to-nearest-even
