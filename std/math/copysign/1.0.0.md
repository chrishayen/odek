# std.math.copysign

Returns the value with the magnitude of `magnitude` and the sign of `sign`.

## Signature

(magnitude: f64, sign: f64) -> f64

## Behavior

Returns `|magnitude|` if `sign >= 0`, else `-|magnitude|`. Preserves NaN magnitude; the sign bit of the result equals the sign bit of `sign`, except NaN propagates through magnitude.

## Tests

+ returns 3.0 for (3.0, 1.0)
+ returns -3.0 for (3.0, -1.0)
+ returns -3.0 for (-3.0, -1.0)
+ returns +0.0 for (0.0, 1.0)
+ returns -0.0 for (0.0, -1.0)
- returns NaN for a NaN magnitude

## Assumptions

? IEEE-754 copysign semantics
