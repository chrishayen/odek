# std.math.exp

Returns `e` raised to the power of `x`.

## Signature

(x: f64) -> f64

## Behavior

Computes `e^x`. Returns `+Infinity` for very large positive inputs, `+0` for very large negative inputs. Returns NaN for NaN.

## Tests

+ returns 1.0 for exp(0)
+ returns ~e for exp(1)
+ returns +Infinity for very large positive input
+ returns +0 for very large negative input
- returns NaN for NaN input

## Assumptions

? precision bounded within a few ULP
