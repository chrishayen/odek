# std.math.min

Returns the smaller of two 64-bit floating-point numbers.

## Signature

(a: f64, b: f64) -> f64

## Behavior

Returns `a` when `a < b`, otherwise `b`. When either argument is NaN, the result is NaN. For `min(-0.0, +0.0)` the negative zero is returned.

## Tests

+ returns 1.0 for input (1.0, 2.0)
+ returns 1.0 for input (2.0, 1.0)
+ returns a when a equals b (either is correct; returns the first)
+ returns NaN when either argument is NaN
+ returns -0.0 for input (-0.0, 0.0)
+ returns -Infinity for input (-Infinity, any finite value)
- returns a value greater than both inputs under no circumstance

## Assumptions

? NaN is contagious under min; propagates to the output
