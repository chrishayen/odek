# std.math.fmod

Returns the remainder of `x / y` with the sign of `x` (IEEE-754 fmod).

## Signature

(x: f64, y: f64) -> f64

## Behavior

Returns `x - n * y` for the integer `n = trunc(x / y)`. The result has the sign of `x`. Returns NaN when `y` is 0 or either input is NaN. Returns NaN when `x` is infinite.

## Tests

+ returns 1.0 for fmod(7, 3)
+ returns -1.0 for fmod(-7, 3) (sign of x)
+ returns 0.0 for fmod(6, 3)
- returns NaN when y is 0
- returns NaN for infinite x

## Assumptions

? differs from remainder, which uses round-half-to-even quotient
