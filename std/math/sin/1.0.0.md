# std.math.sin

Returns the sine of `x` (x in radians).

## Signature

(x: f64) -> f64

## Behavior

Computes `sin(x)` under IEEE-754 double precision. Returns NaN when `x` is NaN or infinite. Returns `x` when `x` is `±0`.

## Tests

+ returns 0.0 for sin(0)
+ returns ~1.0 for sin(π/2)
+ returns ~0.0 for sin(π)
+ returns -0.0 for sin(-0)
- returns NaN for NaN input
- returns NaN for infinite input

## Assumptions

? argument reduction follows IEEE-754 recommendations; error bounded within a few ULP for finite inputs
