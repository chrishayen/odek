# std.math.trunc

Returns `x` truncated toward zero (fractional part removed, sign preserved).

## Signature

(x: f64) -> f64

## Behavior

Drops the fractional part of `x`. For positive values, equivalent to floor. For negative values, equivalent to ceil. Returns `x` unchanged for integral values, ±Infinity, and NaN.

## Tests

+ returns 2.0 for 2.7
+ returns -2.0 for -2.7
+ returns 2.0 for 2.0
+ returns +Infinity for +Infinity
- returns NaN for NaN input

## Assumptions

? differs from floor (toward -Inf) and ceil (toward +Inf)
