# std.math.tanh

Returns the hyperbolic tangent of `x`, in (-1, 1).

## Signature

(x: f64) -> f64

## Behavior

Computes `sinh(x) / cosh(x)`. Saturates to ±1 for large |x|.

## Tests

+ returns 0.0 for tanh(0)
+ returns 1.0 for tanh(+Infinity)
+ returns -1.0 for tanh(-Infinity)
+ returns a value in (-1, 1) for any finite nonzero input
- returns NaN for NaN input

## Assumptions

? saturates to exactly ±1 for infinite inputs
