# std.math.abs

Returns the absolute value of a 64-bit floating-point number.

## Signature

(x: f64) -> f64

## Behavior

Returns `x` unchanged when `x` is zero or positive; returns `-x` when `x` is negative. Follows IEEE-754 rules for special values: `abs(-0.0)` is `+0.0`, `abs(NaN)` is `NaN`, and `abs(-Infinity)` is `+Infinity`.

## Tests

+ returns 5.0 for input 5.0
+ returns 5.0 for input -5.0
+ returns 0.0 for input 0.0
+ returns +0.0 for input -0.0 (sign bit is cleared)
+ returns +Infinity for input -Infinity
+ returns NaN for input NaN
- returns 0.0 only for the two zeros, not for any non-zero input

## Assumptions

? arithmetic is IEEE-754 double precision
