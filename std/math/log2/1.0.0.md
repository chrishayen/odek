# std.math.log2

Returns the base-2 logarithm of `x`.

## Signature

(x: f64) -> f64

## Behavior

Computes `log₂(x)`. Returns `-Infinity` for 0, NaN for negative inputs.

## Tests

+ returns 0.0 for log2(1)
+ returns 1.0 for log2(2)
+ returns 3.0 for log2(8)
+ returns -Infinity for log2(0)
- returns NaN for negative input
- returns NaN for NaN input

## Assumptions

? dedicated log2 is more accurate than log(x) / log(2)
