# std.math.log10

Returns the base-10 logarithm of `x`.

## Signature

(x: f64) -> f64

## Behavior

Computes `log₁₀(x)`. Returns `-Infinity` for 0, NaN for negative inputs.

## Tests

+ returns 0.0 for log10(1)
+ returns 1.0 for log10(10)
+ returns 2.0 for log10(100)
+ returns -Infinity for log10(0)
- returns NaN for negative input

## Assumptions

? dedicated log10 is more accurate than log(x) / log(10)
