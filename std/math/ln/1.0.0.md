# std.math.ln

Returns the natural logarithm of `x` (base e).

## Signature

(x: f64) -> f64

## Behavior

Computes `ln(x)`. Returns `-Infinity` for 0, NaN for negative inputs.

## Tests

+ returns 0.0 for ln(1)
+ returns 1.0 for ln(e)
+ returns -Infinity for ln(0)
- returns NaN for negative input
- returns NaN for NaN input

## Assumptions

? precision bounded within a few ULP
