# std.math.clamp

Returns `x` confined to the inclusive range [low, high].

## Signature

(x: f64, low: f64, high: f64) -> f64

## Behavior

Returns `low` when `x < low`, `high` when `x > high`, else `x`. Returns NaN if any input is NaN.

## Tests

+ returns x when within [low, high]
+ returns low when x < low
+ returns high when x > high
+ returns low when x equals low; high when x equals high
- returns NaN when any input is NaN

## Assumptions

? requires low <= high; behavior is unspecified otherwise
