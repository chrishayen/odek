# std.math.round

Returns the nearest integer value to the input, rounding ties to even.

## Signature

(x: f64) -> f64

## Behavior

Rounds `x` to the nearest integer. Halfway cases (e.g. 0.5, 1.5, -2.5) round to the nearest even integer (banker's rounding). Returns the input unchanged for infinities and NaN.

## Tests

+ returns 3.0 for input 2.6
+ returns 2.0 for input 2.4
+ returns 2.0 for input 2.5 (ties to even)
+ returns 4.0 for input 3.5 (ties to even)
+ returns -2.0 for input -2.5 (ties to even)
+ returns +Infinity for input +Infinity
- returns NaN for input NaN

## Assumptions

? tie-breaking uses round-half-to-even; not round-half-up
