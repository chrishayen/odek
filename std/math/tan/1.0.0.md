# std.math.tan

Returns the tangent of `x` (x in radians).

## Signature

(x: f64) -> f64

## Behavior

Computes `tan(x)` = `sin(x) / cos(x)`. Returns very large values near π/2 + kπ (where cos approaches 0). Returns NaN for NaN or infinite input.

## Tests

+ returns 0.0 for tan(0)
+ returns ~1.0 for tan(π/4)
+ returns a large magnitude near π/2
- returns NaN for NaN input
- returns NaN for infinite input

## Assumptions

? no wrapping/normalization of input; large inputs lose precision
