# std.math.atan2

Returns the angle in radians from the positive x-axis to the point (x, y), in (-π, π].

## Signature

(y: f64, x: f64) -> f64

## Behavior

Computes the principal value of the argument of the complex number x + iy, accounting for the quadrant. Returns `+π` for negative x, positive zero y; `-π` for negative x, negative zero y; 0 for positive x, zero y.

## Tests

+ returns 0 for (0, 1)
+ returns π/2 for (1, 0)
+ returns π for (0, -1) with y = +0
+ returns -π/2 for (-1, 0)
- returns NaN when either argument is NaN

## Assumptions

? signed-zero conventions match IEEE-754
