# std.math.is_nan

Facade over `std.float.is_nan` for callers working in the math namespace.

## Signature

(x: f64) -> bool

## Behavior

Delegates to `std.float.is_nan` unchanged.

## Tests

+ returns true for NaN
+ returns false for 0.0
+ returns false for any finite or infinite non-NaN

## Assumptions

? facade exposed in math/ for discoverability; canonical impl lives in float/
