# std.math.acos

Returns the arccosine of `x`, in radians, in [0, π].

## Signature

(x: f64) -> f64

## Behavior

Returns the angle whose cosine is `x`. Valid input range is [-1, 1]. Returns NaN for out-of-range or NaN.

## Tests

+ returns 0.0 for acos(1)
+ returns π for acos(-1)
+ returns π/2 for acos(0)
- returns NaN for acos(1.1) (out of range)
- returns NaN for NaN input

## Assumptions

? domain is closed [-1, 1]
