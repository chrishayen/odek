# std.math.sign

Returns -1.0, 0.0, or 1.0 according to the sign of `x`. NaN passes through unchanged.

## Signature

(x: f64) -> f64

## Behavior

Returns `-1.0` for negative finite `x` and `-Infinity`. Returns `1.0` for positive finite `x` and `+Infinity`. Returns `x` unchanged for `0.0`, `-0.0`, and NaN.

## Tests

+ returns 1.0 for positive input
+ returns -1.0 for negative input
+ returns +0.0 for +0.0
+ returns -0.0 for -0.0
+ returns NaN for NaN input
- returns no value outside {-1.0, 0.0, -0.0, 1.0, NaN}

## Assumptions

? signed zero is preserved to avoid losing sign-bit information
