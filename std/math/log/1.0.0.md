# std.math.log

Returns the logarithm of `x` to the given base.

## Signature

(x: f64, base: f64) -> f64

## Behavior

Computes `ln(x) / ln(base)`. Returns NaN when `x < 0`, `base <= 0`, or `base == 1`. Returns `-Infinity` for `log(0, b)` with `b > 1`.

## Tests

+ returns 2.0 for log(100, 10)
+ returns 3.0 for log(8, 2)
+ returns 0.0 for log(1, any base)
- returns NaN for a negative x
- returns NaN for base <= 0
- returns NaN for base == 1
- returns -Infinity for log(0, b>1)

## Assumptions

? computed via natural-log division; precision degrades near base = 1
