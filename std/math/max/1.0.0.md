# std.math.max

Returns the larger of two 64-bit floating-point numbers.

## Signature

(a: f64, b: f64) -> f64

## Behavior

Returns `a` when `a > b`, otherwise `b`. When either argument is NaN, the result is NaN. For `max(-0.0, +0.0)` the positive zero is returned.

## Tests

+ returns 2.0 for input (1.0, 2.0)
+ returns 2.0 for input (2.0, 1.0)
+ returns a when a equals b
+ returns NaN when either argument is NaN
+ returns +0.0 for input (-0.0, 0.0)
+ returns +Infinity for input (+Infinity, any finite value)
- returns a value smaller than both inputs under no circumstance

## Assumptions

? NaN is contagious under max; propagates to the output
