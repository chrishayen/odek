# std.math.hypot

Returns sqrt(x² + y²) without intermediate overflow or underflow.

## Signature

(x: f64, y: f64) -> f64

## Behavior

Computes the Euclidean distance √(x²+y²). Uses a numerically stable algorithm so inputs with wildly different magnitudes don't lose precision.

## Tests

+ returns 5.0 for (3, 4)
+ returns |x| for (x, 0)
+ returns |y| for (0, y)
+ returns +Infinity when either input is infinite
- returns NaN for NaN inputs (unless at least one is infinite)

## Assumptions

? implementation avoids overflow for large finite inputs
