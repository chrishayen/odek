# std.math.ceil

Returns the smallest integer value not less than the input, as an f64.

## Signature

(x: f64) -> f64

## Behavior

Rounds `x` toward positive infinity. Returns `x` unchanged when `x` is already an integer value, positive/negative infinity, or NaN. The sign of zero is preserved.

## Tests

+ returns 3.0 for input 2.3
+ returns -2.0 for input -2.7
+ returns 2.0 for input 2.0 (already integral)
+ returns +Infinity for input +Infinity
+ returns -Infinity for input -Infinity
+ preserves +0.0 for input +0.0
- returns NaN for input NaN

## Assumptions

? rounding direction is toward positive infinity (not toward zero)
