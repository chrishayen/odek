# std.math.pow

Raises `base` to the power of `exponent` under IEEE-754 double precision.

## Signature

(base: f64, exponent: f64) -> f64

## Behavior

Computes `base ** exponent`. Follows IEEE-754 special-value semantics: `pow(x, 0)` is `1` for every `x` including NaN, `pow(1, y)` is `1` for every `y`, and `pow(0, negative)` is `+Infinity`. A finite negative base with a non-integer exponent produces NaN.

## Tests

+ returns 8.0 for input (2.0, 3.0)
+ returns 1.0 for input (x, 0.0) for any x
+ returns 1.0 for input (1.0, y) for any y
+ returns 0.5 for input (2.0, -1.0)
+ returns +Infinity for input (0.0, -1.0)
- returns NaN for input (-2.0, 0.5) — negative base with non-integer exponent
