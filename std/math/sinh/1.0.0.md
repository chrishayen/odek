# std.math.sinh

Returns the hyperbolic sine of `x`.

## Signature

(x: f64) -> f64

## Behavior

Computes `(e^x - e^-x) / 2`. Returns +Infinity for large positive inputs, -Infinity for large negative inputs.

## Tests

+ returns 0.0 for sinh(0)
+ returns +Infinity for large positive input
+ returns -Infinity for large negative input
- returns NaN for NaN input

## Assumptions

? precision bounded within a few ULP
