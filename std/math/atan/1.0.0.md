# std.math.atan

Returns the arctangent of `x`, in radians, in (-π/2, π/2).

## Signature

(x: f64) -> f64

## Behavior

Returns the angle whose tangent is `x`. Accepts all finite values. Returns ±π/2 for ±infinity. Returns NaN for NaN.

## Tests

+ returns 0.0 for atan(0)
+ returns π/4 for atan(1)
+ returns -π/4 for atan(-1)
+ returns π/2 for atan(+Infinity)
+ returns -π/2 for atan(-Infinity)
- returns NaN for NaN input

## Assumptions

? infinities map to ±π/2 exactly
