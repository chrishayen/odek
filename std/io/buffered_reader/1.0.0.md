# std.io.buffered_reader

Wraps a readable stream with an in-memory buffer to reduce underlying read syscalls.

## Signature

(stream: string, buffer_size: i64) -> string

## Behavior

Returns a new stream handle backed by a fixed-size buffer. Reads from the new handle pull from the buffer; when the buffer empties, it refills from the underlying stream.

## Tests

+ returns a handle that reads the same bytes as the underlying stream
+ reduces the number of underlying read syscalls
+ applies the requested buffer size
- returns error when the underlying stream is not readable
- returns error when buffer_size is non-positive

## Assumptions

? typical buffer size is 4 KiB to 64 KiB
