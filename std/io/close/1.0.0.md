# std.io.close

Closes a stream, releasing its underlying handle.

## Signature

(stream: string) -> result[void, string]

## Behavior

Flushes any buffered writes, then closes the underlying OS handle. Subsequent operations on the stream error.

## Tests

+ flushes buffered writes before closing
+ releases the OS handle
- returns error when the stream is already closed
- returns error when flushing fails (e.g. disk full)

## Assumptions

? idempotent close is NOT guaranteed; second call errors
