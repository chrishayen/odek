# std.io.read_line.strip_trailing_cr

Returns a copy of a byte buffer with the terminating LF and any immediately preceding CR removed.

## Signature

(line: bytes) -> bytes

## Behavior

If the buffer ends in LF, removes it. If a CR immediately precedes the removed LF, removes that too. Buffers without a terminating LF pass through unchanged.

## Tests

+ returns the input unchanged when there is no trailing LF
+ removes a trailing LF
+ removes a trailing CRLF pair
+ returns an empty buffer for input {LF} and for {CR, LF}
- does not remove a lone CR that is not immediately before an LF

## Assumptions

? only exactly one terminator pair is trimmed; multi-line buffers are left alone
