# std.io.read_line.validate_utf8

Validates that a byte buffer is well-formed UTF-8 and returns it as a string.

## Signature

(data: bytes) -> result[string, string]

## Behavior

Calls `std.utf8.valid`; returns the buffer viewed as a string when true, or an error when false.

## Tests

+ returns the input as a string when it is valid UTF-8
+ accepts an empty buffer
- returns error when any byte sequence is not valid UTF-8
- returns error on a lone surrogate half

## Assumptions

? delegates the validation rules entirely to std.utf8.valid
