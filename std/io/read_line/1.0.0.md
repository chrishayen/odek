# std.io.read_line

Reads a single line from a named stream, excluding the line terminator.

## Signature

(stream: string) -> result[optional[string], string]

## Behavior

Reads bytes from `stream` ("stdin" is the only supported name unless extended) until an LF byte or end-of-stream. Strips the trailing LF, and any immediately preceding CR, from the returned line. Returns none at end-of-stream. Input bytes are validated as UTF-8 before being returned as a string.

## Tests

+ returns the next line for a stream positioned at the start of a line
+ strips trailing LF from the returned line
+ strips trailing CRLF from the returned line (accepts both line endings)
+ returns none at end-of-stream
+ advances the stream past the line terminator
- returns error when the stream bytes are not valid UTF-8
- returns error when the named stream is not open for reading

## Assumptions

? line length is bounded only by available memory
? only "stdin" is recognized by default; extended implementations may accept file handles
