# std.io.read_line.read_until_lf

Reads bytes from a named stream up to and including the next LF byte, or to end-of-stream.

## Signature

(stream: string) -> result[optional[bytes], string]

## Behavior

Reads one byte at a time until LF is consumed or EOF is reached. Returns `some(bytes)` including the LF (when present), or `none` when EOF is reached before any byte is read.

## Tests

+ returns some(bytes) with the LF included when a line is found
+ returns some(bytes) without an LF when EOF ends the partial line
+ returns none at end-of-stream before any byte is read
- returns error when the named stream is not open for reading

## Assumptions

? the LF is included in the returned buffer; callers strip it via the next rune
? stream names follow the io conventions ("stdin", "stdout", "stderr")
