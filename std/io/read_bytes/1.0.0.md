# std.io.read_bytes

Reads up to `n` bytes from a stream. May return fewer if the stream is exhausted.

## Signature

(stream: string, n: i64) -> result[bytes, string]

## Behavior

Returns the next up-to-`n` bytes. Unlike `read_exact`, accepts a short read — returns fewer bytes if the stream has fewer available. Returns empty at EOF.

## Tests

+ returns up to n bytes when the stream has at least that many
+ returns fewer than n bytes near EOF
+ returns an empty buffer at EOF
- returns error when n is negative

## Assumptions

? permits short reads; caller loops if exact length is required
