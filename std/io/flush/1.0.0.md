# std.io.flush

Forces any buffered bytes to be written to the underlying destination.

## Signature

(stream: string) -> result[void, string]

## Behavior

Flushes the stream's buffered writes. Does not fsync to disk; use a separate fs rune for durability.

## Tests

+ drains buffered bytes to the underlying destination
- returns error when the stream is closed
- does not guarantee durability to stable storage

## Assumptions

? flush is for user-space buffers; durability requires an explicit fsync
