# std.io.read_all

Reads every remaining byte from a named stream into a single buffer.

## Signature

(stream: string) -> result[bytes, string]

## Behavior

Reads from `stream` until end-of-stream is reached and returns the accumulated bytes. May block until the stream is closed by the writer. An already-exhausted stream produces an empty buffer.

## Tests

+ returns every remaining byte from stdin until EOF
+ returns empty bytes when the stream is already at EOF
+ preserves binary content without any encoding transformation
- returns error when the named stream is not open for reading
- returns error when a read operation fails partway through

## Assumptions

? entire contents are buffered in memory; use a streaming API for very large inputs
? blocks the caller until the writer closes the stream
