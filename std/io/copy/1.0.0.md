# std.io.copy

Streams bytes from a readable stream to a writable stream until the source is exhausted.

## Signature

(src: string, dst: string) -> result[i64, string]

## Behavior

Reads from `src` in chunks and writes each chunk to `dst`. Returns the total byte count. Stops when `src` reaches EOF.

## Tests

+ copies every byte from src to dst
+ returns the correct byte count
+ returns 0 when src is empty
- returns error when reading from src fails
- returns error when writing to dst fails

## Assumptions

? stream names refer to already-opened handles; for files use fs.copy_file
