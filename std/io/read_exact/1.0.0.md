# std.io.read_exact

Reads exactly `n` bytes from a stream, retrying on short reads. Returns an error if EOF arrives first.

## Signature

(stream: string, n: i64) -> result[bytes, string]

## Behavior

Repeatedly reads until `n` bytes have been collected. Returns error when the stream ends before `n` bytes arrive.

## Tests

+ returns exactly n bytes
+ retries on short reads
+ returns an empty buffer when n is 0
- returns error when the stream ends before n bytes arrive
- returns error when n is negative

## Assumptions

? blocking read; for non-blocking variants see separate rune
