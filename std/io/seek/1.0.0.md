# std.io.seek

Moves the cursor of a seekable stream and returns the new absolute position.

## Signature

(stream: string, offset: i64, whence: string) -> result[i64, string]

## Behavior

`whence` is `"start"`, `"current"`, or `"end"`. The new position is `offset` relative to that origin. Returns the resulting absolute position in bytes.

## Tests

+ seeks to an absolute position with whence="start"
+ advances by offset with whence="current"
+ positions relative to end with whence="end" (usually negative offset)
+ returns the new absolute position
- returns error when the stream is not seekable
- returns error when whence is not one of the three accepted values

## Assumptions

? seeking past the end is allowed on most OSes (creates sparse regions on write)
