# std.io.write_line

Writes a string to a named stream followed by a single LF byte.

## Signature

(stream: string, text: string) -> result[void, string]

## Behavior

Writes the UTF-8 bytes of `text` to `stream` ("stdout" or "stderr"), then writes a single LF byte. Performs a single best-effort write and does not flush explicitly unless the stream is line-buffered on the host.

## Tests

+ writes the text followed by LF to stdout
+ writes the text followed by LF to stderr
+ accepts an empty text and writes only the LF byte
+ writes a trailing LF even when text already ends with LF
- returns error when stream is neither "stdout" nor "stderr"
- returns error when the underlying file descriptor is closed

## Assumptions

? line terminator is always LF; callers that need CRLF write the bytes directly
? no flush is performed; output may be buffered by the host stdio layer
