# std.io.buffered_writer

Wraps a writable stream with an in-memory buffer; writes accumulate until flushed or the buffer fills.

## Signature

(stream: string, buffer_size: i64) -> string

## Behavior

Returns a new stream handle backed by a fixed-size buffer. Writes fill the buffer; when full, the buffer flushes to the underlying stream. Callers must call `close` or `flush` to avoid data loss.

## Tests

+ accumulates writes without hitting the underlying stream until the buffer fills
+ flushes on buffer-full
+ flushes when explicitly flushed
- returns error when the underlying stream is not writable
- returns error when buffer_size is non-positive

## Assumptions

? typical buffer size is 4 KiB to 64 KiB
? not flushing before drop leaks buffered bytes; callers must close
