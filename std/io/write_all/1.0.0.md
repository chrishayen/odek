# std.io.write_all

Writes every byte of a buffer to a named stream, retrying on short writes.

## Signature

(stream: string, data: bytes) -> result[void, string]

## Behavior

Issues repeated write calls against `stream` until the entire buffer has been written. A partial write advances the cursor and the function continues until all bytes are flushed to the underlying file descriptor. Does not explicitly flush stdio buffers.

## Tests

+ writes every byte of the input buffer to stdout
+ writes every byte of the input buffer to stderr
+ accepts an empty buffer and performs no writes
+ retries on short writes until the buffer is exhausted
- returns error when stream is neither "stdout" nor "stderr"
- returns error when the underlying file descriptor is closed mid-write

## Assumptions

? partial writes (short writes) are transparently retried
? does not fsync or flush stdio buffers
