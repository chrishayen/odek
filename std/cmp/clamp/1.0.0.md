# std.cmp.clamp

Returns `value` confined to the inclusive range [low, high] under a comparator.

## Signature

(value: T, low: T, high: T, cmp: func(T, T) -> i32) -> T

## Behavior

Returns `low` when `cmp(value, low) < 0`, `high` when `cmp(value, high) > 0`, else `value`.

## Tests

+ returns value when it is within [low, high]
+ returns low when value is below low
+ returns high when value is above high
+ returns value when value equals either endpoint
- returns a value outside [low, high] under no circumstance

## Assumptions

? requires low <= high by the comparator; otherwise behavior is undefined
