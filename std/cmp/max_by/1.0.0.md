# std.cmp.max_by

Returns the larger of two values under a user-supplied comparator.

## Signature

(a: T, b: T, cmp: func(T, T) -> i32) -> T

## Behavior

Evaluates `cmp(a, b)`. Returns `a` when the result is >= 0, `b` otherwise.

## Tests

+ returns a when cmp(a, b) > 0
+ returns b when cmp(a, b) < 0
+ returns a when cmp(a, b) == 0 (ties to first argument)
- never returns a value not present in the inputs

## Assumptions

? ties resolve to the first argument
