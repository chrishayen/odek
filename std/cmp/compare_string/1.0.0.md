# std.cmp.compare_string

Compares two strings lexicographically by Unicode code-point order and returns -1, 0, or 1.

## Signature

(a: string, b: string) -> i32

## Behavior

Walks both strings code-point by code-point. Returns -1 when the first differing code point in `a` is less than `b`, 1 when greater. A proper prefix is "less" than the longer string. Equal strings return 0.

## Tests

+ returns 0 for identical strings
+ returns -1 for ("a", "b")
+ returns -1 for ("abc", "abd")
+ returns -1 when a is a proper prefix of b
+ returns 1 when b is a proper prefix of a
- returns a value outside {-1, 0, 1} under no circumstance

## Assumptions

? byte/UTF-8 order matches code-point order for valid UTF-8
? no locale-specific collation
