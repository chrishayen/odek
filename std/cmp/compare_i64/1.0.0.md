# std.cmp.compare_i64

Compares two i64 values and returns -1, 0, or 1.

## Signature

(a: i64, b: i64) -> i32

## Behavior

Returns -1 when `a < b`, 0 when `a == b`, 1 when `a > b`.

## Tests

+ returns -1 for (1, 2)
+ returns 0 for (1, 1)
+ returns 1 for (2, 1)
+ returns exactly one of {-1, 0, 1} for any input
- returns any value other than -1, 0, 1 under no circumstance

## Assumptions

? comparison is signed
