# std.json.pretty_print.serialize_value_indented

Serializes a JSON value with depth-based indentation for pretty-printed output.

## Signature

(value: json_value, depth: i32, indent: i32) -> string

## Behavior

Scalars (null, bool, number, string) delegate to their compact serializers — they emit on the current line. Arrays and objects emit `[` or `{`, a newline, each child on its own indented line, and a closing line at the outer depth. Empty arrays/objects emit `[]`/`{}` inline.

## Tests

+ emits a scalar inline without added whitespace
+ emits "{}" on one line for an empty object
+ emits "[]" on one line for an empty array
+ indents nested objects by depth * indent spaces
+ separates key and value with ": " (colon + single space)
- emits no trailing comma on the last child

## Assumptions

? separator between key and value is ": " (colon + single space)
? indent is assumed >= 0; negative values are clamped to 2
