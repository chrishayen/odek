# std.json.pretty_print

Serializes a tagged value tree as a JSON document with indentation.

## Signature

(value: json_value, indent: i32) -> string

## Behavior

Emits a JSON document formatted with `indent` spaces per nesting level. Each container element is placed on its own line, with one newline separating the opening brace/bracket from its first child and one before the closing brace/bracket. Keys and values are separated by `": "`. Empty objects and arrays are emitted on a single line.

## Tests

+ emits "{}" on one line for an empty object
+ emits each key on its own line for a non-empty object
+ indents nested structures by (depth * indent) spaces
+ uses "indent = 2" when indent <= 0 (sensible default)
+ separates keys and values with ": " (colon + single space)
- emits no trailing newline at the end of the document

## Assumptions

? no trailing comma on the last element of a container
? indent >= 0; negative values are clamped to 2
