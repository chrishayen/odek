# std.json.pretty_print.indent_line

Returns a string of spaces equal to `depth * indent`, used as the leading indentation for one pretty-printed line.

## Signature

(depth: i32, indent: i32) -> string

## Behavior

Returns `depth * indent` space characters. Never returns newlines. Zero depth or zero indent returns the empty string.

## Tests

+ returns "" for depth == 0
+ returns "    " for depth == 2 and indent == 2
+ returns "        " for depth == 2 and indent == 4
- returns no characters other than space
- returns "" when indent is zero regardless of depth

## Assumptions

? indentation character is always ASCII space; tabs are not used
? negative inputs are treated as zero
