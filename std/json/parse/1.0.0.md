# std.json.parse

Parses a JSON document into a tagged value tree.

## Signature

(raw: string) -> result[json_value, string]

## Behavior

Consumes a JSON document conforming to RFC 8259. Produces a `json_value` that can be a null, bool, number, string, list, or object. The whole input must be a single document; trailing non-whitespace bytes cause an error.

## Tests

+ parses an empty object "{}" to an empty map value
+ parses an empty array "[]" to an empty list value
+ parses a number "42" to a number value
+ parses a string "\"hello\"" to a string value
+ parses nested structures such as {"a":[1,2,3]}
+ accepts leading and trailing whitespace around the document
- returns error on trailing garbage after a valid document
- returns error on unterminated strings
- returns error on unquoted keys
- returns error on empty input

## Assumptions

? numbers are parsed as either i64 or f64 depending on the literal form
? duplicate object keys are resolved last-wins
