# std.json.parse.parse_value

Parses any single JSON value — the dispatcher that selects the right sub-parser based on the leading byte.

## Signature

(input: string, pos: i64) -> result[value_result, string]

## Behavior

Skips whitespace, then inspects the byte at the resulting position: `{` dispatches to `parse_object`, `[` to `parse_array`, `"` to `parse_string`, `t`/`f` to `parse_bool`, `n` to `parse_null`, `-` or a digit to `parse_number`. Returns the parsed value and the position after it.

## Tests

+ dispatches to parse_object for a leading "{"
+ dispatches to parse_array for a leading "["
+ dispatches to parse_string for a leading quote
+ dispatches to parse_number for a leading digit or "-"
- returns error when the byte at pos is not the start of any JSON value
- returns error when the recursion depth exceeds the implementation limit

## Assumptions

? recursion depth is bounded to prevent stack exhaustion (default 64)
