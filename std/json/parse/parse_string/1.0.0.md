# std.json.parse.parse_string

Parses a JSON double-quoted string literal and returns the decoded string and the position after the closing quote.

## Signature

(input: string, pos: i64) -> result[string_result, string]

## Behavior

Expects `"` at `pos`. Reads characters until the next unescaped `"`, decoding each `\`-escape via `parse_string_escape` and concatenating surrogate pairs when two `\uXXXX` escapes form one. Returns the decoded string and the index after the closing quote.

## Tests

+ parses "\"hello\"" into "hello"
+ decodes escape sequences inside the string
+ handles surrogate pairs (e.g. "\\uD83D\\uDE00" -> one emoji code point)
+ parses an empty string literal "\"\""
- returns error on an unterminated string (no closing quote)
- returns error on a lone high surrogate with no following low surrogate
- returns error on a raw control byte (below 0x20) inside the string

## Assumptions

? input outside the quoted region is left to the caller
? non-ASCII UTF-8 bytes inside the string pass through unchanged
