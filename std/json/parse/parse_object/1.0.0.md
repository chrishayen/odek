# std.json.parse.parse_object

Parses a JSON object literal and returns the key/value map and the position after the closing brace.

## Signature

(input: string, pos: i64) -> result[object_result, string]

## Behavior

Expects `{` at `pos`. Loops: skip whitespace, parse a string key, skip whitespace, consume `:`, skip whitespace, parse a value, skip whitespace, consume either `,` (continue) or `}` (return). An empty `{}` returns an empty map.

## Tests

+ parses "{}" into an empty map
+ parses "{\"a\":1,\"b\":2}" into a two-entry map
+ accepts whitespace around colons, commas, and braces
- returns error on a missing colon between key and value
- returns error on a trailing comma before closing brace
- returns error on a non-string key like {1:2}

## Assumptions

? duplicate keys are last-wins
? insertion order of the returned map follows the input order
