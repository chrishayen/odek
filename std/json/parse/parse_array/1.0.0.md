# std.json.parse.parse_array

Parses a JSON array literal and returns the list of values and the position after the closing bracket.

## Signature

(input: string, pos: i64) -> result[array_result, string]

## Behavior

Expects `[` at `pos`. Loops: skip whitespace, parse one value via `parse_value`, skip whitespace, consume either `,` (continue) or `]` (return). An empty `[]` returns an empty list.

## Tests

+ parses "[]" into an empty list
+ parses "[1,2,3]" into a list of three numbers
+ accepts whitespace around commas and brackets
- returns error on a trailing comma before the closing bracket
- returns error on a missing comma between elements
- returns error on a missing closing bracket

## Assumptions

? allows heterogeneous element types in one array
? stack-depth limit for nested arrays is enforced by parse_value, not here
