# std.json.parse.parse_bool

Recognizes the JSON literals `true` or `false` and returns both the value and the position after it.

## Signature

(input: string, pos: i64) -> result[bool_result, string]

## Behavior

Inspects the bytes at `pos`. If they are `true`, returns `(true, pos + 4)`. If they are `false`, returns `(false, pos + 5)`. Otherwise returns an error.

## Tests

+ returns (true, pos + 4) for input "true" at pos
+ returns (false, pos + 5) for input "false" at pos
- returns error when the bytes at pos match neither literal
- returns error when fewer bytes remain than the literal's length

## Assumptions

? comparison is byte-exact (case-sensitive)
