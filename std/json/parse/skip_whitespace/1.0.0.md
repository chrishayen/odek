# std.json.parse.skip_whitespace

Advances a cursor past any JSON whitespace bytes and returns the new position.

## Signature

(input: string, pos: i64) -> i64

## Behavior

Scans `input` forward from `pos`, consuming every byte that is space, tab, LF, or CR (the four JSON whitespace characters). Returns the index of the first non-whitespace byte, or `len(input)` if the rest of the input is whitespace.

## Tests

+ returns pos unchanged when the byte at pos is not whitespace
+ advances past a run of space, tab, LF, and CR bytes
+ returns len(input) when the rest is all whitespace
- does not advance past any character outside the four JSON whitespace bytes

## Assumptions

? JSON whitespace per RFC 8259 is {0x09, 0x0A, 0x0D, 0x20}; no other bytes are treated as whitespace
