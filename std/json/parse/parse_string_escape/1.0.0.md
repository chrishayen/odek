# std.json.parse.parse_string_escape

Decodes one JSON backslash escape sequence starting at the byte after the backslash.

## Signature

(input: string, pos: i64) -> result[escape_result, string]

## Behavior

Reads the byte at `pos` (the char after the `\`). Returns the decoded rune and the position after the escape. Simple escapes (`"`, `\`, `/`, `b`, `f`, `n`, `r`, `t`) advance by 1 past `pos`; `\uXXXX` advances by 5.

## Tests

+ decodes "n" into LF (U+000A)
+ decodes "\"" into a double-quote character
+ decodes "u0041" into the letter "A"
- returns error on an unrecognized escape letter like "\\q"
- returns error when "\\u" is not followed by 4 hex digits

## Assumptions

? surrogate pairs in `\uXXXX\uXXXX` are handled by the caller (parse_string), not here
? hex digits in `\uXXXX` are case-insensitive
