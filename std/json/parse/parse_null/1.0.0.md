# std.json.parse.parse_null

Recognizes the JSON literal `null` starting at a cursor and returns the position after it.

## Signature

(input: string, pos: i64) -> result[i64, string]

## Behavior

Checks whether the four bytes at `pos` are `null`. Returns `pos + 4` when they match. Returns an error otherwise.

## Tests

+ returns pos + 4 when input[pos..pos+4] is "null"
- returns error when fewer than 4 bytes remain from pos
- returns error when the bytes at pos are not exactly "null" (e.g. "NULL", "nul", "nuLl")

## Assumptions

? comparison is byte-exact (case-sensitive); JSON literals are lowercase per RFC 8259
