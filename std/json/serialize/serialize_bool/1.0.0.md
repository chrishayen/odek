# std.json.serialize.serialize_bool

Returns the JSON boolean literal for a bool value.

## Signature

(b: bool) -> string

## Behavior

Returns `true` when `b` is true, `false` when `b` is false.

## Tests

+ returns "true" for input true
+ returns "false" for input false

## Assumptions

? literals are lowercase
