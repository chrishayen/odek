# std.json.serialize.serialize_value

Dispatches any JSON value to the appropriate type-specific serializer.

## Signature

(value: json_value) -> string

## Behavior

Inspects the variant tag of `value` and calls the matching leaf: null → `serialize_null`, bool → `serialize_bool`, number → `serialize_number`, string → `serialize_string`, list → `serialize_array`, map → `serialize_object`. Returns the concatenated compact form.

## Tests

+ dispatches each variant to the correct leaf
+ produces a round-trippable output for every shape (parse → serialize → parse yields the same value)
- returns no output for an unrecognized variant tag (rejected upstream)

## Assumptions

? variant discrimination is exhaustive; a malformed json_value panics upstream
