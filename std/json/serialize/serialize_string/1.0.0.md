# std.json.serialize.serialize_string

Wraps a string in double quotes and escapes its contents per JSON rules.

## Signature

(s: string) -> string

## Behavior

Produces `"` + `escape_string(s)` + `"`. Always emits the surrounding quotes, even for the empty string.

## Tests

+ returns "\"\"" for the empty string
+ returns "\"hello\"" for "hello"
+ uses escape_string to handle control characters and quotes
- emits no unescaped double-quote inside the output

## Assumptions

? always uses `"` as the delimiter (never single quotes — JSON disallows them)
