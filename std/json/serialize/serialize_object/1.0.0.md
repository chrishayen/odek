# std.json.serialize.serialize_object

Serializes a key/value map into a compact JSON object literal.

## Signature

(entries: map[string, json_value]) -> string

## Behavior

Emits `{`, then for each entry the key via `serialize_string`, a `:`, the value via `serialize_value`, joined by `,` with no spaces, then `}`. An empty map emits `{}`.

## Tests

+ returns "{}" for an empty map
+ returns a compact object form like "{\"a\":1,\"b\":2}"
+ emits keys in the map's iteration order
+ separates key-value pairs with "," and no space
- emits no trailing comma

## Assumptions

? iteration order is the map's insertion order
? keys are always quoted, per JSON
