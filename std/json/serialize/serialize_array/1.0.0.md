# std.json.serialize.serialize_array

Serializes a list of JSON values into a compact JSON array literal.

## Signature

(values: list[json_value]) -> string

## Behavior

Emits `[`, then each element serialized via `serialize_value` joined by `,` with no spaces, then `]`. An empty list emits `[]`.

## Tests

+ returns "[]" for an empty list
+ returns "[1,2,3]" for a list of three numbers
+ separates elements with "," and no space
+ handles a list that contains nested arrays or objects
- emits no trailing comma

## Assumptions

? output is compact; pretty-print is a separate path
? element order is preserved
