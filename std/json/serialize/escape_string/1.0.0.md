# std.json.serialize.escape_string

Applies JSON backslash escapes to a string without adding surrounding quotes.

## Signature

(s: string) -> string

## Behavior

Walks the input code point by code point. Emits `\"` for `"`, `\\` for `\`, `\b` for 0x08, `\f` for 0x0C, `\n` for LF, `\r` for CR, `\t` for HT, and `\uXXXX` for any other byte below 0x20. All other bytes (including non-ASCII UTF-8) pass through unchanged.

## Tests

+ emits "\\n" for an LF byte
+ emits "\\\"" for a double-quote character
+ emits "\\u0001" for control byte 0x01
+ leaves non-ASCII UTF-8 bytes unchanged
- does not add surrounding quotes (caller handles that)

## Assumptions

? `/` is emitted unescaped (allowed by RFC 8259)
? non-BMP code points pass through as their UTF-8 bytes, not surrogate pairs
