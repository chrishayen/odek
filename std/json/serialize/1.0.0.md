# std.json.serialize

Serializes a tagged value tree into a compact JSON document.

## Signature

(value: json_value) -> string

## Behavior

Emits a single JSON document with no extra whitespace. Strings are escaped per RFC 8259: control characters below 0x20 and the characters `"`, `\`, are escaped; non-ASCII code points pass through as UTF-8. Object keys are emitted in insertion order.

## Tests

+ serializes an empty object to "{}"
+ serializes an empty array to "[]"
+ serializes a string containing a quote with the quote escaped
+ serializes control character 0x0A as "\n"
+ serializes nested structures without extra whitespace
- emits no trailing newline
- emits no spaces around colons or commas

## Assumptions

? output is compact (no pretty-printing); use pretty_print for indented output
? object keys preserve insertion order, not alphabetical order
