# std.regex.find_with_groups

Returns the first match along with each capture group's matched substring, or none.

## Signature

(re: regex, s: string) -> optional[groups_match]

## Behavior

Returns `some(m)` where `m` contains the whole match and a list of capture-group matches (indexed 1..N). Each group may be `some(text)` or `none` (did not participate). Returns none when the regex doesn't match.

## Tests

+ returns groups for a pattern with captures
+ returns none when no match
+ includes none for groups that did not participate in the match
+ returns whole-match at index 0

## Assumptions

? group indices are 1-based; 0 is the whole match
? the dialect supports parenthesized capture groups
