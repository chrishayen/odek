# std.regex.compile

Compiles a regex pattern into a reusable matcher object.

## Signature

(pattern: string) -> result[regex, string]

## Behavior

Parses the pattern using a linear-time-safe regex dialect (RE2-style, no backreferences) and builds an internal automaton. The returned regex value is immutable and may be reused across many match calls without recompilation.

## Tests

+ compiles a simple literal pattern like "hello"
+ compiles a character class like "[a-z]+"
+ compiles anchors "^" and "$"
+ compiles alternation "foo|bar"
- returns error on an unbalanced group like "(abc"
- returns error on an unbalanced character class like "[a-"
- returns error on an invalid repetition like "*abc"

## Assumptions

? dialect is RE2-compatible (no backreferences, no lookaround)
? patterns are parsed as UTF-8; bytes outside UTF-8 cause an error
