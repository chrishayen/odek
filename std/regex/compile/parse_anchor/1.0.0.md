# std.regex.compile.parse_anchor

Recognizes `^` or `$` at a token position and returns the matching anchor kind.

## Signature

(tokens: list[regex_token], pos: i64) -> anchor_result

## Behavior

Inspects the token at `pos`. Returns `begin` for `^`, `end` for `$`, `none` for anything else. Advances by 1 position when an anchor is recognized.

## Tests

+ returns begin for `^`
+ returns end for `$`
+ returns none when the token is neither anchor
+ advances pos by 1 when an anchor is recognized
- does not advance pos when the token is not an anchor

## Assumptions

? does not distinguish multiline mode; that is applied at match time
