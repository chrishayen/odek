# std.regex.compile.build_automaton

Translates a parsed regex AST into an NFA suitable for linear-time matching.

## Signature

(ast: regex_ast) -> regex

## Behavior

Walks the AST and builds a non-deterministic finite automaton with ε-transitions per Thompson's construction. Each AST node produces a small sub-automaton; these are wired together to form the full machine.

## Tests

+ produces an NFA whose initial state is the entry of the first sub-automaton
+ produces an NFA whose accepting state is the exit of the last sub-automaton
+ wires alternation as parallel sub-automata joined by ε-transitions
+ wires concatenation as sequential sub-automata
- produces no cycles in the state graph for a zero-repetition subpattern

## Assumptions

? no backreferences; the dialect is regular
? matching is linear in the length of the input
