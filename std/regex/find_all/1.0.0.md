# std.regex.find_all

Returns every non-overlapping match of the regex in the input string.

## Signature

(re: regex, s: string) -> list[string]

## Behavior

Scans `s` left-to-right for non-overlapping matches of `re`. Each match is appended to the result list as the exact matched substring. The scan resumes at the byte position immediately after the previous match. Empty matches advance the cursor by one code point to avoid infinite loops.

## Tests

+ returns ["foo", "foo"] for pattern "foo" against "foo bar foo"
+ returns an empty list when the regex does not match
+ returns every digit for pattern "[0-9]" against "a1b2c3"
+ advances past empty matches instead of looping
- does not return overlapping matches (pattern "aa" against "aaaa" returns 2 matches, not 3)

## Assumptions

? scan is leftmost, longest (POSIX) within the dialect's semantics
? matches are the exact matched substring; no capture-group extraction
