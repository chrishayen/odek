# std.regex.find_first

Returns the byte span (start, end) of the first match, or none.

## Signature

(re: regex, s: string) -> optional[match_location]

## Behavior

Scans left-to-right. Returns `some((start, end))` where `s[start..end]` is the matched substring. Returns none if no match.

## Tests

+ returns the correct (start, end) for a simple match
+ returns (0, n) when the regex matches the entire input
+ returns none when no match
+ covers only the first match
- returns no span with end < start

## Assumptions

? byte indices, not code-point indices
