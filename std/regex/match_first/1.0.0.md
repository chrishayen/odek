# std.regex.match_first

Returns the first match of the regex in the input, or none if no match.

## Signature

(re: regex, s: string) -> optional[string]

## Behavior

Scans left-to-right and returns the exact matched substring of the first match. Returns none when no match exists.

## Tests

+ returns some(match) for the first occurrence
+ returns none when the pattern does not match
+ returns the exact matched substring (not the whole input)
- returns only the first match, not all matches

## Assumptions

? leftmost-longest (POSIX) semantics within the dialect
