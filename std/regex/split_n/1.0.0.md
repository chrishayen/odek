# std.regex.split_n

Splits the input on regex matches at most `n-1` times, returning at most `n` segments.

## Signature

(re: regex, s: string, n: i64) -> list[string]

## Behavior

Splits like `regex.split` but stops after `n - 1` splits. The final element contains any unsplit remainder. When `n <= 0`, behaves like unlimited `split`.

## Tests

+ returns 2 segments for n=2 regardless of how many regex matches exist
+ returns full split when n is 0 or negative
+ returns [s] when n is 1
- does not exceed n segments

## Assumptions

? last element carries the remainder, with un-split matched text preserved
