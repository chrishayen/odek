# std.regex.replace

Returns a copy of the string with every non-overlapping regex match replaced by the given replacement.

## Signature

(re: regex, s: string, replacement: string) -> string

## Behavior

Scans `s` left-to-right for non-overlapping matches of `re`. Each match is removed from the output and `replacement` is emitted in its place. The replacement string is emitted literally; capture-group references like `$1` are not interpreted.

## Tests

+ returns "XbXc" for pattern "a" on input "abac" with replacement "X" -> "XbXc"
+ returns the input unchanged when the regex does not match
+ replaces every occurrence, not just the first
+ returns "" for an empty input
- does not interpret "$1" or "\1" as capture references; emits them literally

## Assumptions

? replacement is literal; use a dedicated format function for back-reference expansion
? substitution is greedy left-to-right and non-overlapping
