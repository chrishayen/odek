# std.regex.match

Returns true when the regex matches anywhere in the input string.

## Signature

(re: regex, s: string) -> bool

## Behavior

Searches the entire input for any match of the regex. Returns true on the first match, false if none. Does not return the matched text.

## Tests

+ returns true when the pattern matches
+ returns false when the pattern does not match
+ returns true when the pattern matches only part of the input
+ returns true for an empty regex (matches empty string)
- returns false for an empty input unless the regex matches empty

## Assumptions

? unanchored search; anchors in the pattern still work
