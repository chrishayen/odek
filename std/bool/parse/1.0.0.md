# std.bool.parse

Parses the strings "true" or "false" into a bool. Case-sensitive.

## Signature

(s: string) -> result[bool, string]

## Behavior

Returns true for exactly `"true"`, false for exactly `"false"`. All other inputs are errors.

## Tests

+ returns true for "true"
+ returns false for "false"
- returns error for "True" (case-sensitive)
- returns error for "1" or "0"
- returns error for "yes" or "no"
- returns error for an empty string

## Assumptions

? strict lowercase; case-insensitive parsing is a separate rune
