# std.bool.format

Formats a bool as the string "true" or "false".

## Signature

(b: bool) -> string

## Behavior

Returns `"true"` for true, `"false"` for false.

## Tests

+ returns "true" for true
+ returns "false" for false

## Assumptions

? always lowercase
