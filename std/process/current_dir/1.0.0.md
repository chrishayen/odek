# std.process.current_dir

Returns the absolute path of the process's current working directory.

## Signature

() -> result[string, string]

## Behavior

Queries the host OS for the calling process's current working directory and returns it as an absolute path. Does not resolve symbolic links — returns the logical path as set by the most recent successful `chdir`.

## Tests

+ returns an absolute path
+ returns a path that the process can stat
+ returns the same path on successive calls when no chdir has occurred between them
+ reflects the new path after a successful chdir
- returns error when the current directory has been removed under the process

## Assumptions

? returns the logical cwd; symbolic link components are preserved
? no implicit caching between calls
