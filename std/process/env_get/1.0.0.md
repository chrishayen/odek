# std.process.env_get

Returns the value of an environment variable, or none when it is not set.

## Signature

(name: string) -> optional[string]

## Behavior

Looks up `name` in the process environment table and returns the associated value. Returns none when the variable is not set. An empty-string value is considered set and is returned as the empty string (distinct from none).

## Tests

+ returns the value for a set variable
+ returns none for a variable that is not set
+ returns the empty string for a variable explicitly set to ""
+ preserves case of the name on POSIX hosts
- returns none (not error) for any unset variable

## Assumptions

? variable names are case-sensitive on POSIX and case-insensitive on Windows
? the environment is a snapshot at call time; not cached
