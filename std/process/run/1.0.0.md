# std.process.run

Runs an external command synchronously and returns its exit status and captured output.

## Signature

(cmd: string, args: list[string]) -> result[process_result, string]

## Behavior

Spawns a child process executing `cmd` with `args`, captures its stdout and stderr into separate byte buffers, and waits for it to exit. Returns the exit code along with the captured streams. Does not invoke a shell; arguments are passed directly without expansion.

## Tests

+ returns exit_code 0 for a command that succeeds
+ returns non-zero exit_code when the command exits with an error
+ captures stdout separately from stderr
+ preserves the order of bytes within each captured stream
+ passes args verbatim, without shell quoting or expansion
- returns error when cmd is not found on the host PATH
- returns error when cmd is not executable
- returns error when the child is killed by a signal before any exit code is set

## Assumptions

? stdout and stderr are buffered in memory; use a streaming API for long-running commands
? no shell is invoked; glob patterns and variables are not expanded
